## 下载

### Docker

```
docker pull apachecn0/vkdoc-db-zh
docker run -tid -p <port>:80 apachecn0/vkdoc-db-zh
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g vkdoc-db-zh
vkdoc-db-zh <port>
# 访问 http://localhost:{port} 查看文档
```
