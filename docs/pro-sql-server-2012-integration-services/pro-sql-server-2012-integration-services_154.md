## 关于作者

**弗朗西斯·罗德里格斯** 毕业于马里兰州洛约拉大学计算机科学专业。他也是纽约市耶稣会学校里吉斯高中的校友。他目前在纽约市担任商业智能顾问。他是企业商业智能项目的资深开发专家。他的专长包括基于 SQL Server 和 SQL Server Integration Services (SSIS) 的提取、转换和加载 (ETL) 解决方案。在业余时间，他会在纽约地区各地进行山地自行车运动。

**迈克尔·科尔斯** 拥有十多年的 SQL Server 数据库设计和管理经验。他是一位多产的 SQL Server 各方面文章作者，尤其擅长 T-SQL 的高级应用，持有 MCDBA 和 MCP 认证。他以优异成绩毕业于佐治亚州的美国洲际大学，获得信息技术理学学士学位。他是美国陆军预备役成员，在 9/11 事件后被激活服役了两年。

**大卫·戴伊** 是 Microsoft SQL Server MVP、讲师和作者，专注于关系数据库管理系统、商业智能系统、报表解决方案和 Microsoft SharePoint。在过去的 9 年里，David 的专业知识一直聚焦于 Microsoft SQL Server 的开发和管理工作。他的工作为他赢得了认可：2009 年和 2010 年的 Microsoft MVP，Microsoft Developer Network for SQL Server 论坛版主，2009 年《SQL Server Magazine》评选的年度创新者亚军，以及 2011 年 4 月入选 Training Associates 技术培训师聚焦榜。David 目前担任 APress 出版社 SQL Server 2012 系列的技术审稿人和合著者，以及 Packt 出版社的作者。

## 关于技术审稿人

**罗德尼·兰德鲁姆** 使用 SQL Server 的时间久到自己都记不清了。他经常撰写关于包括 Integration Services、Analysis Services 和 Reporting Services 在内的各种技术的文章。他著有《SQL Server Tacklebox》和三本关于 Reporting Services 的书籍。他定期为 SQLServerCentral、SQL Server Magazine 和 Simple-Talk 撰稿。他的日常工作涉及管理奥兰多市一个大型的 SQL Server 基础设施。他声称自己拥有“日常工作涉及数据库”这句短语的所有权。任何不同意这点的人都想和他掰手腕比赛并输掉一臂。
