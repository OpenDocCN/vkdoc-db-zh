# 第 9 章 变量、参数与表达式

#### 字符函数

`REVERSE( «character_expression» )` 返回一个字符串，其字符顺序与输入字符串相反。

`RIGHT( «character_expression», «number» )` 返回一个字符串，由字符表达式中最右侧指定数量的字符组成。

`RTRIM( «character_expression» )` 返回一个字符串，去除了输入字符串末尾的空白字符。

`SUBSTRING( «character_expression», «start», «length» )` 返回一个字符串，由字符表达式中从指定起始索引开始、指定长度的字符组成。

`TOKEN( «character_expression», «delimiter_expression», «occurrence» )` 返回一个字符串，由字符表达式中从指定出现次数到首次遇到的分隔符表达式之间的所有字符组成。

`TOKENCOUNT( «character_expression», «delimiter_expression» )` 返回基于分隔符表达式作为令牌终止符，字符表达式中存在的令牌数量。

`TRIM( «character_expression» )` 返回一个字符串，去除了输入字符串开头和末尾的空白字符。

`UPPER( «character_expression» )` 返回一个字符串，由输入字符的大写形式组成。

#### 日期函数

日期函数对日期和时间戳值执行操作。以下是表达式生成器中可用的日期函数：

`DATEADD( «datepart», «number», «date» )` 返回一个日期值，该值是在日期输入的指定部分添加指定数量的时间单位后得到的。

`DATEDIFF( «datepart», «startdate», «enddate» )` 返回起始日期和结束日期之间指定时间单位的长度值。

`DATEPART( «datepart», «date» )` 返回一个整数值，表示指定日期的一部分。如果日期部分指定为月份，则一月份的日期将返回 1。

`DAY( «date» )` 返回一个整数值，表示输入日期的日。

`GETDATE()` 返回主机上设置的当前日期和时间。

`GETUTCDATE()` 返回主机上设置的当前日期和时间（协调世界时）。

`MONTH( «date» )` 返回一个整数，表示输入日期的月份编号。

`YEAR( «date» )` 返回一个整数，表示输入日期的年份编号。

#### NULL 函数

SSIS 中存在的最后一类函数是 NULL 函数。此类别中除一个函数外，其余函数都返回针对特定数据类型转换的 NULL 值。为了返回特定数据类型的 NULL，你需要将函数 `NULL(«data type»)` 作为表达式传入，并填写所需的数据类型名称。`ISNULL(«expression»)` 函数将对输入表达式进行求值，并返回一个布尔值，指示其是否计算为 NULL。

#### 运算符

表达式语言中的运算符与 C# 中的运算符工作方式类似。有单目运算符和双目运算符。在表达式求值时，运算符确实具有优先级顺序。为了保持清晰，我们建议尽可能使用括号，否则在快速阅读代码时可能会出现歧义。以下是 SSIS 中可用的运算符：

`+` （加/连接）将两个表达式相加。也可用于连接字符串。

`-` （减/取负）将两个表达式相减。也可用于返回数值表达式的负值。

`*` （乘）将两个数值表达式相乘。

`/` （除）将两个数值表达式相除。

`%` （取模）返回两个数值表达式相除的整数余数。

`()` （括号）用于优先计算表达式。

`==` （等于）逻辑上评估两个表达式是否相等。

`!=` （不等于）逻辑上评估两个表达式是否不相等。

`>` （大于）逻辑上评估运算符左侧的表达式是否大于右侧的表达式。
