### 总结

随着 SSIS 中新部署模型的引入，您现在可以将 ETL 流程部署到 SQL Server，而无需承担早期版本中存在的开销。新的环境和环境变量消除了为部署到不同测试和生产环境维护配置的需要。我们向您展示了如何构建 SSIS 项目并将其部署到 SQL Server，以及如何将遗留的 SSIS 包迁移到新模型。下一章将介绍源代码控制和 SSIS 管理与安全。
