# 第三章 - 你好世界——你的第一个 SSIS 2012 包

### 平面文件连接管理器编辑器

##### 常规页面

`文件名`字段存储将与此特定数据集关联的文件夹路径和文件名。`浏览`按钮允许你使用 Windows 资源管理器定位文件，并将路径自动填充到字段中。使用`浏览`选项时，请注意默认选择的文件类型是`.txt`。在我们的案例中，我们使用的是`.csv`文件作为数据源，因此必须更改文件类型才能在 Windows 资源管理器中查看。`区域设置`下拉列表提供了一个区域列表，用于特定语言的时间格式排序。`Unicode`复选框定义数据集使用 Unicode，这将禁用`代码页`功能。`代码页`选项允许你为平面文件定义非 Unicode 代码页。

平面文件连接管理器编辑器`常规`页面的`格式`部分允许你定义 SSIS 将如何读取文件。有三种格式可供选择：`分隔符`、`固定宽度`和`右端不齐`。`分隔符`是默认格式，允许你在`列`选项卡中定义分隔符。对于`.csv`（逗号分隔值）文件，分隔符是逗号。`固定宽度`将平面文件中的列定义为每个列都具有确定的列长度。长度以字符为单位测量。标尺有助于你为每个列定义单独的字符。你可以从 50 种不同的字体中选择，以准确显示字符。`右端不齐`遵循与`固定宽度`相同的规则，除了最后一列。最后一列具有行分隔符，而不是固定宽度。

`文本限定符`文本框允许你指定标识文本数据的限定符。限定文本最常用的方式是使用引号。`标题行分隔符`列表定义了标记数据第一行的字符。通常，这与行分隔符相同。对于`.csv`文件，行分隔符是回车符后紧跟换行符。当我们从人口普查局网站下载数据时，我们清理了一些标题记录，只保留了列名和实际数据。标记为`首行数据包含列名`的复选框允许 SSIS 轻松读取文件并捕获`.csv`文件中定义的列名，如`图 3-4`所示。

这可以清理名称，这样你就不会看到诸如`Column1`、`Column2`等名称，而是用`.csv`文件中定义的实际列标题替换这些通用名称。

`注意：`我们在本书的这个阶段不会创建配置文件，但此文件夹路径将是存储在连接字符串属性中的关键信息。在循环遍历具有相同元数据的文件列表时，也可以在运行时更改连接字符串。

`图 3-4. 平面文件连接管理器编辑器 - 列页面`

##### 列页面

平面文件连接管理器编辑器的`列`页面允许你定义行分隔符和列分隔符。对于`.csv`文件，行由回车符和换行符分隔，列由逗号分隔。设置这些值后，你可以使用定义的`分隔符`值预览行的样本。如果需要对分隔符进行任何更改，可以刷新`预览`窗格以查看更改将如何影响连接管理器对文件的读取。`重置列`按钮将删除除原始列之外的所有列。

使用`固定宽度`选项时，`预览`窗格顶部有一个标尺。在此模式下，预览将忽略实际文件中的换行符。标尺上的标记将划分列的宽度。每个列的宽度都可以使用标尺定义。`预览`窗格将在视觉上辅助显示连接管理器将如何读入数据。`右端不齐`选项将有一个额外的字段，允许你定义行分隔符。

##### 高级页面

平面文件连接管理器编辑器的`高级`页面（如`图 3-5`所示）允许你修改每个列的元数据。此处显示的列取决于通过`列`页面定义的列。在`高级`页面中，你可以修改每个列的名称、列分隔符、数据类型、输出列宽度和文本限定符。对于平面文件中的最后一列，列分隔符将是行分隔符。根据选择的数据类型（数值数据），数据精度和数据小数位数变为可用。列类型决定这些选项，对于分隔符格式，这些是可用列。对于其他两种格式，你可以单独定义输入列宽度。此处的更改将反映在`列`页面的`预览`窗格中。

`图 3-5. 平面文件连接管理器编辑器 - 高级页面`

#### 预览页面

`预览`页面根据在其他页面中定义的元数据显示来自源的示例数据集。`图 3-6`演示了平面文件连接管理器编辑器的此页面。它还具有跳过源中数据行的选项。这与`常规`页面上的选项不同。`常规`页面选项允许忽略标题行，以便可以使用适当的行作为列名的源。此`预览`页面选项将忽略紧接在包含列标题的行之后的行。

`图 3-6. 平面文件连接管理器编辑器 - 预览页面`

### OLE DB 目标连接

数据源准备好提取后，我们需要设置加载目标。在此示例中，我们将使用 SQL Server 12 数据库作为目标。我们将创建一个适合源元数据的表，实质上是将数据从文件系统的平面文件存储到关系数据库管理系统（`RDBMS`）中。创建对象链接与嵌入数据库（`OLE DB`）连接管理器的第一步是在设计器窗口中的`连接管理器`窗格中右键单击。如果尚未有数据连接，你将需要使用如`图 3-7`所示的对话框添加到服务器的数据连接。创建数据连接时，需要准备好登录信息。如果不知道服务器名称，可以输入服务器的`IP`地址，或者如果要将数据加载到本地计算机的实例中，可以使用`(local)\实例名`来调用它。

`图 3-7. 配置 OLE DB 连接管理器`

此管理器允许你存储在`SSIS`中使用的数据连接。这不仅限于特定项目内使用的连接。你可以使用如`图 3-8`所示的向导配置新连接。默认情况下，存储在连接管理器中的连接名称格式如下：`数据源.初始目录`。`数据连接属性`窗格将显示该特定连接的连接字符串信息和默认值。单击`新建`按钮将提示如图`图 3-8`所示的连接管理器向导。连接管理器不仅限于提取或加载任务；它们可以同时执行这两项任务，尽管同时执行可能会引发争用问题。
