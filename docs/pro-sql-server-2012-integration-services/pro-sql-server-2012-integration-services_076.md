`>=` (大于或等于) 在逻辑上评估运算符左侧的表达式是否大于或等于右侧的表达式。

`<=` (小于或等于) 在逻辑上评估运算符左侧的表达式是否小于或等于右侧的表达式。

`&&` (逻辑与) 对运算符左右两侧的表达式执行逻辑与操作。

`||` (逻辑或) 对运算符左右两侧的表达式执行逻辑或操作。

`? :` (条件) 对问号前的表达式进行条件检查。如果表达式求值为真，则返回问号和冒号之间的表达式。如果表达式求值为假，则返回冒号后的表达式。

`&` (按位与) 对运算符两侧的两个整数值执行按位与操作。

`|` (按位或) 对运算符两侧的两个整数值执行按位或操作。

`^` (按位异或) 对运算符两侧的两个整数值执行按位异或操作。

`~` (按位非) 对整数值执行按位取反。

`!` (逻辑非) 对布尔表达式的值进行取反。
