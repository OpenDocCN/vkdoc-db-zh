# 第 1 章 - 集成服务简介

### SSIS 新特性

在 SQL Server 12 版本的 SSIS 中，有许多改进之处。我们将在本书中讨论这些新功能和增强特性。为了方便起见，在深入后续章节的细节之前，我们在此总结一下这些新特性：

*   **项目部署模型**：新的项目部署模型是 SQL Server 12 SSIS 中新增的功能。此新部署模型的关键特性包括集成服务目录、环境和参数。新部署模型旨在简化 SSIS 包和 ETL 系统在多个环境中的部署和管理。我们将在第 17 章讨论项目部署模型。
*   **T-SQL 视图和存储过程**：新的项目部署模型包含了多个针对 SQL Server 的 SSIS 专用视图和存储过程。我们将在第 17 章讨论项目部署模型时介绍这些视图和存储过程。
*   `BIDS` **可用性增强**：`BIDS` 得到了改进，使得包开发和编辑更加简单。`BIDS` 设计器现在更加灵活，UI 方面的增强（如 `集成服务` 工具箱）使其更易于使用。我们将在第 3 章介绍 `BIDS` 和 `SQL Server Management Studio (SSMS)` 的新特性。
