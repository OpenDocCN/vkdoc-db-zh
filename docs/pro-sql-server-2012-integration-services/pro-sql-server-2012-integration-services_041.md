#### “Foreach 循环编辑器”—“集合”页面

如图 6-30 所示的`Foreach 循环编辑器`的`集合`页面，允许您配置许多将决定容器循环结构的属性。该页面将根据您需要使用的枚举器类型，动态修改可用的属性。枚举器类型包括：`Foreach 文件枚举器`、`Foreach 项枚举器`、`Foreach ADO 枚举器`、`Foreach ADO.NET 架构行集枚举器`、`Foreach 来自变量枚举器`、`Foreach 节点列表枚举器`和`Foreach SMO 枚举器`。

`集合`页面允许您配置以下特定于上下文的属性：

`枚举器`列出了`Foreach 循环`容器可遍历的所有可用枚举器类型。

##### Foreach 文件枚举器

`Foreach 文件枚举器`允许容器遍历文件列表。

`文件夹`指定要枚举的文件所在的工作文件夹。

`浏览`允许您导航文件系统，轻松找到文件夹。

`文件`以逗号分隔的格式列出要用作枚举器的文件。可以使用通配符通过扩展名或通用命名标准来标识文件。

`完全限定`在容器遍历文件列表时，检索文件夹路径、文件名和文件扩展名。此值可以分配给变量，供容器内的可执行文件使用。

`名称和扩展名`在容器遍历文件列表时，检索文件名和文件扩展名。此值可以分配给变量，供容器内的可执行文件使用。

`仅名称`在容器遍历文件列表时，检索文件名。此值可以分配给变量，供容器内的可执行文件使用。

`遍历子文件夹`允许您将子文件夹内的文件包含在文件列表中。

##### Foreach 项枚举器

`Foreach 项枚举器`允许您定义一个自定义项列表，供容器循环处理。`集合`页面上的一个窗口允许您识别列表的不同元素。`列`按钮用于向枚举器添加列。在最后一行填充值后，会向列表添加一个新行。

`列`按钮打开一个对话框，允许您向枚举器添加列并定义其数据类型。

`删除`从枚举器中删除一个元素。

##### Foreach ADO 枚举器

`Foreach ADO 枚举器`允许使用`ADO`或`ADO.NET`对象作为容器的枚举器。`ADO`对象将枚举单个数据表的行，而`ADO.NET`对象可以枚举多个数据表中的行或数据表本身。

`ADO 对象源`列出容器作用域内所有用户创建的变量。此变量将作为数据表或一组数据表被遍历。

`第一个表中的行`仅遍历出现在第一个数据表中的行。此选项可应用于`ADO`和`ADO.NET`对象。

`所有表中的行（仅限 ADO.NET 数据集）`遍历所有数据表中的所有行。



所有表（`仅限 ADO.NET 数据集`）遍历数据表本身。

Foreach ADO.NET 架构行集枚举器使用用于 OLEDB 的 .NET 提供程序连接到 SQL Server 并枚举特定的数据库对象。

`连接`列出包中定义的、用于连接到 SQL Server 的 OLEDB 的 .NET 提供程序。

`架构`定义要用作枚举器的方案。

`设置限制`为架构的每个属性创建限制。一个对话框允许您基于字符串值或变量值来定义限制。

Foreach 来自变量的枚举器允许您枚举对象变量中的数据。

`变量`列出所有用户创建的变量。

Foreach 节点列表枚举器允许您枚举由 XPath 表达式产生的 XML 语句的特定节点。

`DocumentSourceType`定义 XML 语句的输入方法。`文件连接`选项允许您指定一个指向包含所需节点的 XML 文件的连接管理器。`变量`选项表示 XML 存储在 SSIS 变量中。`直接输入`选项要求您直接在编辑器中键入 XML。

`DocumentSource`根据其类型指定 XML 源。对于`文件连接`和`变量`选项，您会得到一个下拉列表，分别包含文件连接或 SSIS 变量。对于`直接输入`，编辑器提供一个文本字段用于键入 XML。

`EnumerationType`定义 XML 节点的枚举类型。`导航器`选项利用`XPathNavigator`来枚举 XML。`节点`选项使用 XPath 表达式来检索 XML 节点。`节点文本`选项使用 XPath 表达式来检索文本节点。`元素集合`选项使用 XPath 表达式来检索元素节点。前三个选项需要使用应用于 XML 的外部 XPath 表达式。最后一个选项使用外部 XPath 表达式，并且可以对元素集合应用内部 XPath 表达式。

`OuterXPathStringSourceType`为应用于 XML 语句的 XPath 表达式提供源类型。`文件连接`选项允许您指定一个指向 XPath 表达式的连接管理器。`变量`选项表示 XPath 表达式存储在 SSIS 变量中。`直接输入`选项要求您直接在编辑器中键入 XPath 表达式。

`OuterXPathString`根据其类型指定 XPath 表达式源。对于`文件连接`和`变量`选项，您会得到一个下拉列表，分别包含文件连接或 SSIS 变量。对于`直接输入`，编辑器提供一个文本字段用于键入 XPath 表达式。

`InnerElementType`指定列表中内部元素的类型。类型选项包括`导航器`、`节点`和`节点文本`。

`InnerXPathSourceType`为应用于元素集合的 XPath 表达式提供源类型。`文件连接`选项允许您指定一个指向 XPath 表达式的连接管理器。`变量`选项表示 XPath 表达式存储在 SSIS 变量中。`直接输入`选项要求您直接在编辑器中键入 XPath 表达式。

`InnerXPathString`根据其类型指定 XPath 表达式源。对于`文件连接`和`变量`选项，您会得到一个下拉列表，分别包含文件连接或 SSIS 变量。对于`直接输入`，编辑器提供一个文本字段用于键入 XPath 表达式。
