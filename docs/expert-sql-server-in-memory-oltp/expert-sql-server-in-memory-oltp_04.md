# 下载代码

您可以从 Apress 网站（`www.apress.com`）的源代码部分，或从我博客的出版物部分（`http://aboutsqlserver.com`）下载本书使用的代码。源代码包含一个 `SQL Server Management Studio` 解决方案，其中包含一组项目（每章一个）。此外，它还包括若干 .NET `C#` 项目，这些项目提供了第 2 章和第 13 章示例中使用的客户端应用程序代码。

我已在为 `SQL Server` 提供 8GB 内存的环境中测试了所有脚本。在某些情况下，如果您的可用内存较少，您将需要减少某些脚本生成的测试数据量。您也可以考虑删除一些未使用的测试表以释放更多内存。
