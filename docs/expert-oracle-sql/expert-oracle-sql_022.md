# 关于作者

![image](img/9781430259770_unFM-01.jpg)

**Tony Hasler** 是一名独立的软件顾问，专门从事 Oracle 数据库的 SQL 调优。他于 1978 年从英国兰卡斯特大学获得计算机研究荣誉学位，随后的 15 年一直从事软件开发工作。

Tony 在职业生涯的头几年在英国和荷兰工作，然后搬到了新泽西州，在那里他遇到了他未来的妻子 Marianne。Tony 于 1988 年回到英国，加入数字设备公司（DEC），担任 OpenVMS 操作系统开发团队的团队负责人。在 DEC 的五年期间，Tony 为日志记录协议的优化申请了一项专利，并且作为英国标准协会（BSI）的代表，他参与了国际标准化组织（ISO）多项出版物的制定工作。

1993 年离开 DEC 后不久，Tony 开始了他的第二个职业生涯——支持工作，主要专注于为 Wall Street Systems (`http://www.wallstreetsystems.com/`) 的应用程序进行性能调优。2004 年，Tony 离开了应用程序和操作系统领域，开始专注于 Oracle 数据库技术。在撰写本文时，Tony 正在全职担任 SQL 性能专家。

Tony 是英国著名的演讲者，并为英国 Oracle 用户组发表过数篇文章。当 Tony 发现有值得谈论的有趣事情时，他偶尔也会在他的博客 `http://tonyhasler.wordpress.com/` 上发表文章。

尽管 Tony 和他的妻子 Marianne 住在苏格兰，但他目前的客户都没有在苏格兰的业务；他们是总部设在伦敦的全球性金融机构。远程办公的时代已经到来。

Tony 有两个已成年的孩子，他们都子承父业。Timothy 在邓迪的阿伯泰大学获得计算机游戏学位，现在爱丁堡担任软件开发人员，而 Thomas 在获得爱丁堡大学的软件工程学位后，正在伦敦担任软件开发人员。
