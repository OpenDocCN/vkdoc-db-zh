# 运行示例

本书清单中 SQL 语句的运行脚本可以从 `http://www.apress.com/9781430259770` 下载。如果您想自己运行脚本，我建议使用数据库产品的 12cR1 版本，尽管大多数脚本也能在 11gR2 上运行。以下是附加要求：

*   数据库应具有 8k 的块大小，并且您应将初始化参数 `db_file_multiblock_read` 设置为 128。
*   需要安装示例方案（`SCOTT, OE, HR, PM` 和 `SH`）。有关更多详细信息，请参阅《示例方案手册》以及 `$ORACLE_HOME/rdbms/admin/scott.sql` 脚本。

在本书审阅过程中，发现不同的示例方案安装方式可能导致执行计划的不一致。可下载的材料包含一组可用于示例方案的对象统计信息，可通过数据泵导入安装。这些统计信息应能帮助您重现本书中显示的结果。完整说明包含在材料中的 README 文件中。

在出版时，Oracle 数据库 12cR1 唯一可用的点版本是 12.1.0.1，不幸的是，存在一个与连接基数估计相关的缺陷，这使得第 6 章和第 20 章中描述的 TSTATS 技术的一个关键要素无法按所述方式工作。希望这个缺陷在以后的点版本中得到修复，但与此同时，可下载的代码包含了一个变通方案：不是完全移除高值和低值列统计信息，而是将列统计信息的高值设置得非常高，低值设置得非常低。
