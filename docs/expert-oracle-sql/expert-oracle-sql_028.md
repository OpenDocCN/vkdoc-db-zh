# 第 3 部分

提供了对 CBO 的研究。我没有陷入大量成本计算的公式中；我涵盖了在您 SQL 调优生涯中将需要的访问方法、连接顺序和连接方法的基本细节。我还详细介绍了您可能遇到的所有优化器转换。

## 第 4 部分

涵盖了 SQL 优化。既然我们已经牢固掌握了工具知识，现在终于可以看看如何将所有这些知识应用于解决真实的 SQL 性能问题。我涵盖了物理数据库设计和重写 SQL，然后详细探讨了最具争议的话题：提示。还有一章专门讨论排序，另一章涵盖了解决不寻常问题的一系列高级技术。

## 第 5 部分

是一个专门讨论 TSTATS 的章节，这是一种在生产环境中管理对象统计信息的技术。TSTATS 几乎消除了不需要的执行计划变更，而无需管理 SQL 计划基线的存储库。这无疑是一个有争议的章节，但本章描述的技术已在多个关键任务应用程序中经过多年验证。
