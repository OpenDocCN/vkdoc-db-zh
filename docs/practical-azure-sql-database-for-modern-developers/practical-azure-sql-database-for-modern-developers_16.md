# 电池已备

保持简单，尽可能简单。但不要过于简单！这就是 Azure SQL 可以提供巨大帮助的原因。就像 Python 以其“电池已备” (`https://aka.ms/pep0206`) 而闻名一样，我们也喜欢用同样的定义来形容 Azure SQL。它几乎提供了开发任何项目所需的一切。从关系型支持到图模型，从 JSON 文档到列存储索引，你可以利用与采用多语言持久性理念相同的优点，而复杂性和缺点却少得多。

本书将帮助你学习所有可用于创建现代解决方案的功能，利用 Azure SQL 将所有数据操作和管理卸载到云端，这样你就可以专注于只有你能做的事情：实现业务逻辑和针对你所构建的解决方案独一无二的正确架构。
