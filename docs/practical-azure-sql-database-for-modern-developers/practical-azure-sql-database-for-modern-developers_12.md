## 桑杰·米什拉（Sanjay Mishra）
我最近完成了在微软的 15 年工作。在我早期的微软岁月中，有两个人不遗余力地帮助我适应并成长，我想肯定他们的贡献和影响——迈克·鲁斯拉夫（Mike Ruthruff）和普雷姆·梅赫拉（Prem Mehra）。迈克在我早期花费了无数时间帮助我理解我所接触的新数据库产品，这个产品与我加入微软时熟知的数据库产品如此相似却又如此不同。从安装 Windows（是的，在那些安腾服务器上安装和配置 Windows 可不容易）到连接调试器并在我们的`SQLCAT`实验室中调试具有复杂客户工作负载的早期版本，迈克是我解决一切问题的求助对象。迈克的友善和慷慨时间帮助我精通了`SQL Server`；否则，这本书将只是一个遥远的梦想。



Prem 对我倾囊相授，悉心指导我如何与各个层级的客户打交道——从数据库开发人员到首席技术官。每当 Prem 与客户开会时，我都会坐在一旁，观察他如何应对——无论客户是生气还是高兴，是沮丧还是困惑，Prem 的情绪始终保持稳定；我从未见他慌乱过。客户们都非常尊重他。如果我需要单独会见客户，并且知道 Prem 之前曾与他们合作过，我会在自我介绍时说“我和 Prem 一起工作”，这能让我立刻与客户建立起融洽的关系，而这种关系通常需要数日乃至数月才能建立。多年以后，我因对客户的同理心得到了许多人的赞誉，但这一切都归功于 Prem 的指导。

Prem 正享受着他的退休生活，Mike 也在他的初创公司乐在其中，但我每天在工作中都思念着他们。如果没有 Mike 和 Prem 在我早年的影响，我在微软的 15 年历程以及这本书都将不复存在。谢谢你，Mike！谢谢你，Prem！

**Jovan Popovic**

我要感谢我的女儿们和妻子，在这段旅程中给予我的耐心与支持。
