# 第三部分：添加、更新、删除数据

**第 13 章：理解数据类型。** Oracle 支持许多不同的数据类型，每种类型都有其特定用途。在本章中，您将了解几种主要的数据类型。

**第 14 章：创建表。** 数据存储在数据库的表中，本章将向您展示如何创建表。

**第 15 章：向表中添加数据。** 要查看表中的数据，首先需要将其添加到表中。本章解释了如何在 `SQL` 中完成此操作。

**第 16 章：更新和删除数据。** 偶尔您需要更改数据库中已存在的数据。您可以在 Oracle `SQL` 中执行此操作，本章将向您展示如何操作。

**第 17 章：更新或删除表。** 本章解释了如何更改表的结构，或者在不再需要时将其删除。
