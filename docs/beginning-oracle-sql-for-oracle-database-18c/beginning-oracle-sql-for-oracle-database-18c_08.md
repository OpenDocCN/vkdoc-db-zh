# 第七部分：附录

## 附录：如何查找和浏览 Oracle SQL 参考手册
Oracle 网站包含大量关于 SQL 的信息。本附录将说明如何找到该网站并进行浏览。

### 关于作者

自 2004 年上大学以来，我一直与 Oracle 数据库打交道。大学毕业后，我的第一份工作是支持一系列构建在 Oracle 数据库上的应用程序，此后我在软件开发行业担任过多种职位。在此期间，我获得了“Oracle SQL 专家”认证，备考过程也帮助我提升了 Oracle 数据库知识。我还运营着网站 databasestar.com，该网站为数据库开发人员提供实用建议，尤其专注于 Oracle SQL。

现在你对我以及你将在本书中学到的内容有了些许了解，是时候开始使用 Oracle Database 18c 学习 SQL 了。
