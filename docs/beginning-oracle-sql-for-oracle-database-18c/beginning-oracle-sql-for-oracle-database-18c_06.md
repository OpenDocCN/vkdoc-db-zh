# 第五部分：函数

## 第 22 章：在 SQL 中使用函数
SQL 包含对数据执行计算和转换的函数。本章你将了解它们是什么以及如何使用。

## 第 23 章：编写条件逻辑
本章中，你将学习如何实现条件逻辑，即条件为真时显示一个值，为假时显示另一个值。

## 第 24 章：理解聚合函数
SQL 中的一些函数被称为聚合函数，它们将多行数据组合在一起。本章解释它们是什么以及如何使用。

## 第 25 章：对结果进行分组
使用聚合函数时，你可能需要对结果进行分组。本章解释其含义以及如何操作。

## 第 26 章：什么是索引？
本章中，你将学习索引是什么、如何创建索引以及一些相关技巧。
