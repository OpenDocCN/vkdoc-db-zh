# 第二部分：查看数据

**第 3 章：检索数据。** 本章介绍 `SELECT` 语句，您可以用它从数据库中检索数据并进行查看。

**第 4 章：选择特定列。** 本章扩展了前一章的内容，向您展示如何使用 `SELECT` 语句查看单个列。

**第 5 章：限制结果。** 您将学习如何使用 `WHERE` 子句仅查看表中的部分记录，因为有时您并不想看到每条记录。

**第 6 章：比较数据。** 有时您可能希望基于部分匹配进行过滤，您将在本章中学习如何操作。

**第 7 章：应用多个过滤器。** `SQL` 允许您根据多个条件过滤记录。在本章中，您将学习如何做到这一点。

**第 8 章：处理空值。** 数据库有一个称为 `NULL` 的概念，表示未知值。您将了解更多关于其含义以及如何在查询中处理它的信息。

**第 9 章：移除重复结果。** 编写查询时，您偶尔会得到重复的结果。本章将解释如何移除重复项，仅查看唯一的结果。

**第 10 章：对列表和值范围应用过滤器。** 本章将解释如何通过使用值列表，更高效地使用多个值来过滤您的结果。

**第 11 章：对数据排序。** 如果您曾经需要以特定顺序显示数据，读完本章后您就会知道如何操作。

**第 12 章：应用表别名和列别名。** 在本章中，您将了解什么是表别名和列别名，以及它们的优点。
