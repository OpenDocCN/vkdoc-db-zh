# 第 23 章 ■ 内存优化 OLTP 表与过程

## 总结

本章介绍了内存表和本地编译存储过程的概念。这些是实现潜在大规模性能提升的高端方法。然而，在广泛实施这些新对象方面存在很多限制。你需要拥有 SQL Server 的 `Enterprise` 版本以及一台具有足够内存来支持额外负载的机器。你还需要在将这种方法投入生产环境之前，仔细测试你的数据和负载。但是，如果你确实需要让你的 OLTP 系统比以往任何时候都运行得更快，这是一项可行的技术。
