# 第 25 章：数据库工作负载优化
演示了如何利用前面所有章节中呈现的信息，并将其应用于真实的数据库工作负载。您将识别性能最差的存储过程，并对它们应用各种调优方法以获得更好的性能。

## 第 26 章：SQL Server 优化清单
将前面所有章节总结为一套清单和最佳实践。本章的目标是让您能有一个地方快速回顾从本书其他部分学到的所有内容。

## 下载代码
您可以从 Apress 网站的源代码部分（[www.apress.com](http://www.apress.com)）下载本书使用的代码示例。大多数代码是纯 T-SQL，存储在 `.sql` 文件中，可以在任何 SQL Server T-SQL 编辑工具中打开和使用。有几个 `PowerShell` 脚本需要通过 `PowerShell` 命令行运行。
