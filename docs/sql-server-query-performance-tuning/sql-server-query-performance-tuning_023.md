# 第 1 章：SQL 查询性能调优

查询性能调优在当今的数据库应用程序中仍然至关重要。是的，硬件性能在不断提高。SQL Server 的升级——尤其是对有助于决定查询执行方式的优化器以及执行查询的查询引擎的升级——本身就带来了更好的性能。与此同时，SQL Server 实例被部署到虚拟机上，无论是本地还是托管环境，其硬件行为都无法得到保证。数据库正在走向平台即服务系统，例如 Amazon RDS 和 Windows Azure SQL Database。您仍然需要处理基本的数据库设计和代码生成。

简而言之，查询性能调优仍然是提高数据库管理系统性能的关键机制。查询性能调优的妙处在于，在许多情况下，对索引或 SQL 查询的微小改动就能以极低的成本使应用程序效率大幅提升。在这些情况下，性能的提升可能比使用速度稍快的 CPU 或稍好一点的优化器所带来的提升还要高出几个数量级。
