# SQL Server 优化检查清单

如果你已经阅读了本书前 25 章，那么你应该理解了性能优化涉及的主要方面。你也明白这是一项具有挑战性且持续进行的活动。

我希望在本章做的是提供一份性能监控检查清单，可以作为数据库开发人员和 `DBA` 在现场时的快速参考。其理念类似于可撕下的最佳实践卡片。本章并未涵盖所有内容，但它在一个地方总结了一些主要的调优活动，这些活动可以快速并显著地影响你的 `SQL Server` 系统的性能。我已将这些检查清单项目分类为以下几个部分：

- 数据库设计
- 配置设置
- 数据库管理
- 数据库备份
- 查询设计

每个部分都包含若干优化建议和技术。在适当的情况下，每个部分也交叉引用了本书中提供更详细信息的具体章节。
