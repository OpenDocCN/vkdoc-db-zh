# 总结

在本章中，我们使用 `IKM File-Hive to Oracle` 知识模块在 Oracle Data Integrator 中将 MongoDB 数据集成到 Oracle 数据库。

本章是本书的结尾。您已经了解了 MongoDB 开发的各个方面，例如使用 Java、PHP、Ruby 和 Node.js 访问 MongoDB。我们还讨论了将其他一些 NoSQL 数据库（Apache Cassandra 和 Couchbase）迁移到 MongoDB。我们学习了在 MongoDB 中使用 Java EE 框架 Spring Data 和 Kundera。我们还介绍了在 MongoDB 中使用 Hadoop 和 Oracle Data Integrator。
