# 报告与脚本相关术语

## `sqldx.sql`脚本

关于`sqldx.sql`脚本，它需要指定 CSV 目录、FCSV 目录、FHTML 目录、GCSV 目录、HTML 目录和日志目录。

它生成 CSV 格式文件和层次化列表，其报告格式取决于许可级别。

主执行后得到最终结果，可打包为 zip 文件，并支持导入 CSV 结果和查看 HTML DBA_HIST_SNAPSHOT 报告。

## `sqlhc.sql`脚本

关于`sqlhc.sql`脚本，它提供详细报告，包含并行按钮页面和计划统计按钮。

## 诊断报告

关于诊断报告，它包含 HTML 报告形式的信息和历史信息。
