## 偏斜度如何使你的执行时间变得不稳定

偏斜度是一个由于某种原因经常被忽视或被误解的话题。偏斜度是基于成本的优化器处理数据的一个重要部分。它会影响优化器决定做什么。它如此重要，以至于 Oracle 创建了诸如自适应游标共享这样的新功能，就是为了处理偏斜度带来的后果。

### 偏斜度

在本节中，我们将探讨什么是偏斜度，它是如何产生的，以及它如何影响你的执行计划。如果你每天都处理偏斜度，你可能会立即知道哪些情况可能产生偏斜数据。有时偏斜数据也可能意外出现。我们还将看看如何检测偏斜度，既可以从 SQL 层面，也可以通过 SQLT。
