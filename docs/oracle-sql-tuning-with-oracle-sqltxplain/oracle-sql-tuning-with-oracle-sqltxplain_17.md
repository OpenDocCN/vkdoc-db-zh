# 引言

本书旨在作为一本实用指南，介绍一个名为 `SQLTXPLAIN` 的宝贵工具，通常简称为 `SQLT`。你可能从未听说过它，但如果你的工作涉及 `Oracle` 调优，`SQLT` 将是你能找到的最有用的工具之一。最棒的是，它可以从 `Oracle` 免费获得。你需要做的就是学会如何使用它。

## 本书的由来

我担任 `DBA` 已超过二十年。在那段时间里，我处理过许许多多的调优问题，但直到我开始为 `Oracle` 工作时，我才了解到 `SQLT`。作为 `Oracle` 支持部门调优团队的一员，我每天都使用 `SQLT` 来解决客户最复杂的调优问题。我很快意识到我的经历并非个例。在 `Oracle` 之外，很少有 `DBA` 知道 `SQLT` 的存在。知道如何使用它的人就更少了。因此，本书应运而生。

## 不要购买本书

如果你正在寻找一本关于抽象调优理论或如何调优“原始”SQL 的书。本书是关于如何使用 `SQLT` 进行 `Oracle` SQL 调优的。所采用的方法完全实用，并使用大量示例来展示 `SQLT` 工具的实际操作。

## 请购买本书

如果你是一名开发人员或 `DBA`，并且参与处理 `Oracle` SQL 调优问题。无论你的系统多么复杂，或者你与数据之间隔着多少层技术，让你的查询高效运行才是关键所在。无论你是一名初入职场的初级 `DBA`，还是一名经验丰富、见多识广的老手，本书旨在向你展示一些完全实用、对你日常工作非常有用的东西。
