# H

## HADR_LOGCAPTURE_WAIT/HADR_WORK_QUEUE 等待类型
始终可用的可用性组、数据库镜像、同步提交/异步提交。

## HADR_SYNC_COMMIT 等待类型
添加列、始终可用的可用性组、`AO_Test` 表创建、仪表板、数据库副本组、数据修改、`DBMIRROR_SEND`、性能监视器计数器、主/辅助副本、复制模式、存储子系统、同步复制。

## 高可用性 (HA)/灾难恢复 (DR) 配置
高可用性/灾难恢复配置包括数据库备份、数据库镜像、`HADR_LOGCAPTURE_WAIT/HADR_WORK_QUEUE`、`HADR_SYNC_COMMIT`、`REDO_THREAD_PENDING_WORK`、测试机器、虚拟机配置。
