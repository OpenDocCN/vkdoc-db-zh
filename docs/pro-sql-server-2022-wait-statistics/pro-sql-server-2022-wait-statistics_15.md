# I, J, K

## 内存 OLTP 等待类型
内存 OLTP 等待类型包括检查点文件对 (CFPs)、缺点、隔离级别、共享内存数据库要求、饥饿问题、传统/内存优化表、事务日志更改、`WAIT_XTP_CKPT_CLOSE`、`WAIT_XTP_HOST_WAIT`、`WAIT_XTP_OFFLINE_CKPT_NEW_LOG`。

## Internet 信息服务 (IIS)
IIS。

## IO_COMPLETION 等待类型
数据库备份、查询操作、存储子系统、存储系统、`sys.dm_os_wait_stats` 动态管理视图 (DMV)、事务日志备份。

## IO 相关等待类型
IO 相关等待类型包括 `ASYNC_IO_COMPLETION` 等待类型、`ASYNC_NETWORK_IO`、`CMEMTHREAD`、组件、`IO_COMPLETION`、`LOGBUFFER/WRITELOG`、`RESOURCE_SEMAPHORE`、`RESOURCE_SEMAPHORE_QUERY_COMPILE`、`SLEEP_BPOOL_FLUSH`、`WRITE_COMPLETION`。
