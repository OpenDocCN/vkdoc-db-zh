# 总结

窗口函数是基于一个“窗口”或一组行，逐行计算出值的函数。

窗口函数包括：

*   聚合函数
    包括所有主要的非窗口聚合函数，如 `count()` 和 `sum()`。
*   排名函数和分组
    包括 `row_number()`、`rank()` 和 `dense_rank()` 用于生成位置，以及 `ntile()` 用于生成有序分组。
*   从其他行获取数据的函数
    包括 `lag()` 和 `lead()`。

### 窗口子句

窗口函数具有一个 `OVER()` 子句：

```sql
fn() OVER (...)
```

`OVER()` 子句包含以下内容：

*   `ORDER BY`：用于定义数据的行顺序。
*   `PARTITION BY`：用于定义数据的子组。
*   一个帧子句：用于确定数据是按行号还是按值定义。它还决定了窗口的起始行和结束行。
