# 数据仓库

数据仓库是存放 **OLAP**（联机分析处理）数据结构的经典存储库。数十年的经验被用于开发极其高效的数据仓库软件，几乎每个主流数据平台都有其专有的数据仓库解决方案。

诸如微软的 **Azure Synapse**、亚马逊 **RedShift** 和谷歌 **BigQuery** 等解决方案，是作为数据仓库解决方案的范例。它们旨在与其他这些厂商提供的数据存储组件协同使用。

使用这些技术的好处在于，它们是**预先构建、开箱即用**的，并且针对分析数据进行了优化。这些系统的缺点是，它们涉及学习一种新的架构、新的脚本语言，并需要获得足够的专业知识以随时间推移进行维护。大型组织通常能够在这方面进行专门化投入，但即便如此，有时为了降低复杂性，人们可能不愿意一开始就选择这些方案。
