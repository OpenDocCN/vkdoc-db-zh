# 第三方分析软件

无论是在本地还是云端，都有许多分析软件解决方案可供使用。即使只是列出这些解决方案，也需要花费数小时、占用许多页篇幅，而且很可能还不完整。

与数据仓库解决方案类似，第三方分析解决方案的主要好处是，它是为特定目的而构建的，并且可以用相对最少的努力来使用。与数据仓库解决方案不同，分析软件在结构、接口和使用方式上差异很大。一个组织的数据分析需求将决定所需解决方案的类型，有些分析软件能满足这些需求，而有些则不能。

使用第三方供应商的缺点是**成本**和**供应商锁定**。成本通常会随着数据量和/或使用量的增加而上升。因此，一个分析项目可能开始时成本不高，但当数据量增长到初始规模的十倍、百倍甚至百万倍时，最终可能变得相当昂贵。在架构任何数据解决方案时，务必根据当前和未来的需求确定成本。这可以确保当数据及其使用量不可避免地呈数量级增长时，不会出现意外情况。

供应商锁定也可能带来挑战。将数据导入分析解决方案很容易，但将来是否有完善且便捷的方式将数据提取出来？如果没有，那么迁离该解决方案可能变得成本高昂且耗时。如果这些成本超过了留在该平台上所需的费用，那么组织可能不得不无奈地选择无限期地将其数据保留在那里。不过，对于那些习惯并熟悉特定数据分析解决方案的人来说，这可能不是问题。
