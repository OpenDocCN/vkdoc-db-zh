# 引言

使用 `T-SQL` 可以让您相对快速地编写代码并查看结果。创建 `T-SQL` 语句时也有很大的灵活性。在许多情况下，可以通过多种方式从 `T-SQL` 中获得相同的结果。本书面向对 `T-SQL` 有基本了解但希望提高整体代码质量的数据库开发人员和数据专业人士。在开始阅读本书之前，您应该理解 `T-SQL` 语法并知道如何编写 `SELECT`、`INSERT`、`UPDATE` 或 `DELETE` 语句。《Pro T-SQL 2019》将帮助您编写一致且性能更佳的代码。您还将学习如何通过使用源控制和改进数据库部署流程来保护您的 `T-SQL` 代码。总的来说，本书的目标是为您提供一个编写更好 `T-SQL` 代码的框架。作为数据专业人士，我们可能会遇到需求高或截止日期短的情况。《Pro T-SQL 2019》的编写旨在帮助您编写出未来能节省时间和精力的代码。

本书分为四个部分。第一部分介绍如何提高 `T-SQL` 代码的可读性。概述了 `T-SQL` 的各种数据类型，并提供了如何最佳使用这些数据类型的指导。这第一部分解释了 `SQL Server` 中各种数据库对象的优缺点。其他章节讨论了 `T-SQL` 代码的标准化和设计。第二部分解释了如何编写高效的 `T-SQL` 代码。本部分包括使用基于集合的设计来编写 `T-SQL` 代码，以及理解硬件与 `T-SQL` 设计之间的关系。您还将学习如何使用执行计划和 `SQL Server 2019` 中的新功能来提高 `T-SQL` 代码的性能。第三部分讨论如何管理您的 `T-SQL` 代码。本部分包含的章节涉及制定编码标准和使用源控制来存储您的代码。为了进一步管理您的 `T-SQL` 代码，您还将学习一些测试和部署数据库代码的方法。第四部分探讨如何编写能够长期保持可持续性的 `T-SQL` 代码。这些章节包括安全添加新功能的方法、记录数据库内数据更改的方法以及随时间推移管理数据增长的方法。
