# 致谢

首先，最重要的是，我要感谢我的家人在我撰写本书期间给予的支持。他们在我撰写本书第一版时已经有过一些这方面的经验，但撰写本书的第二版仍然是一项重大的任务。

我要感谢 Apress，特别是 Jonathan 和 Jill，感谢他们帮助我撰写本书。当 Jonathan 问我是否有兴趣更新第一版时，我只能回答“愿意！”从第一版到本书出版的这四年里，SQL Server 发生了如此多的变化，能够更新本书以反映这些变化，对我来说意义重大。

非常感谢技术编辑 Eelco Drost 和 Borbala Toth-Apathy。他们提供的评论帮助澄清了本书中的许多方面，并大大提高了本书的质量。

最后，向 SQL Server 社区致以崇高的敬意。你们是一个无与伦比的技术社区，我很荣幸能成为其中的一员。
