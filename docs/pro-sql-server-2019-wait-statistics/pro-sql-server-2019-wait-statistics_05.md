# 第二部分：等待类型

-   第 `5` 章：“与 CPU 相关的等待类型”介绍了与 CPU 相关的等待类型。

-   第 `6` 章：“与 IO 相关的等待类型”描述了与 IO 相关的等待类型。

-   第 `7` 章：“与备份相关的等待类型”介绍了与备份事件相关的等待类型。

-   第 `8` 章：“与锁相关的等待类型”首先简要介绍了 SQL Server 中的锁定和阻塞，然后深入探讨与锁相关的等待类型。

-   第 `9` 章：“与闩锁相关的等待类型”首先深入探讨了闩锁，描述了它们是什么以及在 SQL Server 内部如何工作。介绍之后，我们将准备查看不同的与闩锁相关的等待类型。

-   第 `10` 章：“高可用性和灾难恢复等待类型”描述了与 SQL Server 中可用的各种 `HA` 和 `DR` 配置相关的等待类型。

-   第 `11` 章：“抢占式等待类型”介绍了与 SQL Server 的操作系统有直接关系的不同等待类型。

-   第 `12` 章：“后台和杂项等待类型”包含了由 `SQLOS` 作为后台进程生成的各种等待类型。本章还将描述无法归入前述类别的等待类型。

-   第 `13` 章：“内存中 OLTP 相关的等待类型”描述了与 SQL Server 2014 中发布的 `内存中 OLTP` 功能相关的一些等待类型。

-   附录 I：“SQL Server 示例机器配置”描述了本书中我使用的虚拟机的配置。

-   附录 II：“自旋锁”解释了所谓的轻量级同步原语——自旋锁的工作原理，以及它们可能对您的 SQL Server 配置产生的影响。

-   附录 III：“闩锁类”包含了 SQL Server 内部大部分不同闩锁类的列表。该列表结合了在线手册的信息以及关于特定闩锁类的补充信息。
