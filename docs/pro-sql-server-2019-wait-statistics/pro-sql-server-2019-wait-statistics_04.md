# 第一部分：等待统计分析基础

-   第 `1` 章：“等待统计内部机制”首先简要介绍了 SQL Server 等待统计的历史，并探讨了 `SQLOS` 架构。由于等待统计与您系统的处理器密切相关，我们将详细讨论调度器、任务和工作线程。

-   第 `2` 章：“查询 SQL Server 等待统计”介绍了通过使用 `DMVs`、扩展事件和 `Perfmon` 访问 SQL Server 数据库引擎内部等待统计信息的各种方法。

-   第 `3` 章：“查询存储”探讨了 SQL Server 2016 中引入的一项新功能——`查询存储`。由于 `查询存储` 对您执行性能分析的方式可能产生巨大影响，我专门用一章来介绍如何使用这个出色的新功能。

-   第 `4` 章：“建立可靠的基线”涵盖了为性能故障排查建立和使用基线的重要性。基线在分析等待统计时尤其重要，我将向您展示如何为等待统计分析创建一个基线的示例。
