# 高水位线

`高水位线`是段中已用空间与未用空间的分界。已使用的块位于高水位线之下，因此，未使用的块位于高水位线之上。高水位线之上的块从未被使用或初始化。

在正常情况下，需要空间的操作（例如 `INSERT` 语句）仅当在高水位线之下没有更多可用空间时才会提升高水位线。一个常见的例外是直接插入操作，因为它们专门使用高水位线之上的块（参见第 11 章）。

释放空间的操作（例如 `DELETE` 语句）不会降低高水位线。它们只是将空间释放出来供其他操作使用。如果以等于或低于空间重用的速率释放空闲空间，那么使用高水位线之下的块应该是最优的。否则，高水位线之下的空闲空间会稳步增加。长期来看，这不仅会导致段大小不必要地增大，还会导致性能不达最优。实际上，全表扫描会访问高水位线之下的所有块。即使这些块是空的，扫描也会发生。要解决此问题，应对段进行重组。
