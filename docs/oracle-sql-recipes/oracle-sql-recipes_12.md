# 本书结构

我们将本书分为五个部分，每个部分涵盖为 Oracle 数据库编写 SQL 的一个广泛方面。

**第 1 部分** 讲述 Oracle SQL 的基础知识以及 SQL 各个方面的基础内容。

**第 2 部分** 介绍 Oracle 数据类型的复杂性，以及管理和操作不同形式数据的解决方案。

**第 3 部分** 讨论如何将 Oracle 作为开发环境使用，并介绍了帮助开发人员和数据库管理员解决复杂 SQL 任务的解决方案。

**第 4 部分** 是一个特殊主题的纲要，范围从 XML 到直接从数据库生成动态报表网页。

**第 5 部分** 面向数据库管理员，展示了如何应用 SQL 解决管理员日常面临的许多问题和任务。
