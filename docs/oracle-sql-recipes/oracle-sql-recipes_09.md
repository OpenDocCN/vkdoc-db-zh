# 关于作者

## 关于作者

- **格兰特·艾伦** 在 IT 领域工作了 20 年，最近担任一家领先的澳大利亚软件供应商的首席技术官，之后在谷歌担任现任职务。他在全球范围内的行业、政府和学术界工作过，为大规模系统设计、开发、性能、技术创新和颠覆性变革提供咨询。格兰特是会议和行业活动的常客，经常就数据挖掘、协作技术、关系数据库以及技术商业等主题发表演讲。他现在是谷歌的一名团队负责人，使用数据库技术解决谷歌规模乃至更大规模的问题。

- **鲍勃·布莱拉** 是 Oracle 9i、10g 和 11g 认证专家，拥有超过 20 年的数据库设计、数据库应用开发、数据库管理和培训经验。鲍勃是位于威斯康星州道奇维尔的 Lands' End 公司的互联网数据库分析师和 Oracle DBA。他撰写了几本面向新手和资深专业人士的 Oracle DBA 书籍。

- **达尔·库恩** 是一名 Oracle DBA 和开发人员。他合著了另外三本书：*《Linux 食谱：面向 Oracle DBA》*、*《RMAN 食谱：Oracle 数据库 11g》* 和 *《Oracle RMAN 口袋参考》*。他还在雷吉斯大学教授高级数据库课程，并为落基山 Oracle 用户组提供志愿 DBA 工作。他拥有科罗拉多州立大学的研究生学位，目前与妻子海蒂以及女儿布兰迪和丽莎居住在科罗拉多州莫里西奥峡谷附近。

## 关于技术审校者

- **斯蒂芬·法鲁尔** 于 1983 年首次接触关系数据库和 SQL 语言。他加入了早期的 Oracle 法国公司（在 IBM 短暂工作和渥太华大学任教一段时间之后），并很快对性能和调优主题产生了兴趣。1988 年离开 Oracle 后，他短暂尝试过改革并做了一些运筹学研究，但一年后他又重拾关系数据库。自那时起，斯蒂芬一直持续从事数据库咨询工作，不懈地改进查询，并试图向开发人员传授良好的 SQL 规范。他是 *《The Art of SQL》* 和 *《Refactoring SQL Applications》* 的作者。
