# Oracle SQL 食谱：问题解决方法

通过本书，您将学习 Oracle 中 SQL 的诸多精妙之处，以及可供您调用的强大功能和特性，以解决从统计建模、多级财务报表到甚至仅使用 SQL 直接从数据库动态生成网页等多样化的问题。

我们处理的每一个 Oracle SQL 问题都以问题/解决方案的格式呈现。首先，我们陈述一个待解决的问题。然后，我们展示用 SQL 编写的解决方案。最后，我们会详细讨论您需要了解的任何注意事项，以及该解决方案为何有效。

*Oracle SQL 食谱* 适合任何 SQL 熟练程度的读者阅读，其中包含的食谱能够拓展从初学者到资深专家的 SQL 爱好者的技能。

无论您是 Oracle 数据库中数据的用户、应用程序开发人员还是 DBA，本书都有满足您需求的内容。我们非常高兴能为您呈现这本食谱集。作为作者，我们撰写本书是为了突显 SQL 在 Oracle 中的强大功能和能力，以及它如何能够解决许多通常被认为在应用程序代码之外难以编写的问题。掌握了本书中的知识，您将精通最新、最强大的 Oracle 数据库功能，并能迅速即兴创作您自己的食谱。我们希望这些食谱能契合您在日常使用 Oracle 和 SQL 时遇到的许多现实世界问题，并期待看到您从本书灵感中创造出的食谱。
