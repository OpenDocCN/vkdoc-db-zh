## 前言

欢迎阅读《Oracle SQL 食谱》。您手中这本书包含了针对编写针对 Oracle 数据库运行的 SQL 时遇到的常见问题的现成解决方案。我们为那些时间紧迫、需要解决特定问题然后继续工作的人编写了本书。每个食谱都针对一个特定问题，并提供了针对该问题的可靠、可行的解决方案。对于感兴趣的人，每个食谱还提供了对解决方案的扩展讨论，有时还包括替代解决方案。

我们将这些食谱精心设计，旨在将您提升到美食大厨的高度——至少在 Oracle SQL 方面是如此。尽管我们将食谱按逻辑章节划分，涵盖了数据类型或报表生成等主题，但您可以随意按任何顺序、以任何组合方式采样和使用这些食谱。
