# 本章目标

本章旨在让你熟练管理容器/可插拔数据库环境。学完本章，你应该理解如何创建可插拔环境、配置 PDB、在 PDB 内连接和导航，以及将 PDB 从一个 CDB 转移到另一个 CDB。这一切的基础始于理解可插拔架构。图 22-1 可能过于简化，但展示了包含 CDB 和 PDB 的数据库整体视图。它提供了多租户架构的基本视图。

![../images/214899_3_En_22_Chapter/214899_3_En_22_Fig1_HTML.jpg](img/214899_3_En_22_Fig1_HTML.jpg)

**图 22-1** Oracle 多租户概览，推荐一个 CDB 配一个 PDB

> **注意**：Oracle 多租户是企业版可用的额外付费选项。但是，所有版本都允许使用一个可插拔数据库。即使不打算使用该选项或创建更多 PDB，也建议创建一个 CDB 和一个 PDB。图 22-1 显示了即使不使用该选项，架构也会是这种样子。这样设置数据库将便于将来使用多租户选项而无需迁移。详情请查看 Oracle 许可指南。
