# 引言

云计算的新时代已经到来。“云”的含义不再仅仅表示连接到互联网。如今，像 Oracle 这样的公司正在为各组织提供完整的资源套件和工具，使其能够构建自己的计算基础设施，而无需购买成百上千的硬件设备、无数机架以及一支小型工程师团队来安装、配置和管理构成其客户所用应用程序托管基础设施的众多组件。

Oracle 提供的资源之一就是 `MySQL Database Service`——一个完全托管的 `MySQL` 服务，您可以用作数据库后端。是的，您不再需要自己安装和配置 `MySQL`！现在，只需点击几下鼠标，您就可以在 `Oracle Cloud Infrastructure (OCI)` 中创建一个 `MySQL` 数据库服务器。最重要的是，您可以根据业务需求配置资源，例如将系统调整为使用最小的计算、内存和磁盘空间，从而降低您的开销成本。`MySQL Database Service (MDS)` 是全球领先的开源数据库服务器悠久传统中的一个全新篇章。

## 目标读者

我撰写本书是为了分享我对 `MySQL` 的热情，并延续我很幸运参与其中的 `MySQL` 传统。我特别想展示的是，即使您没有系统工程背景，任何人都可以在 `OCI` 中使用 `MDS`。`OCI` 确实非常易于使用。因此，目标读者包括任何有兴趣学习如何在云中利用 `MySQL` 的人，例如爱好者和发烧友，更重要的是，还包括希望了解 `MDS` 如何以比本地解决方案显著更低的成本扩展其基础设施的系统工程师和 IT 规划者。

## 本书结构

本书旨在引导读者从了解 `MySQL` 和 `OCI` 的基本知识，到在 `OCI` 中创建 `MySQL` 服务器并将其连接到应用程序。以下是本书各章节的简要概述：

*   第 1 章，“开始使用云端的 MySQL”：本章介绍 `Oracle Cloud Infrastructure` 和 `MySQL Database Service Oracle`。它还概述了 `Cloud Infrastructure` 以及您将使用的资源，以便您能充分利用 `MySQL Database Service`。
*   第 2 章，“Oracle Cloud Infrastructure”：本章更深入地介绍 `OCI`，并包含对其部分功能的浏览。您还将了解如何开始使用一个特殊的免费账户来学习使用 `OCI`。
*   第 3 章，“MySQL 简明教程”：本章是关于 `MySQL` 的教程。您将学习使用 `MySQL` 的基础知识，包括其主要功能概述，以及如何使用基本、常用的结构化查询语言 (`SQL`) 命令执行基本的数据库操作。
*   第 4 章，“MySQL Database Service”：本章通过学习如何设置和使用 `MDS` 数据库系统（包括通过云控制台简要浏览 `MDS` 服务和一个 `DB System`），更深入地探究 `OCI`。
*   第 5 章，“备份与恢复”：本章讨论 `MDS` 的恢复功能，包括如何备份数据库（数据）以及在需要时如何恢复。本章还讨论了将备份和恢复纳入您恢复计划的重要性。
*   第 6 章，“时间点恢复”：本章讨论 `MDS` 中的一个高级恢复功能，称为时间点恢复，它允许您将数据恢复到五分钟内的时间窗口。对于数据可能频繁变更或希望确保在数据因人为错误或系统故障受损时能进行最新恢复的用户来说，此功能使使用 `MDS` 更加安全。
*   第 7 章，“数据导入与导出”：本章介绍几种将数据迁移到 `MDS` 或从 `MDS` 迁出数据的方法，从数据导入和导出的概念、策略和工具开始。
*   第 8 章，“高可用性”：本章介绍 `MDS` 的一个高级功能——创建具有容错能力从而实现高可用性的系统的能力。您将了解什么是高可用性以及如何在 `MDS` 中实现它。本章以一个关于高可用性的简明教程开始。
*   第 9 章，“OCI 命令行和应用程序编程接口”：本章介绍一种通过 `OCI` 提供的命令行和应用程序编程接口来访问和交互 `OCI` 及 `MDS` 的替代机制。本章概述了用于处理 `OCI` 和 `MDS` 的命令行接口 (`CLI`) 和应用程序编程接口 (`API`) 的功能，并提供了每个接口的演示。
*   第 10 章，“迁移到 MDS”：本章适用于那些计划将其基础设施迁移到 `OCI` 的用户。本章提出了规划并将其现有的 `MySQL` 安装迁移到 `MDS` 的策略，以及一些更深入的主题，例如获取有关 `MDS` 功能的更多细节或在出现问题时的故障排除技巧。本章将为您开始规划和设计使用 `MDS` 的 `MySQL` 基础设施做好准备。

## 如何使用本书

本书旨在引导您学习更多关于 `OCI` 和 `MDS` 的知识，发现两者的强大功能，并学习如何构建您自己的 `OCI` `MDS` 解决方案。

如果您有 `OCI` 经验，可以跳过那些介绍 `MDS` 所需的最简化 `OCI` 技术的教程部分。同样，如果您有 `MySQL` 经验，可以跳过关于 `MySQL` 的介绍性章节，专注于如何开始使用 `MDS`。无论哪种方式，对于有兴趣在 `OCI` 云中利用 `MySQL` 的每个人来说，本书都有可取之处。
