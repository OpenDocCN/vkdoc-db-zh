# SQLite 已成为许多数据建模项目中的关键组件，因为它如此轻量且灵活。你可以快速搭建一个具备基本功能的 SQLite 数据库。这正是本书的意义所在。如果你手头有数据，并且有使命要基于这些数据构建点什么——无论是网站、移动应用，还是独立软件产品——SQLite 都会是称心如意的选择。

SQLite 可以在多种操作系统上运行。使用 SQLite，你操作的是一个数据库，可以将其从一个平台迁移到另一个平台，从一个开发环境迁移到另一个开发环境。你的原型设计、实验和开发工作可以持续推进，除了关系模型和 SQLite，无需承诺绑定于任何特定平台。你未来可能转向其他关系型数据库工具，而你这个小小的 SQLite 应用也可能演变为面向众多设备或面向单一的多线程企业数据中心的生产级应用。

这种灵活性和极低的入门门槛，是包括我在内的许多人觉得 SQLite 富有吸引力的两大特点。

我希望你会发现本书有所裨益。关于 SQLite 的更多信息可在 sqlite.org 找到，而我的网站 northcountryconsulting.com 上提供了更多信息（以及本书的可下载示例）。

—— 杰西·费勒
