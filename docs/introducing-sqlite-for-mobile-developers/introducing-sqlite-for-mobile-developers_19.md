# 第 3 章 使用 SQLite 基础：存储和检索数据

`SQLite` 是一个轻量级库，包含一个数据库引擎，允许你使用 `SQL` 的一个非常大的子集来存储和检索数据。作为用户，你最熟悉的可能是应用程序界面或数据库管理系统 (`DBMS`) 的界面，该界面提供了对数据库本身的访问。在幕后，你的命令（无论是文本还是图形）都被转换为 `SQLite` 语法。

本章向你展示创建表、向其中添加数据以及检索这些数据的基础知识。这是一个高级概述，让你体验一下 `SQLite`。下一章将深入探讨你可以使用的一些选项和变体。在本章中，你将看到图形用户界面 (`GUI`) 和 `sqlite3` 命令行界面，它们都是 `SQLite` 的一部分。在后续章节中，通常只会看到 `SQLite` 界面，因为它包含了 `SQLite` 命令。在大多数 `SQLite` 的图形编辑器中，你都有直接输入 `SQLite` 代码的选项，因此这些命令在任何一种环境中都应该适用。

## 术语和定义

`SQLite` 在本书中以多种方式使用，正如它在其它文档和讨论中一样。以下是 `SQLite` 世界中各项内容的指南：

-   `SQLite`。如本书所用，`SQLite` 指代任何版本的 SQLite。你可以在 `sqlite.org` 找到源代码和预编译的二进制文件。单独使用时，它指的是 `SQLite` 项目及其任何版本。
-   `SQLite 3` 指的是撰写本文时的当前 `SQLite` 版本。因此，`SQLite 3` 是 `SQLite` 的第三个主要版本。在本书中，`SQLite 3` 指代该版本，以区别于其他版本（例如 `SQLite 2`）。
-   `SQLite 3.9.2` 是 `SQLite` 当前的发布版本。与任何维护中的软件产品一样，它会根据需要进行修订，因此使用主版本号-次版本号-构建号的编号约定。在本书中，不指代具体的发布版本（`3.9.2`），因为它可能会发生变化。
-   `sqlite3`（无空格，全小写）是一个命令行实用程序，用于帮助你构建和调试 `SQLite` 数据库及语法。你可以从 `sqlite.php` 下载它，但如果你的操作系统 (`OS`) 或其他环境中已安装了 `SQLite`，`sqlite3` 很可能已经存在。

当你阅读关于数据库和关系数据库的各种材料时，可能会遇到数据库管理系统 (`DBMSs`) 和关系数据库管理系统 (`RDBMSs`)。当今大多数数据库都是关系数据库，而且几乎都基于 `SQL`。也有其他类型的数据库，但目前使用并不广泛。当今一个特别有趣的数据管理方法是处理非结构化数据，但这超出了本书的范围。

## 使用 `sqlite3`

`sqlite3` 是一个命令行实用程序，是 `SQLite 3` 的一部分（`sqlite2` 版本已不再适用）。它在 `OS X` 上运行（使用终端），或者作为 `sqlite3.exe` 在 `Windows` 上运行。
