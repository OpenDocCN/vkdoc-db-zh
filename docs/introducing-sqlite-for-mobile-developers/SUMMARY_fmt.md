## 摘要

SQLite 已成为许多数据建模项目中的关键组件，因为它如此轻量且灵活。你可以快速搭建一个具备基本功能的 SQLite 数据库。这正是本书的意义所在。如果你手头有数据，并且有使命要基于这些数据构建点什么——无论是网站、移动应用，还是独立软件产品——SQLite 都会是称心如意的选择。
