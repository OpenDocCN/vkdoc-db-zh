变化的事情太多了。通过建立和配置审计，我能够看到是谁在更改什么。真是松了一口气！不再有“为什么会发生这种事？”而我却无法提供任何线索的情况。我可以对他们说：“我在数据库中没有看到任何更改，或者这个用户在 5 号修改了架构。”

最终，我的公司被一家监管更严格的公司收购了，他们需要系统经过审计，并每周审查合适的报告。我的新公司对我已经完成了这些基础工作感到非常满意，这使我们能够轻松地进行审计。最初，我公司的管理层并不认为这是优先事项，但我在其他项目间隙坚持做这件事，并且正好构建了新母公司所需的东西，尽管我事先并不知道他们会需要这个。我所做的并不是什么魔法，我也无法预知未来。我只是遵循了基本的审计原则。这些原则得到了回报！

#### 审计类型

你需要进行的审计类型将取决于你所从事的业务。无论你身处何种行业，审计都会对你有所帮助。以下列出了你可能遇到的审计类型：

**内部审计** – 这种类型的审计由企业自身发起。它帮助企业维持可能并非外部审计师或政府法规所要求的标准。即使没有被要求进行更正式的审计，你可能仍希望进行此类审计的原因，是为了确保你的业务尽可能顺畅高效地运行。内部审计也可用于确保你已为外部审计做好准备。

**外部审计** – 这种类型的审计由第三方进行，例如政府监管机构或会计师。这些外部审计师将熟悉你需要遵守的各类规则和法规，并会确保你严格遵循它们。

**税务审计** – 在美国，美国国税局（IRS）有时会审计个人或企业的纳税申报表，以确保准确性，确认你没有多缴或少缴税款。这类审计往往是随机选取的，但某些纳税事项可能会增加你被审计的可能性。

**财务审计** – 这种类型的审计由外部审计师进行，以确保财务报表和/或工资支付的准确性。

**合规审计** – 这种类型的审计可由内部或外部审计师进行，旨在确保企业符合内部或外部的审计标准。这可以细分为监管合规和公司合规。公司合规是指内部规则和准则，而监管合规是指遵守政府法律和法规。

**运营审计** – 这种类型的审计通常由内部审计师进行，但在某些情况下也可能由外部审计师执行。此项审计旨在帮助确定改进业务运营的方法。

**信息系统审计** – 这种类型的审计可由内部或外部审计师进行，以确保系统向用户提供准确的数据，并确保系统安全，防止未经授权的用户访问或篡改。
