**提示** 确保在你的 S3 bucket 上配置生命周期管理。这样你就无需为永远保存审计数据而付费。更多信息，请访问[`docs.aws.amazon.com/AmazonS3/latest/userguide/object-lifecycle-mgmt.html`](https://docs.aws.amazon.com/AmazonS3/latest/userguide/object-lifecycle-mgmt.html)。

## 创建 Option group

你无法向默认的`option group`添加选项。你需要创建一个新的`option group`并将其应用于数据库实例。
