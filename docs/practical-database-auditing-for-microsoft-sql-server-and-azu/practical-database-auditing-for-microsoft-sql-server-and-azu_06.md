## 为何审计至关重要

*审计* 是对账目或记录的正式检查与核实。很多时候人们指的是财务审计，但审计的应用领域远不止于金融。在美国，当人们听到“审计”时，第一反应可能是美国国税局（IRS）。几乎每个人如果接到 IRS 的电话，都会感到紧张不安，因为他们要被审计了。我经历过一次，当时审计员说：“等等，我们认为我们欠你钱！”当你被审计时，通常的预设是它最终会让你付出代价。

还有一种审计是内部或外部审计员找你谈话，以了解你是否遵循了正确的流程。这些也可能很吓人，难怪审计可能名声不太好。但审计本身并非坏事。我甚至会说，它是件好事。你希望确保人们遵循正确的流程。无论是 IRS 还是你公司的审计员，总有一些规则、法规、流程和法律需要遵守。有时，需要有一些后果来确保合规。

审计的目的不是为了惩罚或找人麻烦，至少本书讨论的数据库审计并非如此。它的目的是揭示可能存在的问题，或那些你可能需要更严格遵守准则的领域。

### 你为何应该进行审计？

有些公司可能不重视审计，而有些公司则因法规要求必须进行审计。即使你的公司没有被要求审计，这也不应阻止你进行审计。审计是有价值的：它让那些有责任了解情况的人能够掌握情况。
