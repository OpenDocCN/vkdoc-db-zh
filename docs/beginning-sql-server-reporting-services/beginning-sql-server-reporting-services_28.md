# 总结

`SQL Server Reporting Services`是一个非常值得学习的主题。许多人以报表开发人员的身份开启其信息技术（`IT`）生涯。对于许多职业（无论是`IT`还是其他领域），报表开发被视为一项理想的技能。我经常与从事非`IT`角色（如人力资源）的人交流，他们为了工作需要学习`T-SQL`和`SSRS`。在我曾经工作的`IT`部门，我曾被要求为我们的服务台工单系统和网络管理工具创建报表。即使在`IT`内部，我们也需要查看来自自身系统的数据。

本书是对流行的报表工具`SQL Server Reporting Services`的入门介绍。本章回顾了一些高级主题，你可以将其视为下一步学习的内容。我希望你喜欢学习`SSRS`。我当然也很享受撰写它的过程！
