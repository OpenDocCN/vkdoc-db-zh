# 其他开发者云服务功能

开发者云服务还具有其他功能，包括：

*   **Wiki**：用于记录各种文档的普通 Wiki 功能。
*   **代码片段**：存储有用代码片段的能力，可以仅供自己私有使用，也可以与项目上的其他开发人员共享。
*   **敏捷开发**：在根据敏捷方法进行开发时有用的一些看板和报告。
*   **构建**：一个构建服务器，您可以在其中定义作业以自动构建和测试代码。提供使用 Ant、Maven、SQL、Shell 脚本和其他工具的各种触发点和构建步骤。
*   **部署**：定义部署配置的能力，以便您可以自动将代码部署到 Oracle Java Cloud Service 等。
