# 面板网格布局示例

如果你查看快速入门示例的源代码，你会发现其中许多都使用了`<af:panelGridLayout>`组件，其中包含`<af:gridRow>`和`<af:gridCell>`组件。

面板网格布局组件只是一个具有一些属性的容器，其布局主要由内部的行和单元格控制。

在最简单的布局（称为“单列（拉伸）”）中，你只会得到一行和一个单元格。该行的高度属性设置为`100%`，这意味着该行将拉伸以占据所有垂直空间。在该行内部，单元格的宽度属性设置为`100%`，这意味着该单元格将拉伸以占据所有水平空间。该单元格的`halign`（水平对齐）和`valign`（垂直对齐）属性也都设置为“拉伸”。这意味着该单元格将尝试拉伸其内部的组件。这些组件和设置的综合效果是，布局将占据浏览器窗口中的所有可用空间，并将其用于其内容。
