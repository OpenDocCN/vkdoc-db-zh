这些技术的一个非常有用的特性是自动页修复。当 SQL Server 检测到数据页已损坏时，它会从另一台服务器用该页的新鲜副本替换此损坏的页。这是一个异步过程，访问了损坏的页并触发页修复的查询可能会被中断并收到错误，直到该页在后台完成修复。

**注意** 您可以在 `http://technet.microsoft.com/en-us/library/bb677167.aspx` 阅读更多关于自动页修复的信息。
