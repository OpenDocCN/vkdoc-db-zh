# 第 31 章 ■ 备份与还原

`重要提示` 管理工具 (Management Studio) 中的用户界面默认使用 `RECOVERY` 选项。在管理工具中使用数据库还原界面时，请始终注意此设置。意外使用 `RECOVERY` 选项将导致你需要从头开始重复还原过程，这对于大型数据库来说可能非常耗时。更安全的方法是始终使用 T-SQL `RESTORE` 命令并全程使用 `NORECOVERY` 选项来还原所有备份。最后，你可以使用 `RESTORE DATABASE WITH RECOVERY` 命令（如代码清单 31-4 所示）来恢复数据库并使其联机。

我们将在本章后面讨论如何在灾难后恢复数据库。现在，让我们介绍一些在还原过程中可以使用的有用选项。
