# 前言

当今的现代商业环境充满活力且瞬息万变——数据在其中扮演着基础性角色。组织需要快速从数据中获得洞察，以改善客户体验并赋能员工。目前，许多公司在混合环境中运营，数据同时存在于云端和本地。这就是为什么我们对即将发布的 SQL Server 2022 如此兴奋，它是我们不断演进的数据与智能战略的关键部分。在微软，我们对 SQL 进行了重大投资，通过这款新产品创建了有史以来连接最紧密的 SQL 数据库——并将其整合到微软数据生态系统中，为组织提供了一种无缝连接其数据库、分析和治理平台的方式。

我认为，没有人比我们的首席架构师、拥有 29 年微软工龄的 Bob Ward 更有资格带您深入探索 SQL Server 2022。Bob 的知识深度仅次于他对 SQL 社区的热情和激情。

在这本信息丰富的书中，Bob 详细介绍了使您能够利用数据做更多事情并指导您利用新功能的新解决方案。Bob 将介绍适用于 SQL Server 2022 的 Azure Synapse Link，它能比以往更快、更轻松地将数据转化为洞察。

Bob 将涵盖数据库引擎的创新，包括内置查询智能、SQL Server 的账本以及包含可用性组，以及适用于 SQL 的 Azure Synapse Link，它结合了本地和云数据。这使得能够对您最重要的运营数据进行近乎实时的分析，而无需构建和管理复杂的包和管道——也无需为 ETL 流动编写一行代码。您将了解用于托管灾难恢复的 SQL 托管实例，并利用内置分布式可用性组（DAG）的功能来播种和链接数据库到 Azure。

此外，Bob 分享了关于 Microsoft Purview 的易于理解的信息，您可以用它来创建访问权限策略，然后在多台 SQL Server 上发布，让您拥有一个集中治理中心。
