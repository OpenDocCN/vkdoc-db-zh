# 5. 内置查询智能变得更强大

在第 4 章中，你了解了内置查询智能背后的历史及其承诺：无需更改代码即可获得更快的性能。你了解了 SQL Server 2022 的新功能，如新的查询存储、内存授权反馈持久化等。这些都不需要更改为 SQL Server 2022 的新数据库兼容级别 160。

如果你想获得第 4 章中读到的所有内容，甚至更多呢？如果你将数据库兼容级别更改为 160，你肯定会获得更多。这包括以下内容：

*   参数敏感计划优化
*   基数估计模型反馈
*   并行度反馈

在本章中，你将了解每一项功能，包括我们要解决的问题、它们的工作原理以及如何使用它们的示例。这个故事始于试图将世界从参数嗅探中拯救出来。
