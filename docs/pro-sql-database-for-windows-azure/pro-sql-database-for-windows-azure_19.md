### 镜像

如图 2-12 所示的**镜像**模式是卸载模式的一种变体，其中次要消费者可以是一个外部实体。此外，该模式意味着存在双向复制拓扑，以便任一数据库中的更改都能复制回另一个数据库。这种模式允许一种*无共享*的集成，即任一消费者都无权直接连接到另一个消费者。
