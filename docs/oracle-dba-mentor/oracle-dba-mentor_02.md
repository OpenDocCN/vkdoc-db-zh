# 致谢

我将我的第一本书 *Oracle RAC 性能调优* 献给了我已故的父亲。在撰写那本书时，我正在为另外几本书拟定提纲。其中一个想法就是创作这本为数据库管理员提供指导建议的书。市面上有大量书籍教授丰富的入门知识，但除了让您购买另一本书之外，没有一本书教您如何自学 Oracle。我希望这本书不仅仅是关于入门信息。我希望赋予读者能够掌控自己职业生涯并朝着他们期望的方向成长的工具。Oracle 真应用集群要求 DBA 具备高级技能。同样，Oracle 性能调优也需要高级技能。将两者结合起来，真正要求您精通自己的技艺。我的第一本书是为经验丰富的 Oracle 老手准备的，而将书献给我生命中经验丰富的影响者——我的父亲，再合适不过了。这本书更偏向入门级别，因此，我将其献给我的三个儿子，Chay、Jace 和 Jenner。父母最重要的角色莫过于指导他们的孩子成长为有责任感的成年人，并能在这个世界上找到自己的道路。我们无法教会他们成功所需的一切知识，但我们可以赋予他们独立解决问题的技能。将此书献给我的儿子们似乎是自然而然的事。我在他们人生的早期阶段指导他们，这类似于我尝试在专业层面上指导读者。

我要感谢 Apress 的 Jill Balzano 和 Jonathan Gennick 为本书付出的辛勤工作。我还要感谢 Apress 所有幕后工作人员。

最好的书籍都有出色的技术审校者来确保作者的诚实。感谢 Mike Donovan 的辛勤工作。他让我思考了一些原稿中没有的想法，这使得本书成为了更好的产品。

我要感谢任何阅读我作品的人，无论是本书、我为网站或博客撰写的文章，还是我在许多不同论坛上对问题的回答。正如您将在本书后面了解到的，我写了很多东西来助力提升我的职业生涯。您需要阅读更多才能理解。如果没有人阅读我的作品，我就不会写那么多，我的职业成长也会受到影响。所以，感谢阅读。这对我的帮助很大，我希望它也能帮助到您。
