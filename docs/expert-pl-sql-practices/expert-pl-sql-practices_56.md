# 注释

在代码中插入注释有两种方式：以 `--` 开头的单行注释，或用 `/*` 和 `*/` 包裹的整段注释块。我决定使用 `--` 语法。我认为它能很好地突显文本本身，因此你无需寻找注释的起始和结束位置。此外，它还具有一些技术优势：即使代码块中包含一些注释（以 `--` 开头），你仍然可以使用 `/*`... `*/` 来为调试目的而注释掉整块代码，而如果代码本身使用的是 `/*`... `*/` 语法，则无法做到这一点。它稍微繁琐一些，但可能非常方便。由你选择。有任何疑问吗？

以下是我如何在代码块中添加注释的示例：

```
l_book_author := NULL; -- 我确保此变量中没有旧值

IF l_book_id < 0 THEN
  -- 除非书籍丢失，否则不应发生此情况。
  -- 继续检查丢失书籍列表。
  ...
END IF;
```

![images](img/square.jpg) **注意** 尽可能多地使用注释！今天对你显而易见的事情，明天对其他人（甚至对你自己）来说可能难以理解。

### 比较运算符

使用 `<>` 而不是 `!=`。`<>` 语法可以避免在快速阅读时与 `=` 混淆。
