# 技术审稿人介绍

## Chris Beck
![images](img/square.jpg) `Chris Beck` 拥有罗格斯大学的计算机科学学位，并使用多种 DBMS 超过 20 年。过去 16 年他一直在 Oracle 公司工作，目前是一名专注于核心数据库技术的首席技术专家。他是两项美国专利的共同发明人，这些软件方法学是现在被称为 Oracle Application Express 的基础。Chris 曾审阅过其他 Oracle 书籍，包括 Tom Kyte 的《`Expert One-On-One`》（Peer Information Inc., 2001）和《`Expert Oracle Database Architecture`》（Apress, 2005），他本人也是两本书的合著者：《`Beginning Oracle Programming`》（Wrox Press, 2005）和《`Mastering Oracle PL/SQL`》（Apress, 2005）。他与妻子 Marta 和四个孩子居住在北弗吉尼亚州；不与家人共度时光时，他通常要么在浪费时间玩电子游戏，要么在看意甲足球。

## Mike Gangler
![images](img/square.jpg) `Mike Gangler` 是一名数据库专家和基础设施架构师，在数据处理行业拥有超过 30 年的经验，主要在大型企业信息系统项目中担任数据库技术负责人、系统分析师和 DBA。作为底特律 Oracle 用户组和安娜堡 Oracle 用户组的前任主席，以及国际 Oracle 用户组董事会的创始成员，Mike 作为一位成就卓著且经过认证的 Oracle DBA 和关系数据库专家获得了全球认可。

## Toon Koppelaars
![images](img/square.jpg) `Toon Koppelaars` 是 Oracle 技术的长期用户，自 1987 年（Oracle 版本 4）起就开始使用 Oracle 数据库和工具软件。在此期间，他一直参与应用开发和数据库管理。他是一名 ACE 总监（数据库开发），并经常在 Oracle 相关活动上演讲。他也是“赫尔辛基宣言（IT 版）”的提出者，该宣言描述了一种以数据库为中心的现代应用开发方法。他与 Lex de Haan 合著了《`Applied Mathematics for Database Professionals`》（Apress, 2007）。
