# 作者介绍

## Michael Rosenblum
![images](img/square.jpg) `Michael Rosenblum` 是 Dulcian, Inc. 的软件架构师/开发 DBA，负责系统调优和应用架构。Michael 通过编写复杂的 PL/SQL 例程和研究新功能来支持 Dulcian 的开发人员。他是《`PL/SQL for Dummies`》（Wiley Press, 2006）的合著者，并撰写了多篇与数据库相关的文章（IOUG Select Journal, ODTUG Tech Journal）。Michael 是 Oracle ACE，经常在各种地区和国家 Oracle 用户组会议（Oracle OpenWorld, ODTUG, IOUG Collaborate, RMOUG, NYOUG 等）上演讲，并获得了 ODTUG Kaleidoscope 2009 年最佳演讲者奖。在他的祖国乌克兰，他获得了乌克兰总统奖学金、信息系统硕士学位，并以优异成绩从基辅国立经济大学毕业。

## Robyn Sands
![images](img/square.jpg) `Robyn Sands` 是思科系统公司的软件工程师，为思科客户设计和开发嵌入式 Oracle 数据库产品。她从 1996 年起就开始使用 Oracle，在应用开发、大型系统实施和性能测量方面拥有丰富的经验。Robyn 的职业生涯始于工业和质量工程领域，她将之前的教育和经验与对数据的热爱相结合，不断寻找构建具有稳定性能和最低维护要求的数据库系统的新方法。她是 OakTable Network 的成员，并合著了两本关于 Oracle 的书籍：`Expert Oracle Practices` 和 `Pro Oracle SQL`（均为 Apress, 2010）。Robyn 偶尔会在 [`adhdocddba.blogspot.com`](http://adhdocddba.blogspot.com) 上发布一些随机博客。

## Riyaj Shamsudeen
![images](img/square.jpg) `Riyaj Shamsudeen` 是 OraInternals 的首席数据库管理员兼总裁，这是一家专注于性能/恢复/EBS11i 的咨询公司。他专精于真正应用集群、性能调优和数据库内部机制。他也经常在其博客 [`orainternals.wordpress.com`](http://orainternals.wordpress.com) 上就这些技术领域发表文章。他还是许多国际会议（如 HOTSOS, COLLABORATE, RMOUG, SIOUG, UKOUG 等）的常驻演讲者。他是 OakTable Network 的骄傲成员。他在使用 Oracle 技术产品方面拥有超过 16 年的经验，并有超过 15 年担任 Oracle/Oracle 应用数据库管理员的经验。
