# 致谢

你手中所持的这本书，无论是电子版还是印刷版，对所有参与者来说都倾注了相当多的心血。在 Exadata 平台上进行敏捷开发，在许多方面既是幸事也是诅咒——是幸事，因为你能欣赏每个版本带来的改进；是诅咒，因为新功能需要写入书中，增加了更多工作。……这个项目是我参与过的最漫长的项目之一，我要（再次！）感谢我的家人，允许我投入大量时间进行研究和写作，结果证明这段时间相当漫长。我保证会弥补！亲身参与组织和写作工作收获巨大，因为这让我得以深入探究 Exadata 实现的精髓。或许唯有在教学和写作中，你才能对所涉及的主题获得最透彻的理解。多少次我以为自己知道要写什么，结果却发现其实一无所知。但值得庆幸的是，我并非孤军奋战。没有同事和朋友们的支持，我不可能完成这一切，他们（有时甚至是不经意间）给予了我灵感。人数太多，无法在此页一一列举——我相信你读到这段时，明白我指的是谁。向你们所有人致以诚挚的“感谢”。

—马丁·巴赫

首先，我要感谢第一版的作者们，为我们提供了出色的原始素材。致克里、兰德尔和坦内尔——对于我们无数次听闻第一版如何精彩的反馈，希望我们没有辜负其声誉。这至少可以说是一段漫长的旅程。与马丁、弗里茨和卡尔全程合作非常愉快。正如弗里茨和马丁所言，这占用了大量本用于其他优先事项（主要是陪伴家人）的时间。我由衷感谢他们愿意让我在无数个深夜闭门伏案，努力将脑海中的想法付诸笔端。这是一次富有启发性的经历，我在写作过程中学到了很多。要跟上这个不断变化的平台，在项目生命周期中免不了大量重写！我享受写作本书的时光，也希望能让你通过阅读本书学到新东西。

—安迪·科尔文

成为本书的作者对我而言是一个学习循环，因为这是我第一次真正参与写作，而不仅仅是“仅仅”评论他人的作品。我从撰写一章开始，本以为工作量不大，花费时间不多，但最终这一章扩展成了三章。当然，由于我曾担任前一版的技术审阅者，我自然也承担了所有非本人撰写章节的技术审阅工作。同时担任作者和审阅者意味着我为创作本书投入了海量时间。我要感谢我的家人，允许我花费无数小时进行写作、审阅、研究、测试等等。正如马丁所言，本书的很大一部分内容得益于同事和朋友们各种形式的协作。谢谢你们。

—弗里茨·胡格兰

首先，我要感谢我的父母丹尼斯和内妮塔，以及我的兄弟凯文。没有你们，我不会努力去做到最好。我爱你们。致阿拉奥和奥古斯丁家族，我的朋友们和挚爱们——感谢你们在我撰写章节时提供支持与欢乐时光，让我保持理智。致克里·奥斯本、维罗妮卡·斯蒂格斯和马丁·佩恩特——感谢你们一直相信我，并给予我所有有趣的挑战和难得的机会。致迪娜·萨隆加和 SQL*Wizard 家庭——感谢你们所有的指导和友谊，以及让我接触大量棘手的客户场景，帮助我在年轻时就成长为一名扎实的数据库管理员。那些充满乐趣的不眠之夜，我永不会忘记。特别感谢乔纳森·詹尼克和 Apress 团队，感谢你们所有的耐心和支持。是的，我们做到了！感谢所有在我研究中提供帮助的人，以及你们宝贵的见解。最后，我要表达我对与 Oracle 社区、各种会议、oracle-l 邮件列表、OakTable Network 以及 Oracle ACE 项目中志同道合者们进行的精彩交流的赞赏。感谢你们所有的启发、学习、分享的想法、友谊和帮助。伟大的想法建立在过去伟大思想和伟大想法之上。让我们始终保持高昂的社区精神。

—卡尔·阿拉奥
