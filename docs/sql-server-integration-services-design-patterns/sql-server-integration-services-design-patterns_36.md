# 关于作者

![9781484200834_unFM-01.jpg](img/9781484200834_unFM-01.jpg)

**Andy Leonard** 是一名 SSIS 培训师和顾问、SQL Server 数据库和 Integration Services 开发人员、SQL Server 数据仓库开发人员、社区导师、SQL Server 最有价值专家 (MVP)、`SQLBlog.com` 博主和工程师。他是 *Professional SQL Server 2005 Integration Services* 和 *SQL Server MVP Deep Dives* 的合著者。他的背景包括 Web 应用程序架构和开发、Visual Basic、ASP、SQL Server Integration Services (SSIS) 以及使用 SQL Server 2000、2005 和 2008 的数据仓库开发。

**Tim Mitchell** 是一名商业智能顾问、数据库开发人员、演讲者和培训师。他使用 SQL Server 已有八年多，主要涉及商业智能、ETL/SSIS、数据库开发和报告。他获得了多项行业认证，拥有德克萨斯农工大学 Commerce 分校的计算机科学学士学位，并且是 Microsoft SQL Server 最有价值专家 (MVP)。Tim 是位于德克萨斯州达拉斯地区的 Artis Consulting 公司的商业智能顾问。作为社区的活跃成员，Tim 曾在众多 SQL Saturday 活动、休斯顿科技节以及各种用户组和 PASS 虚拟分会场发表演讲。他是达拉斯北德克萨斯 SQL Server 用户组的董事会成员和演讲者，担任 PASS BI 虚拟分会场的联合主席，并且是 PASS 的活跃志愿者。Tim 是 `SQLServerCentral.com` 的作者和论坛贡献者，并在 `SQLShare.com` 上发布了数十个 SQL Server 培训视频。你可以访问他的网站和博客 `TimMitchell.net` 或在 Twitter 上关注他 `@Tim_Mitchell`。



**马特·马森** 是一位软件开发工程师，就职于 SQL Server 集成服务 (SSIS) 团队。马特参与了 SSIS 产品的诸多方面，包括升级、性能以及整体用户体验。他是微软技术会议的常客，并维护着 SSIS 团队博客 (`http://blogs.msdn.com/b/mattm/`)。2006 年加入微软之前，马特曾是多个商业智能报告与分析产品的开发者。他居住在魁北克省蒙特利尔市，与位于雷德蒙德的团队进行远程协作。

![9781484200834_unFM-02.jpg](img/9781484200834_unFM-02.jpg)

**杰西卡·M·莫斯** 是微软 SQL Server 商业智能领域知名的作家和演讲者。她为不同行业的公司创建了众多数据仓库和商业智能解决方案，并讲授有关集成服务、报表服务和分析服务的培训课程。在为一家大型服装零售商工作期间，杰西卡参与了 SQL Server 2005 TAP 计划，在那里她为集成服务制定了最佳实施实践。杰西卡为多家杂志、网站和书籍撰写了技术内容，并在 PASS 社区峰会、SharePoint 连接大会和 SQLTeach 国际会议等国际会议上发表演讲。作为发展用户间社区关系的坚定倡导者，杰西卡积极参与弗吉尼亚州中部的地方用户组和代码训练营活动。此外，她还通过 PASS 组织志愿贡献时间，帮助教育大众。

![9781484200834_unFM-03.jpg](img/9781484200834_unFM-03.jpg)

**米歇尔·阿福德** 是微软最有价值专家 (MVP)，也是 GoDaddy 公司数据平台团队的首席工程师。她在数据生命周期的所有阶段都有经验，从 OLTP 到数据仓库，从关系型数据到大数据。米歇尔专注于超大型数据库 (VLDB) 的开发和可扩展数据库架构，并热衷于数据库自动化和数据变现。她是 SQL Server 社区的活跃成员和频繁的演讲者，尤其以在 PASS 峰会上的演讲而闻名。米歇尔在 `SQLFool.com` 拥有一个非常受欢迎的博客，也可以在 Twitter 上通过 `@sqlfool.com` 找到她。
