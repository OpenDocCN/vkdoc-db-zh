# 必备背景知识

本书适用于熟悉数据库系统并希望学习如何在 APEX 中编写具有一定复杂度的 Web 应用的人。编写 APEX 页面所使用的许多技术都涉及以下数据库和 Web 设计语言与技术中的各种技能。

### SQL

你需要掌握的最重要的技能是编写 SQL 查询的能力。APEX 中的所有数据访问都通过 SQL 语句执行，并且许多属性的值都以某种方式涉及 SQL。你对 SQL 越熟练，你的报表和表单就可以越复杂。本书假设你对 SQL 已经比较熟悉。在大多数情况下，`Employee Demo` 应用使用的是相对简单的 SQL 语句，但偶尔我也会包含一些更复杂的语句（如外连接或嵌套查询）来说明可以实现什么。

### HTML

本书还假设你对 HTML 有基本的了解——特别是如何使用 `<p>`、`<b>`、`<a>` 和 `<img>` 等标签来格式化文本和显示图像。我将忽略 JavaScript 和 CSS 等高级特性。

### PL/SQL

APEX 使用 PL/SQL 来指定行为。PL/SQL 是 Oracle 的编程语言；其主要特点是嵌入了 SQL 语法，使得编写与数据库交互的存储过程变得容易。你应该具备基本的编程知识，尽管事先了解 PL/SQL 并非必需。本书在第 7 章介绍了 PL/SQL，并且只使用了该语言的基础特性。

### APEX

最后，本书不要求你有使用 APEX 的经验。尽管不实际使用 APEX 也能跟着本书学习，但这样做似乎没什么意义。因此，你应该获得一个 APEX 账户。获得账户最简单也是最好的方法是访问 `apex.oracle.com` 站点。由于我是从那里创建我的 `Employee Demo` 应用的，所以你应该会看到与本书中出现的相同的屏幕。
