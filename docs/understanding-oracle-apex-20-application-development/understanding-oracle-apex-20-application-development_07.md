# 本版更新内容

尽管 APEX 是一个成熟且功能丰富的应用，但它（并且一直是）一个进行中的作品。APEX 开发人员一直在不懈地改进系统，并且对 APEX 及其用户界面的功能进行重大修改也毫不迟疑。

本书的前一版写于 2015 年，当时 APEX 5 刚刚推出。从那时起，用户界面经历了一系列重大变化。事实上，当我最近重读我的 APEX 5 书籍时，我沮丧地发现它的截图和说明常常不准确，有时甚至毫无意义。许多 APEX 属性现在位于不同的位置，有时名称也不同。新的组件类型（特别是*表单*和*交互式网格*）取代了旧的组件，并且具有不同的功能。显然，是时候修订这本书了。

本书最重要的新增内容是关于表单和交互式网格的两个新章节。在 APEX 5 中，“表单”是一个静态内容区域，包含一套复杂配置的项和内置流程。现在 APEX 有一个专用的 `Form` 区域，它会自动为你生成预配置的项和流程，并提供一组易于理解的属性来管理它们。

交互式网格区域取代了 APEX 5 中的“表格窗体”区域。它们提供了与表格窗体相同的基本功能——即就地更新报表值的能力——但交互式网格更易于使用、功能更强大，并且具有更清晰的语义。事实上，本书的交互式网格章节（第 11 章）比旧的表格窗体章节短得多，因为交互式网格所需的解释要少得多。
