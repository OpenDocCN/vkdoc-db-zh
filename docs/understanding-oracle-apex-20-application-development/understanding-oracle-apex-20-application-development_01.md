# 引言

Application Express（也称为 APEX）是一个与 Oracle 数据库紧密耦合的 Web 应用程序。它有多种用途：你可以使用它的 `SQL Workshop` 工具来查询和修改数据库；你可以使用它的 `App Builder` 工具来创建你自己的、与数据库交互的 Web 应用程序；你还可以运行由你或他人创建的 Web 应用程序。

其中，应用程序构建器（App Builder）尤为有趣，因为它提供了一种简单且非传统的方式来构建网页。你无需直接为页面编写代码；相反，你可以从一组内置模板中选择。有用于整体页面的模板，也有用于你想在页面上放置的每种组件（例如报表、按钮等）的模板。每个模板都有一组属性，这些属性的值决定了每个组件在页面上的位置、外观和行为。你只需为你想要的组件选择模板并为其属性赋值，即可创建一个页面。

APEX 应用程序构建器会将每个组件的属性值保存在其自己的数据库中。当浏览器请求你的应用程序的某个页面时，APEX 服务器会从其数据库中提取与该页面相关的属性值，构建与这些值对应的 HTML 代码，并将该代码返回给浏览器。这个过程被称为 `渲染`（rendering）页面，而 APEX 被称为一个 `HTML 生成器`（HTML generator）。

使用像 APEX 这样的 HTML 生成器的优势在于，你可以在没有任何 HTML（或 CSS、JavaScript、PHP）知识的情况下构建网页。此外，由于 APEX 与 Oracle 数据库紧密耦合，它自动处理了数据库交互的复杂性。APEX 使得仅凭基本的 SQL 知识就能轻松创建美观、功能强大且能感知数据库的页面成为可能。
