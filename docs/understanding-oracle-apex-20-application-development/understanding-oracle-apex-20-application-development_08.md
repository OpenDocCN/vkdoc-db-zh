# 致谢

首先，我要感谢 APEX 用户社区。许多人定期、慷慨地通过撰写博客、创建 APEX 演示站点以及在 APEX 论坛上回答各种问题来分享他们的 APEX 知识。我从他们那里学到了很多。这本书是我尝试回馈的一种方式。

我还要感谢我的 Apress 编辑，Jonathan Gennick 和 Jill Balzano。Jonathan 说服我写这本书，然后又修订它。他在每一步都提供了指导和鼓励。Jill 总是给予支持，并抚平了道路上不可避免的坎坷。

最重要的是，我要感谢我的妻子 Amy。她陪伴我经历了所有三个版本，倾听我的想法，帮助我解决技术问题，研究 APEX 示例，并指出书中需要澄清的段落。她是一位不知疲倦的校对者。没有她，我也可以写这本书，但质量绝不会如此出色。谢谢。
