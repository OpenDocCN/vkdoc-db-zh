Microsoft SQL Server 2017 于 2017 年 10 月 2 日正式发布。这标志着它历史上第 12 个主要版本。SQL Server 现在是业界领先的数据平台。它为全球范围内的网站、行业和业务应用提供支持，从笔记本电脑、小型企业到大型企业服务器无处不在。它被用作私有云和公有云中的数据平台。数据库引擎是 Microsoft Azure SQL Database 背后的动力。多年来，SQL Server 一直以易用性、易管理性以及卓越的性价比而享有盛誉。但通过突破性的性能和可扩展性、值得信赖的安全性以及新的智能功能，SQL Server 现已成为技术领域的一股主要力量。像列存储索引、查询存储、Always Encrypted、图形数据库、Always On 可用性组和机器学习服务这样的特性表明，SQL Server 不仅仅是一个出色的关系数据库引擎，更是一个真正的应用程序数据平台。多年来，我们与 Windows Server 并行运行的辉煌历史是一个不可思议的故事。那么，我们为什么还要让 SQL Server 也能在 Linux 上运行呢？
