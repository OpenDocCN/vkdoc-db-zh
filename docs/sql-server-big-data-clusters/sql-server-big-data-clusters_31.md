## 更高级的数据框处理

到目前为止，我们已经研究了可以对数据框执行的——相对——简单的操作，例如选择特定行、对数据分组以及对数据框进行排序。

然而，当涉及到数据框时，在 Spark 中我们可以做的远不止于此，例如，将多个数据框连接在一起形成一个新的数据框。在本节中，我们将重点进行一些更高级的数据框处理。

首先，到目前为止，我们一直在处理导入到数据框中的单个数据集，该数据框包含美国各个机场的信息。在许多情况下，你并不会拥有一个包含所有所需内容的单一数据集，这意味着你最终会得到多个数据框。使用 PySpark，我们可以根据数据框共享的键将这些数据框连接起来，并构建一个新的、连接后的数据框。

在开始连接数据框之前，我们需要从正在使用的 2015 年航班延误和取消示例中导入其他示例数据集。如果你正在跟随本章中的示例操作，你应该已经拥有一个名为 `df_airports` 的数据框，其中包含 `airports.csv` 文件的数据。如果还没有，你可以运行以下代码（代码清单 6-16）将文件中的数据导入到数据框中。

```
df_airports = spark.read.format('csv').options(header='true', inferSchema="true").load('/Flight_Delays/airports.csv')
```
