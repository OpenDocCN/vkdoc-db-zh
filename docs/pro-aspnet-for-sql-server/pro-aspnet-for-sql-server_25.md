# 第五章

## SQL 提供程序

ASP.NET 2.0 的灵活性在很大程度上归功于提供程序模型的可插拔架构，它将 SQL Server、提供程序的 SQL 实现以及许多 ASP.NET 控件完美地连接起来。这些提供程序以及使用它们的控件不仅高度可配置，而且如果你选择根据独特需求从头开始构建自己的提供程序，它们也是完全可替换的。通过利用可用功能，让你的 ASP.NET 网站完全按照你需要的方式运行是相当容易的，这些功能为你提供了丰富的功能基础，让你能够带着一整套技巧开始上手。

本章涵盖以下内容：

*   `SqlMembershipProvider`
*   `SqlRoleProvider`
*   `SqlProfileProvider`
*   构建 SQL 相册提供程序
*   构建 SQL 站点地图提供程序

ASP.NET 2.0 中的 SQL 提供程序通过将领域模型与用户界面组件配对，为平台提供了极大的灵活性，这些组件可以以高度模块化的方式与其他领域和组件拼接在一起。ASP.NET 2.0 引入的大多数新控件都由提供程序支持，这些提供程序提供了超越 SQL Server 的多种实现。如果你想实现自己的提供程序，框架中有功能可以帮助你以最小的努力开始。你甚至可以设计自己的 SQL
