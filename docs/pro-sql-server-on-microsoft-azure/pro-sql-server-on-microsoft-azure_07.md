# 身份与访问管理

`Windows Azure Active Directory` 提供了强大且安全的身份服务，有助于在云中控制和管理目录及访问管理。它还提供无缝登录 Azure 资源的体验，并通过多因素认证增强安全性。

## Active Directory

Azure Active Directory (`AAD`) 是一种身份和访问管理解决方案，为组织提供了管理用户和组的能力，就像本地 Active Directory 解决方案一样。`AAD` 有助于为本地和云中的资源提供安全访问。`AAD` 提供三种服务层：免费版、基本版和高级版。

### 备份

Azure 提供了大量简单可靠的备份服务，可在需要时用于确保业务连续性和灾难恢复。

## Azure Site Recovery

`Site Recovery` 是一项 Azure 服务，通过协调将你的本地服务器和虚拟机复制到次要本地数据中心或 Azure，来为你的业务连续性和灾难恢复 (`BCDR`) 策略做出贡献。`Site Recovery` 处理复制过程，你可以通过简单点击启动故障转移和恢复。
