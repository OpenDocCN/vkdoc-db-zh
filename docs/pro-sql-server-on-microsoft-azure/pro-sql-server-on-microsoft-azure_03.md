# Azure SQL 数据库

Azure SQL 数据库是一种关系数据库即服务，它提供了 `RDBMS` 的所有重要特性，包括事务一致性、数据完整性和多用户并发数据访问系统。Azure SQL 数据库构建在与 `Microsoft SQL Server` 相同的原则之上，并提供可比的功能。对于已经熟悉 `SQL Server` 的用户来说，适应 Azure SQL 数据库不会很困难。Azure SQL 数据库采用 `PaaS` 服务模型，使用者控制访问模式，而平台则负责其他管理工作。Azure SQL 数据库为数据库提供自动备份和时间点恢复功能。根据所使用的服务层，Azure SQL 数据库还以合理的成本提供高可用性选项。
