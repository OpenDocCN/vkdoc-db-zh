# 字符串

字符串字面量用单引号括起来。一些数据库管理系统也允许使用双引号，但双引号更正确地应用于名称而非值。

*   在一些数据库管理系统和数据库中，大小写可能不匹配。
*   尾部空格应该被忽略，但并非总是如此。
*   嵌套单引号需要两个连续的单引号：`'Don''t do that'`。

### 日期

日期字面量也用单引号括起来。

*   首选日期格式是 ISO 8601 (`yyyy-mm-dd`)。
*   大多数数据库管理系统允许替代格式，但你应该避免 `??/??/yyyy` 格式。
*   日期按照历史顺序比较。

### 多个断言

你可以使用逻辑 `AND` 和 `OR` 运算符组合多个断言。如果组合使用，`AND` 的优先级高于 `OR`。

### `IN` 运算符

`IN` 运算符用于匹配一个列表中的值。它也可以与生成单个值列的子查询一起使用。

### 通配符匹配

可以使用通配符模式和 `LIKE` 运算符进行更宽松的字符串比较。

*   通配符包括特殊的模式字符。
*   一些数据库管理系统允许你将 `LIKE` 用于非字符串数据，隐式地将它们转换为字符串进行比较。
*   一些数据库管理系统用额外的字符补充了标准的通配符。
*   一些数据库管理系统支持正则表达式，这比常规的通配符模式匹配更复杂。
