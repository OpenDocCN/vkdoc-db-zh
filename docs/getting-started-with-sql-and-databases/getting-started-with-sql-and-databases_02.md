# 示例数据

我们将在本书后面更多地讨论示例数据，但本书是基于一个示例数据库编写的。如果你愿意，你可以坐在扶手椅上直接阅读本书，但你可能更愿意动手实践练习。

要完成练习，你需要以下条件：

*   一个数据库服务器和一个合适的数据库客户端。
*   在数据库上进行任何操作的权限。如果你是在本地安装的软件，你可能拥有所有需要的权限，但如果你是在别人的系统上操作，则需要确认。
*   用于生成示例数据库的脚本。

你可以从 [`https://sample-db.net`](https://sample-db.net) 获取该脚本的最新副本，并选择你的选项。
