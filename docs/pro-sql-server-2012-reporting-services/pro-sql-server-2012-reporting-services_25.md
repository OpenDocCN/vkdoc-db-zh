# 添加最终修饰

在许多项目中，最终的润色工作可能相当耗时。现在，你已将员工服务成本报告调整到符合预期，并在开发过程中应用了一定程度的格式。在将报告标记为完成并准备部署到你的 SSRS Web 服务器用于生产之前，只剩下几个最终元素需要应用：

*   添加页眉和页脚
*   添加报告标题
*   添加页码
*   添加报告执行时间

要为报告添加页眉和页脚，请选择“报告”菜单，然后选择“添加页眉”和“添加页脚”。这将向报告中添加两个新部分，你将在其中添加将打印在每一页上的值。

在工具箱中，你会看到两个报告项：文本框和线条，它们可以在页眉和页脚部分使用。首先，将两个文本框拖到页眉，一个拖到页脚。然后，将其中一个页眉文本框对齐到与表格相同的宽度，将你的报告标题输入为`Employee Service Cost by Diagnosis`，将字体大小更改为 16 磅，并应用粗体格式。接着，将一条线拖到页眉部分，并将其定位在你刚刚创建的图表和报告标题文本框之间。

在页眉部分的第二个文本框中，添加以下基于全局参数的表达式（在“编辑表达式”窗口中定义）：

```
="Page " & Globals!PageNumber & " of " & Globals!TotalPages
```

最后，在页脚的文本框中，添加以下用于报告执行时间的表达式：

```
=Globals!ExecutionTime
```

现在，你已准备好在将报告部署给用户之前最后一次预览它。这次，让我们在浏览器中查看最终版本（参见图 6-42）。这就是报告部署到 Web 服务器后的样子。（我们将在第 8 章中讨论部署报告的方法）。预览报告后，滚动到报告底部以查看页脚中的报告执行时间。所有最终润色完成的报告可以在 Pro_SSRS 项目中找到，名为`EmployeeServiceCost_Final.rdl`。
