# 前言

几十年来，SQL Server 一直是数据库行业的重要力量，服务于开发者、企业和小型企业。多年来，Windows Server 是 SQL Server 发展壮大的主要平台。2017 年，我们推出了在 Linux 操作系统上运行 SQL Server 的功能，这对我们和数据库社区来说都是一个改变游戏规则的举措。

SQL Server 对 Linux 的支持向我们的社区敞开了容器技术的大门。容器为可移植性、灵活性、效率和优化开启了新的可能性。SQL Server 现已成为这个世界和生态系统的一部分。随着对容器的支持而来的是 Kubernetes 的自然扩展，这是一个用于大规模部署和管理像 SQL Server 这样的容器化应用程序的系统。

本书由两位行业专家 Anthony Nocentino 和 Ben Weissman 撰写，是您学习 SQL Server 和 Kubernetes 的最佳资源。我曾与 Anthony 和 Ben 都有过合作，我想不出还有谁能比他们更适合撰写这个主题。

我喜欢这本书为读者精心设计的结构。您将从容器基础知识、Kubernetes 架构以及部署 `k8s` 的细节开始构建您的知识。（是的，您需要学会这样称呼它。谁愿意每次都输入全名呢？）掌握了 `k8s` 基础知识，例如如何使用著名的 `kubectl` 工具以及存储概念后，您将学习在 `k8s` 上部署、交互和监控 SQL Server 的所有方面。在每一章的学习旅程中，都有许多分步示例和图表，让您能够直观地确认自己在 SQL 和 `k8s` 上的实践体验。

本书最后提供了 SQL Server 容器在 `k8s` 上实际生产场景中的精彩示例，包括 Azure Arc 和 SQL Server 大数据集群（BDC）。SQL Server 无处不在，而学习 SQL Server 容器如何在 `k8s` 上运行，就是关于在全新平台、新挑战和新技能上的创新。这本书将帮助您迎接这些挑战，它由 SQL Server 和平台行业及社区中的两位专家撰写。
