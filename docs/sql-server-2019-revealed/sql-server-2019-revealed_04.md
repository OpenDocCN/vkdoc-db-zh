# 致谢

我生命中有太多值得感恩的事情，能够撰写本书是其中之一。我相信我所有的祝福都来自上帝，正是通过祂的恩典，我认识到了耶稣基督的强大信息。我首先要感谢我美丽而才华横溢的妻子 Ginger。她是我的伴侣和灵魂伴侣。她不厌其烦地听我所有的抱怨，看着我为这本书熬到深夜，有时不得不让她开车，这样我就可以在副驾驶座上处理这本书。我不知道有谁的信仰比我妻子 Ginger 更坚定，我非常幸运能与她分享美好的生活。今年我们庆祝了结婚 30 周年，我仍然像以往一样，尽可能每一天都与她在一起。我还要感谢我的儿子 Troy 和 Ryan。Troy 如今住在南卡罗来纳州的查尔斯顿，当飓风多里安逼近时，我在那里完成了本书的最后部分。Troy 是我钦佩的人，不仅因为他的品格，还因为他努力让这个世界变得更美好。Ryan 在贝勒法学院读第二年（加油熊队！）。他继续以他的智慧、正直、自信地掌控一切的能力，同时还能抽出时间提高高尔夫球技，这让我感到惊叹。我还要感谢我的母亲 Annette Gibaud，她继续为我树立榜样，教导我每天都要找到方法对他人友善。

有许多人帮助完成了这本书。我首先要感谢 Apress Media 给我另一个写书的机会。Jonathan Gennick 和 Jill Balzano 在每一步都帮助我推动这本书完成。如果没有我的技术审阅 Aaron Bertrand，这本书不可能按时完成。当我考虑写这本书时，Aaron 是我首先想到的审阅者之一，因为他对 SQL Server 的惊人知识和在社区中的专家声誉。Aaron 审阅每一章的速度简直是超人。

在微软方面，首先要感谢 Rohan Kumar、Gayle Sheppard 和 Asad Khan 给我机会传播 SQL Server 2019 的信息，这对本书的详细学习至关重要。我还要亲自感谢我最亲密的两位同事 Buck Woody 和 Anna Hoffman (Thomas)。我在 2018 年和 2019 年与 Buck 和 Anna 一起环游世界，讲述 SQL Server、大数据集群和 Azure 的故事。他们都让我成为更好的讲故事者和教师，并且在旅途中很有趣。微软 SQL Server 工程团队非常出色。能与如此聪明和专业的人一起工作，我感到敬畏，他们中的许多人帮助我提供了您在本书中找到的细节。这一切都要从 Slava Oks 和 Travis Wright 开始，他们帮助讲述了西雅图和 Aris 的故事，并且两人在推动这个版本的许多方面（包括大数据集群）发挥了重要作用。Conor Cunningham 继续以其对产品的深刻理解让我惊叹，同时在交付高质量版本方面发挥了关键作用。
