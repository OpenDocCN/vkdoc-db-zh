# 前言

我们确实正处于技术史上一个独特的转折点，对于从事数据、分析和 AI 领域工作的人来说，从未有过比现在更好的时机。数据的增长速度比以往任何时候都快，而通过 AI 和 ML 实现的数字颠覆为公司们将数据作为其业务竞争优势创造了无限可能。随着数字化的显著加速，我们现在面临的主要问题是如何利用这海量的数据来帮助我们的公司和社区实现转型。

我们看到了一个由智能云和智能边缘驱动的巨大机遇。SQL Server 在为开发人员、数据工程师和管理员在边缘、本地、私有云和公有云层面提供一致性方面，是业界无与伦比的。我们的 SQL Server 社区在这一演进过程中发挥了非常重要的作用，我无法充分感谢他们在过去 25 多年里给予的支持和反馈。

SQL Server 2019 是一个非凡的版本，我为团队所交付的成果感到自豪。SQL Server 2019 建立在 SQL Server 2016 和 SQL Server 2017 所交付的创新基础之上。虽然作为 SQL Server 的每个主要版本所预期的那样，有几项新功能将很好地服务我们的客户，但我最感到兴奋的是那些卓越的创新，它们扩展了客户在数十年间建立起来的管理和从大数据系统中获取洞察的技能。这些创新将在推动客户数字化转型方面发挥关键作用。

Bob Ward 自 SQL Server 团队早期就已加入，并对产品产生了显著影响。很少有人能像他那样拥有如此广泛和深入的理解，这体现在他设法在本书中以简单、易于理解的方式解释复杂的概念。希望您喜欢阅读本书。
