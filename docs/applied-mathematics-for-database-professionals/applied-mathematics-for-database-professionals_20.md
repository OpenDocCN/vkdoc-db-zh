#### 逻辑恒等式与性质

正如数值 `1` 是常规乘法的恒等元（任何给定数与 `1` 相乘结果仍是该给定数），`0` 是常规加法的恒等元（任何数与 `0` 相加结果仍是该数），逻辑中也存在类似的恒等式，如 `清单 3-2` 所示。

`清单 3-2. 恒等式`
```
P ∧ TRUE ⇔ P
P ∨ FALSE ⇔ P
P ∧ FALSE ⇔ FALSE
P ∨ TRUE ⇔ TRUE
```

第一个表达式表明 `TRUE` 是关于与运算（AND）的恒等元。第二个表达式表明 `FALSE` 是关于或运算（OR）的恒等元。最后两个表达式也被称为两个*有界性*恒等式；它们表明任何东西与 `FALSE` 进行与运算（ANDed）结果为 `FALSE`，任何东西与 `TRUE` 进行或运算（ORed）结果为 `TRUE`。

##### 交换律

一个运算符（或连接词）当且仅当它是二元运算符，且其左右操作数互换不影响结果时，才是*可交换的*。`清单 3-3` 展示了一些逻辑运算符的交换律。

`清单 3-3. 交换律`
```
( P ∧ Q ) ⇔ ( Q ∧ P )
( P ∨ Q ) ⇔ ( Q ∨ P )
( P ⇔ Q ) ⇔ ( Q ⇔ P )
```

合取（conjunction）、析取（disjunction）和等价（equivalence）都是可交换的连接词。

##### 结合律

如果你想对一系列操作数重复使用相同的二元运算符，并且处理该运算符出现的顺序无关紧要，那么这个运算符就是*可结合的*。请注意 `清单 3-4` 中命题形式里的括号。

`清单 3-4. 结合律`
```
( ( P ∧ Q ) ∧ R ) ⇔ ( P ∧ ( Q ∧ R ) )
( ( P ∨ Q ) ∨ R ) ⇔ ( P ∨ ( Q ∨ R ) )
```

合取和析取是可结合的。结合律解释了为什么在涉及两个以上操作数的迭代合取或析取表达式中，括号常常被省略。

##### 分配律

就像交换律和结合律一样，你可能还记得常规算术中的*分配律*，通常表达为如下公式（本节开头已展示）：
```
a*(b+c) = (a*b) + (a*c)
```
这个代数性质展示了加法和乘法之间的某种关系；在常规算术中，乘法对加法具有分配律。在逻辑中，合取与析取之间也存在类似的关系，如 `清单 3-5` 所示。

`清单 3-5. 分配律`
```
( P ∨ ( Q ∧ R ) ) ⇔ ( ( P ∨ Q ) ∧ ( P ∨ R ) )
( P ∧ ( Q ∨ R ) ) ⇔ ( ( P ∧ Q ) ∨ ( P ∧ R ) )
```

正如这些重言式所示，合取对析取具有分配律，反之亦然。请注意，在常规算术中，加法对乘法*不*具有分配律。

##### 自反性

常规等号是*自反*运算符的一个例子；对于任何 `x`（无论 `x` 的类型如何），`x = x` 都为 `TRUE`。在逻辑中，蕴含和等价连接词是自反的，因为 `清单 3-6` 所示的谓词始终为 `TRUE`。

`清单 3-6. 自反性`
```
P ⇔ P
P ⇒ P
```

##### 传递性

常规等号是*传递的*，如以下公式所示：`((x = y) ∧ (y = z)) ⇒ (x = z)`

在逻辑中，合取、蕴含和等价是传递的，如 `清单 3-7` 所示。

`清单 3-7. 传递性`
```
( ( P ∧ Q ) ∧ ( Q ∧ R ) ) ⇒ ( P ∧ R )
( ( P ⇒ Q ) ∧ ( Q ⇒ R ) ) ⇒ ( P ⇒ R )
( ( P ⇔ Q ) ∧ ( Q ⇔ R ) ) ⇒ ( P ⇔ R )
```

请注意，析取不具有传递性：`( P ∨ Q ) ∧ ( Q ∨ R )` 并*不*蕴含 `P ∨ R`。

##### 德·摩根定律

德·摩根定律（1806–1871）对于处理涉及否定的表达式非常有用，如 `清单 3-8` 所示。

`清单 3-8. 德·摩根定律`
```
¬( P ∨ Q ) ⇔ ( ¬P ∧ ¬Q )
¬( P ∧ Q ) ⇔ ( ¬P ∨ ¬Q )
```

用平实的英语描述，这些定律表明“析取的否定等于（操作数的）否定的合取”，反之亦然，“合取的否定等于否定的析取”。

##### 幂等性

一些（二元）运算符具有这样的性质：如果它们的运算对象相等，那么结果就等于这个运算对象。这些运算对象被称为在该运算符下是幂等的。在逻辑中，命题在析取和合取运算下是幂等的，如清单 3-9 所示。

**清单 3-9.** 幂等性

`( P ∧ P ) ⇔ P`
`( P ∨ P ) ⇔ P`

##### 双重否定（或对合律）

如果你应用两次否定，你应该会得到原来的运算对象。换句话说，否定运算起到补集的作用，如清单 3-10 所示。

**清单 3-10.** 双重否定

`¬¬P ⇔ P`

##### 吸收律

吸收律的性质不那么为人熟知，乍一看可能有点难以理解。它们列在清单 3-11 中。

**清单 3-11.** 吸收律（“P 吸收 Q”）

`( P ∨ ( P ∧ Q ) ) ⇔ P`
`( P ∧ ( P ∨ Q ) ) ⇔ P`

第一条吸收律说明，在两个谓词的析取式中（例如 `R1 ∨ R2`），如果其中一个析取项（例如右边的 `R2`）比另一个析取项（`R1`）更强，那么可以省略 `R2` 而不影响该谓词的真值。

**注意** 回想一下第 1 章“谓词强度”一节的内容，谓词 `R2` 比谓词 `R1` 强，当且仅当 `R2 ⇒ R1`。在第一条吸收律中，右边的析取项 `P ∧ Q` 总是比 `P`（左边的析取项）更强，因为谓词 `(P ∧ Q) ⇒ P` 是一个重言式。

类似地，第二条吸收律说明，在两个谓词的合取式中，例如 `R1 ∧ R2`，如果其中一个合取项（例如右边的 `R2`）比另一个合取项（`R1`）更弱，那么可以省略 `R2` 而不影响该谓词的真值。

这两个吸收等价式的证明留作本章末尾的练习。

#### 量词

第 1 章阐明了命题和谓词的区别。它讨论了如何将谓词转化为命题。有两种方法可以实现这个目标：
* 为谓词中的参数提供具体的值。
* 通过量化来绑定谓词中的参数。

第一种方法很直接，在第 1 章已有解释；本节解释第二种方法。

为了使用量化来实现这个目标，你必须首先定义谓词的参数（例如 `x`）允许从中选取值的集合。通过对这个可能值集合进行量化，你就可以说明*有多少*这些可能值具有某种特定属性（此时谓词被转化为命题）。例如，你可以说恰好有 42 个 `x` 的值具有某种属性。然而，两种最常见的量化陈述如下：
* 存在某个 `x` 的值，使得 [包含 `x` 的陈述句] 为真。
* 对于所有 `x` 的值，[包含 `x` 的陈述句] 为真。

第一个陈述说明了至少存在一个值；因此它被称为*存在*量化。第二个陈述被称为*全称*量化，因为你陈述了关于给定集合所有元素的性质。你可以用两个符号来表示存在和全称量词，如表 3-1 所示。

**表 3-1.** 存在量词和全称量词

| 符号 | 描述 |
| :--- | :--- |
| `∃` | 存在量词（存在...） |
| `∀` | 全称量词（对于所有...） |

令 `P(x)` 和 `Q(y)` 为给定的谓词，其参数分别为 `x` 和 `y`。令集合 `A` 表示参数 `x` 的可能取值集合，集合 `B` 为 `y` 的可能取值集合。使用量化的表达式的通用语法如清单 3-12 所示。

**清单 3-12.** 通用量词语法

`∃x∈A: P(x)`
`∀y∈B: Q(y)`
