# 第 2 章 ■ 集合论：导论

清单 2-8 展示了与幂集相关的两个特性。

#### 清单 2-8. 幂集特性

`( A ∈ ℘B ) ⇔ ( A ⊆ B )`

`( #A = n ) ⇔ ( #℘A = 2n )`

第一个逻辑等价关系可以直接从幂集的定义得出；第二个关系在你考虑到对于每一个可能的子集，`A` 的每个元素都可以被*选中*作为元素，或者被*排除在外*——也就是说，对于 `A` 的每个元素，你都有两种选择。因此，如果集合 `A` 包含 `n` 个元素，那么可能的子集总数就变成 `2n`（2 的 `n` 次方），两个极端子集是空集（`n` 次“排除”）和集合本身（`n` 次“选中”）。另请参阅前面包含三个元素（`{1,2,3}`）的集合 `V` 及其包含八个元素的幂集 `℘V` 的示例。

幂集的概念是本书第二部分将使用的两个关键基石之一，届时我们将把*数据库宇宙*的概念作为数据库设计的一个形式化定义引入。第二个关键概念是集合函数的*广义积*，将在第 4 章介绍。

#### 集合的并集

在下一节讨论划分的概念之前，我们首先介绍并集运算符的一个稍微更方便的形式。虽然并集运算符是*二元*的（即它接受两个操作数），但经常出现的情况是，你想要“并集”多个集合，而不仅仅是两个；例如：

`A1 ∪ A2 ∪ A3 ∪ ... ∪ An`

为了节省大量输入（每次都重复并集符号），我们为并集运算符引入一种特殊的*一元*表示法（只接受一个操作数），如定义 2-8 所示。

### ■定义 2-8: 一元并集运算符

如果 `A` 是一个由集合构成的集合，那么

`∪A := { x | x 是 A 的某个元素中的元素 }。`

你明白这个定义是做什么的吗？并集 `∪A` 的结果包含所有出现在某个集合（例如 `A` 的一个元素 `Ai`）中的元素 `x`。请记住，为了使此定义有效，集合 `A` 的每个元素都必须是一个集合。

如果你看下面的例子，你会发现这个“集合的并集”确实是一个比我们迄今为止使用的二元并集运算符更强大和通用的并集运算符，因为现在你可以用一个方便的简写符号来产生任意数量集合的并集。在这些例子中，`S` 和 `T` 代表一个集合：

`∪{S} = S`

`∪{S,T} = S∪T`

`∪∅ = ∅`

`∪℘S = S`

#### 划分

既然我们已经介绍了集合的并集，我们现在可以如定义 2-9 所示来定义*划分*。

### ■定义 2-9: 划分

如果 `S` 是一个非空集合，那么 `P` 是 `S` 的一个划分，当且仅当 `P` 是一个由 `S` 的相互不相交的非空子集构成的集合，并且这些子集的并集等于 `S`。

或者，用一种更形式化的方式描述，你可以使用下面的等式：
`(P 是 S 的一个划分 ) ⇔ ( P ⊆ ℘S–{∅} ) ∧`
`( 令 A,B∈P 则 (A≠B ⇒ (A∩B)=∅) ) ∧`
`( ∪P = S )`

这个定义说明了以下内容：

• `P` 是幂集 `℘S` 的一个真子集（因为*空集被排除在外*）。
