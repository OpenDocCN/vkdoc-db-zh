**11.** 员工编号为 1000 的员工在公司内具有特殊地位。看看下面关于该员工的数据库约束：
```
PODC8(OFFR,REG) :=
(∀o∈OFFR: (∃r∈REG: r↓{course,starts}=o↓{course,starts} ∧ r(stud)=1000)
⇒
( #{ r2 | r2∈REG ∧ r2↓{course,starts}=o↓{course,starts)} = 1
∧¬(∃o2∈OFFR: o2↓{loc,starts}=o↓{loc,starts} ∧ o2(course)≠o(course))
)
)
```
该约束规定，每当员工 1000 参加某个课程班时，则不允许其他学生参加此课程班，并且也不能有任何其他课程班（针对不同课程）在与该课程班相同的日期和地点开始。该约束实际上是两个约束的合取，其中一个是（伪装的）表约束，另一个是（更简单的）数据库约束。请运用本书第 1 部分介绍的各种重写规则，找出这两个约束是什么。

**a.** 年度奖金不得超过该员工工资等级的上限。

**b.** 员工只能因（活跃）工作满整年而获得奖金。

为 `BONUS` 表结构提供表征、元组宇宙和表宇宙，并（如果需要）修改 `DB_UEX` 宇宙。
