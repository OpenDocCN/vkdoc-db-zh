# 附录 B：符号

**表 B-1.** `与集合和函数相关的数学符号`

| 符号 | 描述 | 示例 |
| :--- | :--- | :--- |
| `:=` | 定义为 | `P1 := x > y` |
| `∈`, `∉` | （不）是...的元素 | `x ∈ A`, `y ∉ A` |
| `=`, `≠` | （不）等于 | |
| `≥`, `≤`, `<`, `>` | 其他比较运算符 | |
| `⊆` | 是...的子集 | `A ⊆ B` |
| `⊇` | 是...的超集 | `A ⊇ B` |
| `⊂` | 是...的真子集 | `A ⊂ B` |
| `⊃` | 是...的真超集 | `A ⊃ B` |
| `∪` | 并集 | `A ∪ B` |
| `∩` | 交集 | `A ∩ B` |
| `−` | 差集 | `A − B` |
| `÷` | 对称差 | `A ÷ B` |
| `×` | 笛卡尔积 | `A × B` |
| `⊗` | 自然连接 | `A ⊗ B` |
| `∅` | 空集 | |
| `( ; )` | 有序对 | `(EMPNO; 102)` |
| `◊` | 函数复合 | `f◊g` |
| `◊◊` | 属性重命名 | `T◊◊g` |
| `dom` | 函数的定义域 | `dom(F)` |
| `rng` | 函数的值域 | `rng(F)` |
| `π` | 有序对的第一坐标 | `π (a;b)` |
| `π` | 有序对的第二坐标 | `π (a;b)` |
| `N` | 自然数集 | |
| `Z` | 所有整数的集合 | |
| `↓` | 元组的限制 | `t↓{...}` |
| `⇓` | 表的投影 | `T⇓{...}` |
| `℘` | 集合的幂集 | `℘A`, `℘{...}` |
| `∏` | 集合函数的乘积 | `∏(F)` |
| `#` | 集合的基数 | `#A`, `#{...}` |
| `SUM` | 求和（f 在 A 上） | `SUMx∈A: f(x)` |
| `AVG` | 平均值（f 在 A 上） | `AVGx∈A: f(x)` |
| `MAX` | 最大值（f 在 A 上） | `MAXx∈A: f(x)` |
| `MIN` | 最小值（f 在 A 上） | `MINx∈A: f(x)` |
| `↵` | 选择运算符 | `↵S`, `↵{a}` |

**表 B-2.** `与逻辑相关的数学符号`

| 符号 | 描述 | 示例 |
| :--- | :--- | :--- |
| `t`, `T` | 真 (TRUE) | |
| `f`, `F` | 假 (FALSE) | |
| `∃` | 存在量词 | `∃x∈S: y < 42` |
| `∀` | 全称量词 | `∀y∈S: y ≥ 27` |
| `∧`, `∨`, `¬` | 与、或、非 | `P ∧ ( Q ∨ ¬R )` |
| `⇒` | 蕴含 | `P ⇒ Q` |
| `⇔` | 等价 | `P ⇔ Q` |
