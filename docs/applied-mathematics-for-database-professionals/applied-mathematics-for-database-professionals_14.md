# 第 1 章 逻辑学：导论

#### 特殊的谓词类别

有两个谓词类别值得拥有一个专门的名称：重言式和矛盾式。

##### 重言式与矛盾式

一个*重言式*是一种命题形式，对于命题变量的真值所有可能的组合，它都为真。一个*矛盾式*是一种命题形式，无论命题变量碰巧取什么真值，它都为假。重言式的示例如下：

`x = 42 ∨ x ≠ 42`
`10 = 10`
`P ∨ TRUE`
`P ⇒ ( P ∨ Q )`

第一个例子阐释了逻辑学中被称为“*排中律*”（*Tertium non datur*）的原理。第二个例子相当明显。第三个例子无论 `P` 的真值是什么都为真。你可以使用真值表来证明最后一个重言式，如表 1-11 所示。

**表 1-11.** *使用真值表证明一个重言式*

如你所见，表 1-11 的最后一列只包含真（TRUE）；这意味着，无论变量 `P` 和 `Q` 的个别真值如何，相应列标题中的谓词 `P ⇒ ( P ∨ Q )` 总是为真，因此是一个*重言式*。表 1-10（用与非表达非连接词）也展示了一个重言式的例子。

类似地，一个命题形式如果总是求值为假，而不管其组成部分的个别真值如何，则它是一个*矛盾式*。下列表达式是一个矛盾式：

`( P ∨ Q ) ∧ ( ( ¬P ) ∧ (¬Q ) )`

如果你为这个谓词建立一个真值表，你最终只会在相应的列中得到假（FALSE）。你是否认为这个表达式是矛盾式是显而易见的？也许这个例子会让你体会到，能够对逻辑表达式进行纯粹的形式分析，而不考虑所涉及的变量 `P` 和 `Q` 代表什么，是何等重要。

##### 假言推理与拒取式

*假言推理*（拉丁语：*肯定前件式*）和*拒取式*（拉丁语：*否定后件式*）可能是逻辑中最著名的重言式例子。在常规文本中，它们分别表述如下：

-   如果 `P` 蕴含 `Q` 且 `P` 为真，那么 `Q` 必须为真。
-   如果 `P` 蕴含 `Q` 且 `Q` 为假，那么 `P` 必须为假。

你也可以使用逻辑运算符符号来表达这两个重言式：
`( ( P ⇒ Q ) ∧ P ) ) ⇒ Q`
`( ( P ⇒ Q ) ∧ ( ¬Q ) ) ⇒ ¬P`

假言推理代表了日常推理最直接的形式；因此它也被称为*直接推理*。拒取式也被称为*间接推理*，这是一种远不那么熟悉的推理形式。看下面的例子。

令 `P` 代表谓词“员工 `e` 是经理”，令 `Q` 代表“员工 `e` 的月薪超过 1 万”。进一步假设你工作的公司有如下业务规则：“经理的薪水总是超过 1 万”，或者用符号表示为，`P ⇒ Q`。

间接推理允许你推断：如果你的月薪没有超过 1 万，那么你就不是经理。

##### 逻辑等价与重写规则

事实证明，本章介绍的五种逻辑连接词的集合是真值函数完备的。实际上，仅包含 `与`、`或` 和 `非` 这三个连接词的子集也是真值函数完备的（我们将在第 3 章讨论析取范式时再回到这一点）。这意味着你不需要另外两个连接词——蕴含和等价——因为你可以用 `非`、`与` 和 `或` 来表达它们：`( P ⇒ Q )` 可以表示为 `( ¬P ) ∨ Q`

`( P ⇔ Q )` 可以表示为 `( P ∧ Q ) ∨ ( ¬P ∧ ¬Q )`

我们将在“逻辑等价与重写规则”一节中研究（并证明）像前面这样的*等价*。尽管连接词集合 `{与, 或, 非}` 确实足以方便地表示所有可能的复合谓词，但你甚至可以用*单个*逻辑连接词达到相同的目标：`与非` 运算符（not and），也称为谢弗竖线，通常用竖线表示，如表 1-9 所定义。如你所见，`与非` 连接词仅当两个操作数都为真时返回假，在所有其他情况下返回真。

##### 重写规则

一条*重写规则*是一条规则，它允许我们将给定的命题形式 `X` 替换为另一个命题形式 `Y`，其方式保证无论命题变量的值如何，`X` 和 `Y` 都具有相同的真值。当且仅当等价式 `X ⇔ Y` 是一个重言式时，这种替换才是允许的。这个等价式被称为重写规则。

你已经在“函数完备性”一节中遇到了一个作为重言式的等价式。它展示了如何用析取和否定的组合来表达蕴含：

`( P ⇒ Q ) ⇔ ( ¬P ) ∨ Q`

表 1-12 所示的真值表证明了该等价式确实是一个重言式。

## 使用现有重写规则证明新规则

假设你想研究以下逻辑等价，看看它是否是一个重写规则：

`( P ⇒ Q ) ⇔ ( ( P ∧ ¬Q ) ⇒ FALSE )`

**■ 注意** 这条规则的效果是将命题变量 `Q` 移到蕴涵的左边，并在过程中否定它；右侧被简化为一个常量 `FALSE`。这种技巧类似于在算术中求解二次方程（`ax² + bx + c = 0`）的方式，事实证明，在实现非平凡约束时，这也是一种有用的技术，正如你将在本书第 11 章中看到的。

你可以使用真值表，如前所示，但你也可以使用现有的重写规则（例如表 1-13 中列出的规则）来证明这个逻辑等价实际上是一个重态。

表 1-14 展示了这样的证明可能的样子。这里我们利用了前面提到的、能够将蕴涵转换为析取（反之亦然）的重写规则：`( P ⇒ Q ) ⇔ (¬P ∨ Q )`

**表 1-14.** *用重写规则证明重写规则*

| **输入表达式** | **等价表达式** | **注释** |
| :--- | :--- | :--- |
| `( P ⇒ Q )` | ⇔ `( ( P ⇒ Q ) ∨ FALSE )` | 平凡；表 1-13 中的第二个特例 |

#### 章节小结

在继续下一节的练习之前，如果你对本章（关于逻辑的第一章）介绍的以下概念中的任何一个感到不熟悉，可能需要回顾一下本章的某些部分：

•  `值` 是一个具有确定含义的独立常量。

•  `变量` 是一个用于存放（某种表示的）值的容器。

•  `命题` 是一个非真即假的陈述句。

•  `谓词` 是一个带有参数的、返回真值的函数。

•  通过为参数提供值或用量词绑定参数，你可以将谓词转换为命题。

•  你可以通过将 `逻辑连接词` 应用于现有的谓词来构建复合谓词；本章介绍了 `否定`、`合取`、`析取`、`蕴含` 和 `等价`。

•  逻辑连接词可以被视为 `逻辑算子`；它们接受一个或多个（输入）谓词作为操作数，并返回另一个谓词作为输出。

•  复合谓词的输入谓词也称为该复合谓词的 `组成部分`。

•  所有逻辑连接词的精确含义都可以用 `真值表` 来定义，你可以使用真值表来研究复合谓词的真值。

•  `重言式` 是一个恒为真的命题，`矛盾式` 是一个恒为假的命题。

•  `重写规则` 是一个具有等价形式的重言式。

•  你可以使用重写规则来推导出新的重写规则，而无需使用真值表。

#### 集合与元素

集合论显然是关于集合的。在这里，我们遇到了第一个问题：集合`是`什么？集合是对象的汇集，这些对象被称为集合的 `元素`。在数学中，我们喜欢精确——所以我们希望为我们使用的所有术语提供正式的定义。就像在几何学中，我们都对 `点` 是什么有一个共同的理解，但提供一个点的形式定义却很困难。这就是为什么我们将它们声明为 `原始术语`；也就是说，它们只是给定的，你不需要定义它们。关于集合最好的定义可能就是定义 2-1 所给出的那个。

##### 定义 2-1：集合
集合由其互异的元素完全刻画，除了这些元素之外别无其他。

作为一个相当明显但重要的推论，没有两个不同的集合包含完全相同的元素；此外：

•  集合的元素没有任何顺序

•  集合不包含重复的元素

如果集合 A 的每个元素都是集合 B 的元素，并且集合 B 的每个元素也都是集合 A 的元素，那么这两个集合（比如 A 和 B）是相同的。

注意，集合的元素可以是任何类型的对象。它们甚至可以是集合本身；在本书的剩余部分，你将会看到大量包含集合作为元素的集合示例。

集合论有一个数学符号 `∈`，用于声明某个对象是某个集合的元素。下面的表达式声明对象 x 是集合 S 的一个元素：`x ∈ S`

这个命题的否定，即说 x 不是 S 的一个元素，表示如下：`x ∉ S`

后一个表达式是前一个表达式的否定简写：`x ∉ S ⇔ ¬(x ∈ S)`
