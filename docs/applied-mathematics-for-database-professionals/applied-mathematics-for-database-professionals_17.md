##### 定义 2-5：真子集 / 真超集

`A` 是 `B` 的**真子集**，`B` 是 `A` 的**真超集**（记法：`A ⊂ B`） ⇔ `A` 是 `B` 的子集**且** `A ≠ B`。

你可以使用斜线作为否定符号；例如，`A ⊄ B` 表示 `A` **不是** `B` 的真子集。除了这个带有内置否定的 `⊄` 符号外，你总是可以使用单独的否定符号来书写逻辑上等价的陈述，如下所示：`A ⊄ B ⇔ ¬( A ⊂ B )`

##### 代码清单 2-6：为真的子集命题示例

```
{1,2} ⊆ {1,2,3}
{1,2} ⊂ {1,2,3}
{1,2,3} ⊄ {1,2,3}
{1,2,3} ⊆ {1,2,3}
∅ ⊆ {1,2,3}
```

代码清单 2-6 的最后两个例子很有趣，因为它们说明了子集的以下两个重要性质：

- 每个集合都是其自身的子集：`A ⊆ A`
- 空集是每个集合的子集：`∅ ⊆ A`

这里值得提及的另外两个子集性质是：

- 传递性：`( ( A ⊆ B ) ∧ ( B ⊆ C ) ) ⇒ ( A ⊆ C )`
- 相等性：`( A = B ) ⇔ ( ( A ⊆ B ) ∧ ( B ⊆ A ) )`

通常，算子的一个重要性质是 `闭包性质`。如果一个算子在某个给定集合上的运算结果总是属于该集合的（另一个）元素，则称该算子在该集合上是封闭的。

### 集合运算符与空集 (Set Operators and the Empty Set)

作为另一种特殊情况，检查*空集*的行为总是一个好主意，因为我们希望它像任何其他集合一样运作。那么，让我们看看当空集作为操作数之一参与时，并集、交集和差集运算符的行为如何：

`A ∪ ∅ = { x | x∈A ∨ x∈∅ } = A`

`A ∩ ∅ = { x | x∈A ∧ x∈∅ } = ∅`

`A – ∅ = { x | x∈A ∧ x∉∅ } = A`

`∅ – A = { x | x∈∅ ∧ x∉A } = ∅`

`A ÷ ∅ = ( A–∅ ) ∪ ( ∅–A ) = A ∪ ∅ = A`

换句话说，与空集的并集不会增加任何东西。与空集的交集因没有共同元素而总是空集；差集和对称差集不会取出任何元素，因此它们也不会改变任何东西。

以下逻辑等价式也是一个重言式——也就是说，它是一个重写规则：

`(((A – B) = ∅) ∧ ((B – A) = ∅)) ⇔ (A = B)`

你可以分两步证明这个等价性，推理如下：
- 如果 `A – B` 是空集，那么你可以证明 `A` 是 `B` 的子集；
- 如果 `B – A` 也是空集，那么你也可以证明 `B` 是 `A` 的子集；
- 但那样的话，`A` 必须等于 `B`。

### 幂集与划分 (Powersets and Partitions)

幂集和划分是两个更高级的集合论概念。让我们从重要的幂集概念开始，如定义 2-7 所示。

#### **定义 2-7：幂集 (Powerset)**
一个给定集合 `S` 的幂集（符号：`℘S`）是由 `S` 的所有可能子集组成的集合。

`V := {1,2,3}`

`℘V = { ∅, {1},{2},{3}, {1,2},{1,3},{2,3}, {1,2,3} }`
