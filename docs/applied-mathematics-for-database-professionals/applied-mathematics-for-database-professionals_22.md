# 量化空集与嵌套量词

第二个命题——对空集的普遍量化——不太直观。将普遍量化视为迭代的“与”运算（AND），你会发现对空集的普遍量化不产生任何合取项。这样的量化根本不添加任何谓词，因此可以被认为是最弱的谓词，即为**真**（TRUE）（回顾第一章“真值表”小节中关于谓词强度的段落）。正如我们在“量词的否定”一节中讨论关于量词否定的一些额外重写规则时将看到的，这完全合理。表 3-2 列出了关于空集量化的这两个重要的重写规则。

**表 3-2. 空集量化——重写规则**

| 量化 | 求值结果 |
| :--- | :--- |
| `( ∃x∈∅: P(x) )` | **假**（FALSE），与谓词 `P(x)` 无关 |
| `( ∀y∈∅: Q(y) )` | **真**（TRUE），与谓词 `Q(y)` 无关 |

##### 嵌套量词

量词可以嵌套。也就是说，你可以在单个谓词中使用多个量词。先看一个例子。假设有以下两个集合：`S := {1,2,3,4}` 和 `T := {1,2}`。

现在看下面四个（非常相似的）命题：

- `(∀x∈S: (∃y∈S: y = x ))` （对于 `S` 中的所有 `x`，存在 `S` 中的一个 `y` 使得 `y = x`）
- `(∃y∈S: (∀x∈S: y = x ))` （存在 `S` 中的一个 `y`，使得对于 `S` 中的所有 `x` 都有 `y = x`）
- `(∀x∈S: (∃y∈T: y = x ))` （对于 `S` 中的所有 `x`，存在 `T` 中的一个 `y` 使得 `y = x`）
- `(∀x∈T: (∃y∈S: y = x ))` （对于 `T` 中的所有 `x`，存在 `S` 中的一个 `y` 使得 `y = x`）

如果你比较前两个命题，你会发现它们只涉及集合 `S`；此外，两个量词的位置互换了。在最后两个例子中，两个集合 `S` 和 `T` 的位置互换了。现在，如果你根据给定的两个集合检查这些命题以研究它们的真值，可以得出以下结论：

- 第一个命题显然是**真**（TRUE），因为你可以简单地为 `y` 选择与 `x` 相同的值。
- 第二个命题是**假**（FALSE）；不存在这样的值 `y`。一旦你通过选择一个值绑定了变量 `y`，`x` 就只有一个选择（即相同的值）。
- 第三个命题也是**假**（FALSE）；反例是 `x = 3` 和 `x = 4`。
- 最后一个命题是**真**（TRUE），因为 `T` 是 `S` 的子集。

###### 交换嵌套量词

从前一节的前两个例子中，你可以得出这个重要结论：有时你可以安全地交换表达式中嵌套的量词，但有时不能——这会改变表达式的含义。

**> 注意** 嵌套量化并不总是可以无风险地互换。

如果满足以下两个条件，你可以安全地交换公式中两个连续的量词：

- 两个量化类型*相同*；即它们必须都是全称量词（∀）或都是存在量词（∃）。
- 你量化的集合都不依赖于另一个集合所绑定的变量；即内部量化的集合不受你为外部量化选择的变量值的影响。

例如，以下两个表达式是重言式：

```
∀x∈A: ∀y∈B: P(x,y) ⇔ ∀y∈B: ∀x∈A: P(x,y)
∃x∈A: ∃y∈B: Q(x,y) ⇔ ∃y∈B: ∃x∈A: Q(x,y)
```

然而，以下两个表达式*不是*重言式：

```
∀x∈A: ∃y∈B: P(x,y) ⇔ ∃y∈B: ∀x∈A: P(x,y)
∀x∈A: ∀y∈(B∪{x}): Q(x,y) ⇔ ∀y∈(B∪{x}): ∀x∈A: Q(x,y)
```

最后一个例子相当棘手；回想一下第二章，符号 `∪` 代表并集运算符。所以，一旦你为 `x` 选择了一个值，在试图找到一个使 `Q(x,y)` 为**真**（TRUE）的 `y` 值之前，你会将该值添加到集合 `B` 中。请注意，如果你交换量词，表达式甚至会变得非法，因为现在你引用了变量 `x`，而此时它尚未被绑定。



## 第三章 更多逻辑

##### 连续嵌套量词的缩写

如果你有两个连续的、**相同类型**且作用于**同一集合**的量词，通常可以按如下方式缩写你的公式：

`∃x∈N: ∃y∈N: x + y > 42`

前面针对同一集合的存在量化变为：

`∃x,y∈N: x + y > 42`

同样地，对于全称量化：

`∀x∈N: ∀y∈N: x + y > 42`

前面针对同一集合的全称量化变为：

`∀x,y∈N: x + y > 42`

##### 量词的分配律

清单 3-13 展示了全称量词和存在量词的分配律。

**清单 3-13.** *量词的分配律*
* ( ∃x∈S: P(x) ∨ Q(x) ) ⇔ ( ( ∃x∈S: P(x) ) ∨ ( ∃x∈S: Q(x) ) )
* ( ∀x∈S: P(x) ∧ Q(x) ) ⇔ ( ( ∀x∈S: P(x) ) ∧ ( ∀x∈S: Q(x) ) )
* ( ∃x∈S: P(x) ∧ Q ) ⇔ ( ( ∃x∈S: P(x) ) ∧ Q )
* ( ∀x∈S: P(x) ∨ Q ) ⇔ ( ( ∀x∈S: P(x) ) ∨ Q )

注意，在清单 3-13 的最后两个例子中，谓词`Q`没有写成`Q(x)`；这表示谓词`Q`不具有参数`x`。只有当`x`**不**作为参数出现在谓词`Q`中时，你才能使用最后两个逻辑等价式；例如：
`( ∀y∈S: ( ∃x∈S: x>10 ∧ y<5 ) ) ⇔ ( ∀y∈S: ( ∃x∈S: x>10 ) ∧ y<5 ) ⇔ ( ∀y∈S: ( ∃x∈S: x>10 ) ) ∧ ( ∀y∈S: y<5 )`

顺便问一下，你看出最后一个等式中左边的合取项是`( ∀y∈S: Q )`这种形式了吗？更具体地说，它表示的是对一个命题的全称量化；它并不是在对一个参数被量词绑定的谓词进行量化。类似的情况也可能出现在存在量化中。看一下清单 3-14，它针对这种情况引入了两条重写规则。

**清单 3-14.** *对命题进行量化的重写规则*
* ( ∀x∈S: P ) ⇔ ( ( S = ∅ ) ∨ P )
* ( ∃x∈S: Q ) ⇔ ( ( S ≠ ∅ ) ∧ Q )

这里`P`和`Q`代表命题。使用这些重写规则，你可以将前面的例子进一步重写为以下表达式：
`( ( S = ∅ ) ∨ ( ∃x∈S: x>10 ) ) ∧ ( ∀y∈S: y<5 )`

##### 量词的否定

如果你将否定连接词与全称量词结合使用，会发生以下情况：

**1.** `¬(∀x∈S: P(x))` 意味着`P`对于`S`中所有元素`x`**并非**为真。
**2.** 因此，必然存在（至少）一个值使`P`不为真；换句话说：`(∃x∈S: ¬P(x))`

同样地，将否定添加到存在量化中会导致以下结果：
**1.** `¬(∃x∈S: P(x))` 意味着不存在`S`中的元素`x`使得`P`为真。
**2.** 因此，对于`S`中所有值`x`，`P`必须为假；换句话说：`(∀x∈S: ¬P(x))`

因此我们得到两个重要的量词重写规则，如清单 3-15 所示。

**清单 3-15.** *量词否定的重写规则*
* `¬( ∀x∈S: P(x) ) ⇔ ( ∃x∈S: ¬P(x) )`
* `¬( ∃x∈S: P(x) ) ⇔ ( ∀x∈S: ¬P(x) )`

你还记得前面章节“空集上的量化”（见表 3-2）中关于空集上量化的讨论吗？我们不希望空集对这些规则造成任何例外。因此：
* 给定 `(∃x∈∅: P(x)) ⇔ FALSE`，我们可以将其转换为 `¬(∀x∈∅: ¬P(x)) ⇔ FALSE`（使用清单 3-15 中的第一条重写规则）。
* 但这样一来 `¬¬(∀x∈∅: ¬P(x)) ⇔ TRUE`（等价式两边的谓词都进行否定）。
* 所以 `(∀x∈∅: ¬P(x)) ⇔ TRUE`，与谓词无关（划掉全称量词前面的双重否定）。



所以你看，无论冒号后的实际谓词是什么，空集上的全称量化总是得到`TRUE`，这在逻辑上是正确的。

如果你还没有完全信服，这里还有另一种推理方式。正如“量词与有限集”一节中关于量词的段落所解释的，全称量词对应于迭代的`AND`结构。因此，你可以建立以下一系列逻辑等价式：

```
∀x∈S: P(x)
⇔ P(x₁) ∧ P(x₂) ∧ P(x₃) ∧ ... ∧ P(x_{n-1}) ∧ P(x_n)
⇔ TRUE ∧ P(x₁) ∧ P(x₂) ∧ P(x₃) ∧ ... ∧ P(x_{n-1}) ∧ P(x_n)
```

根据表 1-12 中列出的“特殊情况”重写规则之一，添加前缀`TRUE ∧`是允许的。如你所见，你从一个包含`n`个值的集合`S`开始。现在，如果你开始逐个移除集合`S`的元素，你可以从末尾移除`∧ P(x_n)`，然后是`∧ P(x_{n-1})`，依此类推，直到你最终得到空集——因为你移除了集合`S`的最后一个元素。然而，此时等价式就简化为`∀x∈∅: P(x)`，它等价于`TRUE`。

#### 带量词的重写规则

前面的章节已经介绍了一些关于量化的重写规则。

列表 3-16 列出了最常用的量词重写规则。

**列表 3-16.** 带量词的重写规则
```
¬( ∃x∈S: P(x) ) ⇔ ( ∀x∈S: ¬P(x) )
¬( ∀x∈S: P(x) ) ⇔ ( ∃x∈S: ¬P(x) )
( ∃x∈S: P(x) ) ⇔ ¬( ∀x∈S: ¬P(x) )
( ∀x∈S: P(x) ) ⇔ ¬( ∃x∈S: ¬P(x) )
( ∀x∈A: ( ∀y∈B: P(x,y) ) ) ⇔ ( ∀y∈B: ( ∀x∈A: P(x,y) ) )
( ∃x∈A: ( ∃y∈B: Q(x,y) ) ) ⇔ ( ∃y∈B: ( ∃x∈A: Q(x,y) ) )
( ∃x∈S: P(x) ∨ Q(x) ) ⇔ ( ( ∃x∈S: P(x) ) ∨ (∃x∈S: Q(x) ) )
( ∀x∈S: P(x) ∧ Q(x) ) ⇔ ( ( ∀x∈S: P(x) ) ∧ (∀x∈S: Q(x) ) )
( ∃x∈S: P(x) ∧ Q ) ⇔ ( ( ∃x∈S: P(x) ) ∧ Q )
( ∀x∈S: P(x) ∨ Q ) ⇔ ( ( ∀x∈S: P(x) ) ∨ Q )
¬( ∀x∈S: ∀y∈T: P(x,y) ) ⇔ ( ∃x∈S: ∃y∈T: ¬P(x,y) )
¬( ∃x∈S: ∃y∈T: P(x,y) ) ⇔ ( ∀x∈S: ∀y∈T: ¬P(x,y) )
¬( ∀x∈S: ∃y∈T: P(x,y) ) ⇔ ( ∃x∈S: ∀y∈T: ¬P(x,y) )
¬( ∃x∈S: ∀y∈T: P(x,y) ) ⇔ ( ∀x∈S: ∃y∈T: ¬P(x,y) )
```

列表 3-16 中的第三和第四条重写规则指出，你可以将全称量化重写为存在量化（通过使用否定连接词），反之亦然。这意味着你不需要两种量词中的任何一种来表达任意量化谓词。然而，同时拥有这两种量词是很方便的，因为我们在自然语言中两者都使用。

你将在本章末尾的练习中证明其中一条重写规则。

#### 范式

范式（也称为规范形式）在数学中经常使用。这里的英文词“normal”是在符合规范（即标准）的意义上使用的，而不是“通常的”意思。

范式通常用于重写表达式，以便于求解（或进一步分析）。一个典型的例子是众所周知的二次方程范式：`ax² + bx + c = 0`。一旦你将二次方程重写为这种格式，你就可以使用二次公式来求出`x`的值。在逻辑中，也有类似的范式；我们将在接下来的章节中讨论其中两种。一般来说，范式在自动定理证明中很有用。接下来讨论的析取范式和合取范式对于验证一个谓词是永真式还是矛盾式特别有用。

**注意** 范式对于比较两个谓词也很有用。通过将两个看似相似的谓词重写为相同的范式，你可以更仔细地检查两者之间到底有什么区别。或者，在将两者都转换为范式后，你可能会发现它们实际上是同一个谓词。这是本书第二部分的一个应用场景，其中数据完整性规则被指定为形式化谓词。
