## 练习 2

如果你比较 `A ∧ B` 和 `A | B` 的真值表，你会注意到一个模式：

| **A** | **B** | **A** ∧ **B** | **A \| B** |
| :--- | :--- | :--- | :--- |
| T | T | T | F |
| T | F | F | T |
| F | T | F | T |
| F | F | F | T |

每当表达式 `A ∧ B` 为真时，表达式 `A | B` 为假，反之亦然。换句话说，第一个表达式是第二个表达式的否定。这应该能让你直接得出用与非表示与的解决方案：`(A ∧ B) ⇔ ¬ (A | B)`

如果你比较 `A ∨ B` 和 `A | B` 的真值表，你又会注意到一个模式：

| **A** | **B** | **A** ∨ **B** | **A \| B** |
| :--- | :--- | :--- | :--- |
| T | T | T | F |
| T | F | T | T |
| F | T | T | T |
| F | F | F | T |

`A ∨ B` 的真值在真值表中向下读取，并且等于 `A | B` 的真值（如果在真值表中向上读取的话）。因此，如果你先以相反的方式生成 `A` 和 `B` 的四种组合值，然后计算这些值的与非的真值，你应该就能找到解决方案。以“相反的方式”生成相当于对 `A` 和 `B` 的原始输入值取反。在前面的真值表中扩展 `A` 和 `B` 的否定，将得到以下真值表：

| **A** | **B** | **A** ∨ **B** | **A \| B** | ¬**A** | ¬**B** | (¬**A**) \| (¬**B**) |
| :--- | :--- | :--- | :--- | :--- | :--- | :--- |
| T | T | T | F | F | F | T |
| T | F | T | T | F | T | T |
| F | T | T | T | T | F | T |
| F | F | F | T | T | T | F |

现在第三列等于最后一列，因此你可以得出结论：`(A ∨ B) ⇔ ((¬A) | (¬B))`

## 练习 5

你可以使用真值表或现有的重写规则。或者，你可以为涉及的变量找出一个使谓词评估为假的赋值（即，给出一个反例）。

**a.** `P ⇒ (P ∧ Q)`
*反例：P = TRUE 且 Q = FALSE*
```
推导过程
TRUE ⇒ (TRUE ∧ FALSE)
⇔	TRUE ⇒ FALSE
⇔	FALSE
```
因此 `P ⇒ (P ∧ Q)` *不是*重言式。

**b.** `P ⇒ (P ∨ Q)`
*使用现有的重写规则*
```
推导过程	注释
P ⇒ (P ∨ Q)	
⇔	将蕴含改写为析取
¬P ∨ (P ∨ Q)	
⇔	结合律
(¬P ∨ P) ∨ Q	
⇔	特例
TRUE ∨ Q	
⇔	特例
TRUE	
```
因此 `P ⇒ (P ∨ Q)` 是重言式。

**f.** `(P ⇒ Q) ⇒ (P ∧ Q)` 不是重言式。
*反例：P = FALSE 且 Q = TRUE*
```
推导过程
(FALSE ⇒ TRUE) ⇒ (FALSE ∧ TRUE)
⇔	(TRUE) ⇒ (FALSE)
⇔	FALSE
```
注意，Q 的值其实并不重要：`P = FALSE` 且 `Q = FALSE` 是第二个反例。
