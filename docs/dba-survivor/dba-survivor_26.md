# 第四章：对开发者而言，开发服务器就是生产服务器

我记得有一次，有人直接质疑我为何以及如何以某种方式执行任务。那位开发者非常有礼貌地问我为什么在完成某事时多做了几个步骤，而我当时确实无言以对，只能说：“我更喜欢这样做，以保持一切处于标准状态。”另一次，我被命令以特定方式执行任务，因为他们“希望尽快完成”，但他们要求的方式却会花费更长的时间。

对于透露太多关于你所执行任务的细节感到顾虑是完全自然的。但我相信，与其保持沉默，不如沟通你的行动。如果有人建议了不同的完成方式，那么你可能会学到新东西。如果有人建议了错误的方式，那么你将获得一个帮助教育他人的机会——如果曾有过双赢局面，那这就是了。
