# 处理开发者反馈

显然，你肯定不想事情发展到那种地步，但总有些时候你真的很想把开发者揪出来，问问他们脑子里到底在想什么。保持冷静——没有人什么都懂，对吧？但也要设身处地为他们想想。如果你花了几个星期刚写好的代码正在被人逐条批驳，你也会产生抵触情绪，对吧？所以看到开发者产生抵触情绪也很自然。事实上，任何人在面对这种情况时有些防备心理都是正常的，即使他们的代码没有被挑剔。

尽力把这看作一个传授新知识的机会。不过，并非所有人都想听你“上课”，对于那些人，你需要保持冷静，并尽力说服他们从不同的角度看待问题。找出他们的障碍（为什么他们不愿意尝试新事物），并帮助他们克服它。久而久之，人们就会开始信任你的反馈，而不是认为你总是在攻击他们。
