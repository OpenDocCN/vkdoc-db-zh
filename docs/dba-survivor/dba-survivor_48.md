# 第 8 章：培训，开始行动吧

对于想要启动培训计划的个人或公司来说，有许多可用的选择。第一步是确定期望的最终结果；你希望从培训中获得什么？比*从培训中得到什么*更重要的是*你能从中带走什么*。如果你指望别人把东西拱手相送或替你完成，那么你将会失望。

无论你的预算如何，你都能找到某种程度上满足需求的培训。

有大量的免费培训选项（网站）和许多略高于免费的选项（专业培训师）。在安排培训计划时，你应该留意每个选项及其相关的优缺点。

**提示：** 在聘请培训师或参加课程时，做一些调查研究。确保你找到的是有信誉的培训师。

请注意，我所讨论的选项是自助式和有组织的培训课程的混合体。这是因为任何好的培训计划都会包含不止一种选项。如果你只是等待别人来为你安排培训，那么你很可能就会落后。你绝对需要掌控自己的职业发展；在力所能及时抓住自助式选项。

### 网站

互联网提供了大量的培训参考资料。作为数据库管理员，你可能专注于 SQL Server 培训选项，但即使你在寻找不同的东西——也许是一些通用的职业发展培训——你应该也能通过你喜欢的搜索引擎找到网站。参与论坛、新闻组，甚至 Twitter 都能提供有价值的培训材料。

**优点：** 大多数是免费的，但有些内容会收费。网站允许你按自己的时间安排进行培训；通常你不需要在特定的日期和时间在线。论坛和新闻组通常是与其他专业人士联系和分享经验的好方法。像微软和 Oracle 这样的公司提供大量免费培训材料，希望能说服像你这样的专业人士使用他们的产品。

**缺点：** 一分钱一分货。你需要意识到，有些人在论坛问题回复中发表疯狂的言论，还有些人开玩笑地发布幽默的答案而你可能看不出来。很难区分提供高质量内容的网站和那些不提供的，但一个开始的方法是查看每位作者的经验。在涉及你的数据库时，不要听信一个没有资历的陌生人的说法。也不要相信为内容付费就意味着你比免费网站更能信任这些材料；我在收费网站上也看到过很多错误。

### 自我培训

除了从网站、博客和播客获得的自我培训外，你也可以简单地开始一个副业项目，让你学到新东西。

也许你的公司需要建立事务日志传送，而你以前从未做过。把这当作你自学新东西的机会吧。

**优点：** 没有比实践更好的学习方式了。而且通过自我培训，你通常是在做自己感兴趣的事情，这使你更有可能记住所学的知识。你可以按照自己的节奏学习。而且你可能会发现，在某个领域学到的一些技能也适用于其他领域，特别是软技能。

**缺点：** 很容易屈服于分心或变得沮丧。如果你在一个小团队中，这也可能让你与他人隔绝。保持动力可能很难。

### 志愿服务

可能最被忽视的培训机会是志愿服务。当地有许多组织需要各种帮助。信不信由你，如果你想进入 IT 行业，最好找一个能提供一些 IT 帮助的本地团体。

最容易想到的例子是教堂或社区中心。依赖志愿者协助的地方，几乎在你能想象到的每个领域都需要帮助。

而且你不需要觉得你的志愿服务必须与技术相关。你可以自愿花时间指导你团队中的某个人。指导是提升技能的好方法，尤其是你的软技能。

**优点：** 这是丰富你的简历并获得实际经验的好方法。课堂学习有一定价值，但实际经验总是更好。此外，捐出时间帮助他人是一种很棒的感觉。事实上，无论是否有培训机会，你都应该这样做。

**缺点：** 你获得的经验可能相当有限；很可能你做的不会是最前沿的事情。除非你是在指导别人，否则你很可能会独自工作，缺乏指导。

### 在职培训

这也常被称为“便当午餐会”。其理念是通过邀请外部培训师来公司授课，为员工提供培训。或者由一位受人尊敬的同事来主导教学。

**优点：** 通过邀请人员到公司内部进行培训，这是最大化利用培训预算的好方法。如果你是被要求培训他人的员工，那么这也是你学习更多东西的绝佳机会。我经常告诉别人，学习某样东西的好方法是尝试把它教给别人。

**缺点：** 这会让你离开办公桌，但并未离开办公室；如果你需要履行工作职责，你可能会发现很难专注于培训材料。材料可能并不总是你感兴趣的主题。

## 专业协会

你可以找到符合你兴趣的专业协会。立刻想到的两个是：SQL Server 专业人士协会 (`PASS`, `http://sqlpass.org`) 和可用性专业人士协会 (`UPA`, `http://www.upassoc.org`)。两者都让你能够参与用户组会议并举办年度会议。

**优点：** 大多数协会提供基础会员资格，费用很低或免费。会员资格通常包括一份通讯，提醒你即将举行的活动，让你能够联系、分享并向他人学习。许多这类活动都是免费参加的。

**缺点：** 就有组织的协会而言，选择并不多，你可能会对整体选择感到失望。此外，它们可能仅仅侧重于建立人脉，而不是培训或专业发展。你需要做额外的研究来了解会员资格的确切好处。

### 继续教育

当地的学院或大学可能提供相关的课程，可能符合你的兴趣。也许你正在寻求获得 IT 学位，或者更通用的 MBA。或者你可能只想上一门课，以便在某个特定主题上获得更多知识。

**优点：** 赚取学位学分。课程通常在晚上，这意味着你不会损失工作时间。除非是更正式的学位或认证计划的一部分，课程通常持续八周。

**缺点：** 可能费用高昂；咨询你的人力资源部门，看看公司是否有政策协助支付课程费用。课程通常在晚上，意味着你会失去本可以与家人共度的时间。
