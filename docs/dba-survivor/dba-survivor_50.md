# 第八章：培训，来获取一些吧

当你逐一审视这些益处时，你应该能将它们分为两类：那些能轻易证明与业务目标一致的，以及那些与你个人目标一致的。对于个人目标，看看它们是否也能被证明对业务有益。

例如，如果你说其中一项益处是与其他专业人士建立人脉的能力，你的公司很可能不会在意。但如果你说，“我将能够与其他专业人士建立人脉，这能让我提升一些必要的软技能，并且将来当我向这个人脉寻求技术问题帮助时，将为公司带来好处。”那么你的公司可能会非常在意。

你列出的许多个人目标都能被证明能带来业务收益；你只需要稍加思考。

## 确定成本节省

这可能很难计算，因为收益并不总是能体现为硬性成本。

大多数人通常关注易于计算的硬性成本和成本削减。寻找有助于所谓“成本规避”的培训也是可以的。这是指你节省了尚未支出的费用。

一个例子是虚拟化。你可能有一些使用了几年的服务器。你可以购买新硬件来替换旧服务器，或者考虑对旧硬件进行虚拟化。物理环境和虚拟环境之间的成本差异就属于成本规避。

**提示：** 理解成本削减和成本规避之间的区别。你不可能声称通过*不*购买钻石每年为你的家庭节省了数百万美元，对吧？

如果你能量化因培训而节省的时间量，那么通过简化流程也能带来成本节省。另一个寻找成本节省的地方是，如果培训后你能够承担更广泛的职责；也许你可以开始主持未来的培训课程，从而减少公司聘请外部培训师的需求。

### 衡量你的结果！

最后这部分可能是最重要的。在你把其他一切都准备好之后，花些时间思考你将如何衡量你的成功。如果你声称有成本节省，那么写下你计划在培训完成后如何计算这笔节省。成本规避也是如此；务必跟踪所有你能收集到的实际成本，以及如果不进行培训的预估成本。我经常看到任何商业案例中遗漏了这部分内容，好像衡量要么是计划中被遗忘的组成部分，要么就是人们根本不理解其重要性。

没有衡量，你将无法验证培训的结果。如果结果低于预期，那么你有机会从经验中学习，并对未来的培训计划进行修正。如果结果好于预期，那么当你向管理层展示成果时，你的衡量数据将成为宝贵的资产。

我以“每个人都需要培训”作为本章的开头。当你下一轮培训结束时，你需要跟踪在接下来的一年里你运用了多少该培训内容，以及它为业务节省了多少钱。这样做将使公司更容易派你参加下一轮培训，并让你成为更出色的员工。
