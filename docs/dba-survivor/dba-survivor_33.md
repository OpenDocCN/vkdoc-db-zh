# 第五章：生产支持

在法国南部生活和工作是我人生的一个目标。其实哪里都可以，我真的不挑剔。在法国快速搜索一下技术类工作，就能找到不少我符合条件职位。那么，为什么我现在没有在法国生活和工作呢？

因为我的语言水平还不足以让我作为一名数据库专业人士在那里工作。

而掌握当地的语言是一项相当重要的技能。

对于语言能力有限的人来说，有没有可用的数据库专业工作机会呢？可能有，但我又愿意花多少时间去尝试安排与素不相识的人面试呢？更好的做法是让我的社交圈知道我有兴趣，然后等待合适的机会自己找上门来。

**提示：** 我一直很钦佩那些花时间学习第二语言的人，并且在可能的情况下，我也会尽力帮助他们进行交流。

让我们反过来设想一下，如果有来自另一个国家的人来找我求职。如果他们具备所有必要的技能，除了不能用英语交流，我会雇佣他们吗？当然不会——没有人会雇佣一个不能说该语言的人来从事需要良好沟通技巧的工作。

在面试候选人时，我也喜欢评估他们沟通技巧的重要性。如果你需要与客户沟通，那么清晰表达的能力就很重要。如果你只与 DBA 团队内部互动，那么在口头表达能力方面，我可能会更宽容一些。

然而，这在外包中是一种常见做法。无论出于什么原因，人们只是接受沟通能力低于平均水平的人，而不是说：“抱歉，但我们需要一个能更好地说我们语言的人。”在沟通技巧方面，我们从不犹豫。如果在面试中我无法理解候选人，那么我的客户也就根本不可能理解他们。

因此，我们坚持寻找能够沟通的候选人，并且把这一点看得比技术技能更重要。我觉得我可以教会任何人他们需要的技能，但我无法教会他们英语。我不需要一个高级管理员；我宁愿要一个能说会道的初级管理员。而且，为了应对任何人员流动，我们总是会保留一个影子资源（备用人员），这样我们就可以轻松地让新成员过渡到团队中，而不会遭受支持中断的损失。

## 安全性

外包的另一个常见担忧是公司数据的安全性。我确信这是一个自然的担忧，在当今全球电子化社会中也是合理的，但我一直不明白这和你雇佣任何新员工有什么不同。我猜你可能会说不一样，因为外包团队在你的网络之外，或者你无法看到他们在办公桌前在做什么，但在必要时，有很多方法可以监控他们的活动。此外，数据盗窃的风险是真实存在的，而且可能发生在任何地方——你的网络内部或外部。

我们最初也开始将此列为一个担忧。如此重视，以至于我们讨论过不允许我们的外包团队处理任何与安全相关的事情。过了一段时间，我们理解到许多安全请求本质上是运维性的，并且我们很顺利地过渡，将这些任务外包了出去。毕竟，我们需要他们执行这些任务，以便腾出时间专注于实例配置、基线和监控方面的一些更大问题。

我建议你在安全问题上，像对待其他合作伙伴一样对待你的外包合作伙伴。明确哪些任务是允许的；你可能有一些安全项目不允许其他任何人处理。并且要记住，你需要留在内部的工作和你想要留在内部的工作是有区别的。

## 工作保障

大学期间的一个夏天，我在一家家具厂做计件工作。

本质上，我会组装即将成为椅子、沙发等框架的不同部件。我很少能看到成品，因为我们合同规定的工作只是制作一个框架，然后把它送到另一个车间去完成最终的部件。

当我开始适应环境和工作后，我的制作速度也变快了。我会开始工作，找出一种效率最高的方法来最大化我的产出，并专注于完成任务。我喜欢这份工作，也喜欢记录自己的表现。在我工作台旁边工作的那个人也和我保持着同样的节奏。我们可以边工作边聊天，并且完成了大量的工作。

**提示：** 你应该花更多时间专注于自己的努力，少花时间担心别人。专注于你能控制的事情；否则，你很快就会感到沮丧。

有一天，一个同事走过来，简单地说：“慢点干；你们这样让我们看起来很差劲。”

我不知道把工作做好在别人眼里竟然会变成一件坏事。

当然，我知道类似考试中有人考得好会打乱全班成绩曲线的情况，但这不一样。首先，这个人是认真的。

其次，他可以使用很多工具和机器，这些可能会伤害到我，尤其是在一场工业“事故”中。我不知道该说什么，但我旁边的人却很清楚：“走开，”他说，“你自己也试着干点活吧，别再告诉我们干得太快了。让你看起来差劲的只有你自己。”

谈到外包，许多人感到忧虑。他们不想看到自己执行的任务被别人做，尤其不想看到别人开始做得比自己更好。这是一种自然的反应，也完全可以理解。只是，嗯，我不理解这种反应。

也许是我骨子里的`汤姆·索亚`精神在作祟，但如果我能找到别人来帮我粉刷篱笆，我会的。让别人接手我正在做的任务，我从未感到威胁。我一直把这看作是我开始做一些新的、更伟大的事情的机会。我可以转移我的注意力，学习新东西，承担更大的责任，并作为一个个体成长。

我从不害怕把任务交给我们的外包团队。我们部门有人会告诉我，“如果你给他们太多的工作，那他们就不需要你了。”我从来无法理解这种思维方式。从我们决定外包的那一刻起，我就只向前看，从不担心失去自己的工作。

## 所有美好的事物

我的外包经历相当正面。显然，根据许多谈论外包多么糟糕的人的说法，我是个异类。我觉得有趣的是，当有人认为外包就是指将某个特定角色或功能发送到海外，却不明白雇佣一个承包商到现场来做是一回事。如果你雇佣某人来帮助你完成一些明确的任务，他们坐在哪里真的很重要吗？如果他们坐在你旁边做你的一部分工作，你就感到安心，而如果他们离你很远，你就不那么安心，这是为什么呢？
