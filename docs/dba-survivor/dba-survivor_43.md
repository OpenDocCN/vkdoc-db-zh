# 第七章：自助餐在哪里？

## 最重要的一章？

**作者：乔纳森·杰尼尼克**

认识我的人都知道我对培根情有独钟。我非常喜欢它，以至于确信它最终会毁了我。虽然给任何东西加上培根都会立刻变得更美味是事实，但吃太多培根显然会导致健康问题也是事实。IT 专业人士整天都在做出色的技术决策，但一到吃饭时间，我们就会做出糟糕的决定，这些决定日后会回来困扰我们。我们长时间、高压地坐在椅子上，期间穿插着匆忙吃下的最快、最方便的食物。然而，只要适度，你也可以鱼与熊掌兼得。

**提示：** 培根之于食物，犹如黑色之于时尚；它与万物皆相配。

在本章中，我们将讨论以下内容：

1.  食物如何成为最容易获得的药物选择
2.  社交互动如何导致糟糕的营养选择
3.  规律锻炼的必要性
4.  如何避免让你的职业生涯和生命过早结束

你是否曾认为自己的工作很危险？你应该这么想。我们都应该。我们这些在科技行业的人——图书编辑也包括在内！——太容易陷入认为自己拥有“安全”工作的错觉。我们只是坐办公室，对吧？但我们正面临着严重的危险！我们活动量不足。我们整天坐着。我们的身体在萎缩。我们面临各种冠心病的风险。还要担心糖尿病。你读过关于深静脉血栓的文章吗？那是坐得太久引起的。最近背疼了吗？也许是因为缺乏锻炼，你的核心肌肉太松弛，无法很好地保护背部。

想想生活质量。汤姆在本章接近尾声时谈到了这一点。他希望能奔跑着和孩子们玩耍。我喜欢活动，并希望在我的一生中，只要力所能及，就能继续出去远足、滑雪、山地骑行等等。然而几年前，我发现自己不得不放弃越野滑雪，因为有一天我发现自己超出了滑雪板的设计承重极限（我可以买更长的滑雪板，但这并不是这里的目标）。

汤姆在本章描述的所有坏习惯，我都有。我活了足够长的时间，已经尝到了它们带来的一些苦果。我吃得太多，睡得太少，锻炼不足。这是一场持续的战斗，要努力收复我已经失去的阵地。不认为你的工作危险？再想想吧！

### 食物可以是药物

食物是我们大多数人目前最容易获得的药物。它是合法的、随手可得的、能让我们感觉良好并给予我们安慰。我们可以和朋友聚餐，我们告诉自己，我们随时可以停止吃那些不健康的东西，但我们从未真正做到。

例如，我喜欢甜甜圈，真希望它们能配上培根（无论是放在旁边还是像某种油炸卡尔佐内三明治一样直接烤在中间）。我喜欢早上喝咖啡，而且*非常*喜欢咖啡和贝奈特饼。当然，在我生活和工作的大部分地方都找不到贝奈特饼，所以我只能用甜甜圈凑合。我闭着眼睛都能找到一家甜甜圈店。尽管我如此热爱咖啡和甜甜圈，但我并没有每天都沉溺于这种组合。我可能每周喝咖啡时吃一次甜甜圈，因为我明白适度的必要性。

**提示：** 适度和平衡是关键。只要你适度并遵循均衡饮食，你想吃什么都可以。

下次你去自动售货机买点脆脆的、奶酪味的、咸咸的东西来快速补充能量时，停下来问问自己为什么要买那份零食。很可能你会发现，自己吃东西不是因为饥饿，而是出于情绪化进食或无意识进食。

### 情绪化进食

这通常是某种情绪触发的结果。它可能源于你当前工作职能带来的压力。例如，你的公司可能正在经历裁员，而你被要求协助管理这个项目，这意味着你将解雇那些与你共事多年的人的职位。或者，你可能只是有一个同事不太尊重你且要求很高。施加在你身上的压力会直接导致情绪化进食。

**注意：** 情绪化进食的问题在医学界已得到广泛认可。梅奥诊所有一篇关于此主题的优秀文章，网址是 www.mayoclinic.com/health/weight-loss/MH00025。快速在谷歌上搜索“情绪化进食”一词，会为你提供大量其他文章供你参考。

当你情绪化进食时，你是在用食物来控制自己的情绪。这可能简单到只是一种对抗无聊的方式。因为你不喜无聊，于是你开始想到一些冰淇淋。既然你在想冰淇淋，你就不再无聊了，不知不觉中，你就吃下了一个三重份的 Rocky Road 冰淇淋。

情绪化进食是一种你不想陷入的模式。你发誓早餐不再多吃一磅培根，但第二天你又这样做了。你的同事或经理在办公室里对你态度恶劣，于是你伸手去拿一包薯片。你发现自己*在根本不饿的时候*还在吃东西。

情绪化进食可能难以克服。解决任何问题的第一步是首先承认问题的存在。一旦你做到了，下一步可以很简单，比如记录你一天的饮食日记。也可能更复杂，比如寻求治疗师的帮助。最终，你需要更关注自己的健康，减少对情绪化进食做出反应的可能性。

### 无意识进食

无意识进食发生在你没有意识到自己在把食物放进嘴里的时候。听起来很傻，对吧？我的意思是，*你怎么可能不知道自己在吃东西？* 这并不是不知道自己在吃东西的问题，更多的是你没有意识到自己吃了多少或吃了什么。

典型的例子是电影院的爆米花。大多数时候，影院的爆米花是提前很久（有时是提前几天）做好的，存放在大的塑料袋里，直到拿出来放在加热灯下提供给观众。如果你拿到一碗那样的爆米花和一碗刚爆好的爆米花，味道上的差异会相当明显。然而，当你去看电影时，你却心甘情愿地吃完一整桶不新鲜的爆米花，而不是要求新鲜的。

为什么？因为*你在无意识地吃东西*。你被电影分散了注意力。你的焦点不在食物上，而在别处。结果，你继续把爆米花铲进喉咙，直到吃完为止。在电视机前吃饭也是如此，或者……

*在你的电脑屏幕前。*

是的，这是真的。整天坐在办公桌前盯着电脑屏幕，这种程度的分心足以让你开始走向无意识进食的道路。不相信我？离开你的椅子，走出你的隔间，开始留意你部门里其他人的体型。我猜其中超过一半的人超重。而且他们很可能也在做你整天在做的事：坐在办公桌前盯着电脑屏幕。

**提示：** 无论如何，走出你的隔间是个好主意。去伸展一下腿脚几分钟。有机会的话，爬一段楼梯。



## 克服不良习惯

如何对抗情绪化或无意识的进食？如前所述，你需要做的第一件事是认识到问题正在发生。从那里开始，你将有不同的方法来克服不良的饮食习惯。我最喜欢的恰好是食物日记。你只需记下一整天吃下的所有东西。无论份量大小，都写下来。当你到了下午，看到长长的清单时，就很容易明白额外的热量是从哪里以及如何加入你的饮食中的。

另一个我喜欢的小技巧是让自己周围都是更健康的选择。我会带些胡萝卜条或一个橙子作为下午的点心。有些日子我午餐前会吃一个苹果。这帮助我理解，如果我之后又伸手去拿东西吃，那并非因为我饿了；我已经吃过零食了。

### 小贴士：

我喜欢的其他零食包括葡萄、杏仁和芥末豌豆。还有爆米花——我爱爆米花——而一小袋零食份量的爆米花大小刚刚好。

也会有些时候你决定不吃某顿饭，最可能是午餐。不吃正餐会导致新陈代谢减慢，降低你的精力，并削弱你专注的能力。比这更糟糕的是，你会更容易开始吃零食，所以总是检查一下日程安排是个好建议；如果你知道自己时间紧迫，那就给自己准备一些健康的零食。
