# 第九章：连接、学习、分享

在你建立了自己的工作常规，并对 DBA 技能感到得心应手之后，你可能会发现自己被吸引去帮助他人。别惊慌；发现自己愿意帮助他人是完全正常的。在其他数据库专业人士及其技能面前感到胆怯也很正常。每次我坐在一群 DBA 中间时，我总是假设每个人都比我懂得多。然后我会惊讶地发现，桌子对面的人对我也是同样的想法。

我在担任 DBA 期间学到的一件事是，通常你总能找到愿意提供帮助的人。我认识的大多数 DBA 都不害怕承认他们不知道某事。这与其他职业形成对比，在其他职业中，人们倾向于声称自己是某方面的专家（电视脱口秀主持人），或者觉得需要假装是专家，或者根本不想让任何人质疑他们的专业水平（医生或律师）。

因为大多数 DBA 愿意承认自己有所不知，所以大多数其他 DBA 也愿意提供帮助，因为他们知道，总有一天他们自己也需要帮助。

**提示：** 没有一个人能知晓所有事情。

## 沟通

沟通技巧对任何人来说都很重要，无论其职业如何。在当今互联互通的世界中，拥有强大的沟通技巧比以往任何时候都更加重要。电子邮件、博客、电话和面对面讨论对大多数 IT 专业人士来说相当普遍。拥有良好沟通技巧的个人，相对于缺乏这些技巧的人具有优势。

**提示：** 电子邮件是与他人沟通的糟糕载体。拿起电话与人交谈吧。

很多时候，人们告诉我，一谈到良好的沟通技巧，他们往往想到的是高层管理人员。换句话说，这似乎不是他们在其特定职位上需要关心的事情。

虽然你可能确实需要良好的沟通技巧才能攀登公司的晋升阶梯，但同样正确的是，良好的沟通技巧使你能够与你周围的人保持关系。同样正确的是，大多数招聘的技术岗位通常都把“良好的沟通技巧”列为一项要求。

如果你已经具备良好的沟通技巧，那么你与其他专业人士建立联系、学习和分享就会更容易。如果你的沟通技巧需要一些提升（而且大多数人的沟通技巧在某种程度上总是可以改进的），那么只有一个方法可以变得更好：你需要练习。根据我的经验，最好的开始方式就是去做所有你想提高的事情。

想提高打电话的水平？那就多打电话。想提高写作水平？从多做一些额外阅读开始，然后自己写点东西。想提高在会议上激发他人兴趣、接受你想法的能力？改进你的姿势和肢体语言。但是，你该如何以及从哪里开始这一切呢？

**提示：** 别忘了衣着。汤姆提到了姿势和肢体语言。别忘了这两项是包裹在衣服里的。想想你穿什么去办公室。在合理范围内，向上靠拢着装。每个办公室对于可接受的着装都有一个高端和低端的范围。倾向于高端。——乔纳森·詹尼克
