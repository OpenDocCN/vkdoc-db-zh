# 第九章：连接、学习、分享

**作者：贾里德·斯蒂尔 (Jared Still)**

我鼓励任何希望成为更好沟通者的人开始写作，或许可以从撰写个人博客开始。不要以为写博客就意味着你认为有人关心你说什么。相信我，他们并不关心。不，你需要将博客视为提升自己写作能力的一种方式。随着时间的推移，你会考虑写点什么发布到某个网站上。这将是你与他人分享经验、与他们建立联系并帮助别人学到新东西的机会。

渐渐地，在写了很多东西之后，你会想和别人谈谈它。

**发现问题。做研究。写作。**

开始写作的一个好方法是找到一个你不知道答案的问题。研究这个问题。当你感觉有了一些有价值的东西可以贡献时，就提出一个解决方案。你可以将你的发现写在论坛里或作为一篇博客文章发布。选择让你感觉最自在的媒介。

不要过于谦虚。任何做过一些研究并对相关技术有所了解的人都能做出有价值的贡献。你会学到新东西，而且你的努力也很可能会改善你的社交网络。

### 开始演讲

我从 18 岁起就开始担任领导职务。因此，我已经在众人面前站立并发表演讲很长时间了。

然而，每次在他人面前讲话，我都会紧张。我仍然记得作为一名研究生教授代数时有多紧张，那当然不是因为我不熟悉内容，或者我认为听众中有人比我更懂二项式。

有一次我参加一个会议，我的一位好朋友正在做报告。她有点紧张，但她告诉我，为了提高公开演讲能力，她开始唱卡拉 OK，因为如果你能在陌生人面前唱歌，那么讲话就容易了。我同意这个说法；但你不会很快在酒吧里找到我唱歌。对我来说，对抗紧张的唯一方法就是直接开始。一旦我开始了，就会变得容易一些。

真正让我感觉更好的是，当我能与听众中的一些人互动时。

很多时候，在开始之前，我会在前几排找个人聊聊天，而其他人正在就座。这能让我放松下来，并让我稍微热身一下。

**提示：** 在做报告之前，一定要花时间热身，这会提升你报告的质量。

另一种让你自己和你的听众放松的方法是在演讲过程中讲故事。你可能听过人们说，你应该在演讲开始时讲个笑话，以帮助大家放松，包括你自己。我发现，通常最好的演讲者是在整个报告中穿插轶事趣闻的人。讲故事是一种向数百人同时传达信息的有力方式。

现在，并非每个人都需要具备向数百人演讲的能力。我想我一次面对过的最多听众不到两百人。出席演讲的人数对我来说其实并不重要，最重要的是听众和我的材料。这就是为什么对我来说，尽快与听众建立联系很重要。这样做将使我更容易与他们分享我的信息，并希望他们也能学到东西。

我的第一次技术演讲是在一个本地用户小组进行的。我利用小组的反馈，为同年晚些时候在一个全国性会议上的演讲做准备。从那时起，我已经以各种形式演讲了数十次。我发现一个事实是，大多数人对讲故事或轶事趣闻的共鸣，比对直截了当的技术讨论更好。我努力成为一个好的讲故事者，但我离我希望有一天能达到的水平还差得远。这就是为什么我继续做演讲，因为我想不断进步。

我想进步的原因不是因为我总想在会议上发表演讲。我想进步是为了能够随时随地向我的同行发表演讲。更重要的是那些需要向高层管理人员做报告的时候。我希望能够像沟通技巧良好的人一样，自信地进行我的演讲。这意味着我需要练习说话，这意味着我需要做更多的写作，参加更多的活动，并做更多的阅读。

它们共同作用，使你成为一个更好的沟通者。

### 克服恐惧

### 作者：乔纳森·杰尼克 (Jonathan Gennick)

汤姆的朋友用卡拉 OK 来提高她在公开演讲中的信心。对我来说，帮助很大的是与一个青年团体合作。

在我职业生涯早期，我对公开演讲非常恐惧，以至于我会断然拒绝任何在众人面前讲话的请求。后来有一天，我被拉去帮助我所在教堂的一个青年团体。我不太清楚那是怎么发生的，但突然间，我就站在了十几个青少年和准青少年面前，他们会在我感到无聊时随时嘲笑、起哄并愉快地告诉我。

相比之下，成年人要礼貌得多。我现在有时还是会紧张，我也不是最好的演讲者，但恐惧已经消失了。当我紧张时，那是健康的那种紧张，来自于担心一切能否按计划进行。曾经一想到要在一群人面前讲话就会袭来的、令人恐慌的冰冷恐惧感，已经消失了。

## 专业协会

多年前，我有机会参加了一个专业协会的会议。

在那之前，我一直认为会议就是人们去坐在房间里听讲座的地方。当我看到供应商和他们的展台时，我第一次意识到会议不止于此。我开始明白，这不仅仅是参加一些无聊的讲座；这个活动有一种真正的活力。人们不仅仅是在学习；*他们是在建立连接*。

这个会议是由一个专业协会举办的。那时我对专业协会的唯一了解是我在研究生院时加入了几个协会。每个学生默认都注册了一两个协会。这意味着我们会在信箱里收到每月的通讯，仅此而已。我认识的唯一其他会员就是我的同学们。

在这次会议期间，我开始更多地了解加入专业协会的好处。作为会员，人们有机会及时了解其行业的最新趋势。他们获得了无尽的社交机会；与许多其他志同道合的人建立连接、学习和分享。

**提示：** 人多力量大；成为专业协会的会员可以赋予你巨大的力量。

在我成为一名数据库管理员后，我参加了一次我的第一次会议。那是由 SQL Server 专业协会 (PASS) 举办的。我决心尽可能地享受会员的所有好处。我想成为我能做到的最好的 DBA，而我知道实现这一目标的方法之一就是加入一个专业协会。



在会议的第一天，我遇到了两个人，他们至今仍是我亲密的同事。会议结束后，我一直与这两位保持联系，我们时常能够互相帮助。不仅是在遇到问题时，甚至在关于最佳实践的非常一般性的问题上也是如此。当我们彼此无法提供答案时，我们也会尽力指引对方找到正确的方向。

等到我参加第三次 PASS 会议时，我已经建立了一个大约 30 人的关系网，可以向他们寻求类似的帮助。我还惊讶地发现，随着我的经验越来越丰富，我也越来越能够帮助他人。很快，我发现自己开始撰写文章以供发表，因为我想要与他人分享我的经验。毫无疑问，仅仅通过成为 PASS 的一员，我个人作为一名数据库专业人士就得到了成长。

多年来，我参与 PASS 的活动给了我演讲和写作的机会。更重要的是，它给了我结识他人的机会。我想向大家强调一个事实，那就是 PASS 是一个数据库专业人士的*协会*。我经常被问到这样一个问题：“我为什么要费心加入任何专业协会呢？”以下是我经常给出的答案。

### 最新行业信息

成为专业协会的会员可以让您及时了解行业的最新趋势。当然，您可以希望从网站、博客和杂志获取最新信息。然而，当您成为会员后，您往往能*在*信息付印*之前*就获取到它。当您设计自己的环境时，这种内部渠道的访问可能是无价之宝。

会员资格提供的不仅仅是行业趋势的前沿信息。您还可以获取特定于您所在行业的产品和服务信息。通常，供应商在推广其产品时会瞄准专业协会及其会员。这一点在会议上表现得最为明显。您会看到数十家供应商排队向您展示他们的产品，希望能赢得您的业务。这使您能够了解大量产品以及它们如何适用于您的公司。

另一个优势是能够与其他会员交流有关流程的信息；换句话说，就是询问“在你们公司，这是怎么做的？”。您会发现许多会员愿意通过分享他们的经验来帮助其他会员。

### 写作机会

大多数专业协会都提供某种类型的出版物。可能是一个网站，一份电子邮件形式的新闻通讯，或者是一本印刷期刊。采用何种沟通类型和形式其实并不重要；每一种都是您写作和发表作品的机会。您写得越多，发表得越多，随着时间的推移，您的知名度就会越高。

**提示：** 是的，写作机会包括写书，信不信由你。

您可能很快就会获得额外的写作机会。还记得那些瞄准专业协会的供应商吗？嗯，他们喜欢找人撰写文章、技术简报和白皮书。甚至可能有机会让您合作写一本书（或者您自己单独写一本）。

### 演讲机会

许多专业协会都有年度会议或大会。通过成为会员，您增加了获得演讲机会的可能性。一些协会还有本地会议，这些会议不那么正式，但仍然为您提供通过演讲参与的机会。您可能并不总是需要会员身份才能演讲，但这无疑会增加您的机会。

您也可能仅仅通过回应行业出版物或期刊上的广告而找到演讲机会。一家公司或组织可能正在寻找具有特定专长的人。他们可能会联系您的协会，以找到愿意就某个特定主题进行演讲的人。我经常看到公司在寻找人来进行培训时这样做，但我也看到希望组织大型一日活动的公司这样做。

### 人际网络

如果您想建立人际网络，当然不需要加入专业协会。但是，如果您刚开始建立人际网络，与您所在行业的其他人建立联系不是更有意义吗？为什么要花时间与您领域之外的其他专业人士建立联系呢？

除了与您特定领域的其他专业人士建立联系外，您还将能够与供应商建立联系，这可能是非常有价值的，尤其是当您需要一些快速支持时。

当然，建立人际网络的最大优势之一是可能由此产生的职业机会。您永远不知道什么时候会有职位空缺；当机会出现时，您成为某人首先想到的人是件好事。

### 人际网络基础

我认识不止一打的人认为人际网络是一个不受欢迎的词。作为一个害羞的人，我发现和其他人一样，很难适应人际网络。然而，我确实理解建立广泛关系网的价值，以及它如何让您与他人连接、学习和分享。尽管如此，许多人似乎并不理解人际网络的真正目的。

人际网络的目的是建立一系列互利共赢的关系。

说真的，就是这么简单。建立网络的全部理念就是建立对双方都有益的关系。有时您会需要您关系网的帮助。其他时候，您将能够帮助您关系网中的其他人。此外，您给关系网中的人的帮助越多，您可能得到的回报也越多。

**提示：** 把您的职业生涯想象成一艘船，而您的人际网络就是您的船员。如果您想航行得更远，那么您需要一支可靠的船员。您的船越大，您的人际网络就需要越大。

但是，一个人如何着手建立这样一个人际网络呢？对大多数人来说，这当然不是一件容易的事。当您在一个房间里，周围似乎都是彼此互动良好的人时，认识新朋友对许多人来说并不是自然而然的事情。接触陌生人可能让人感到相当不知所措。

### 建立您的人际网络

要建立您的人际网络，您需要认识其他人。这真的没有其他办法；您必须与他人互动。这可以通过面对面的互动（比如说，一个社交活动）或通过所谓的社交网络来完成。社交网络涉及专门用于让您在在线论坛中与他人互动的网站。一些比较知名的社交网络网站是 `Twitter`、`Facebook` 和 `LinkedIn`。

许多人发现在网上与陌生人进行社交互动比在群体环境中面对面与他人互动更容易。即使在并非一定是社交活动的情况下，有些人也发现与陌生人交谈很尴尬。
