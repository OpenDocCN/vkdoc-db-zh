# 第九章：连接、学习与分享

约翰走进商店时感到很不自在，他试图融入其他顾客。他四处寻找车头灯却一无所获。他不敢寻求帮助，因为他不想让人知道他连车头灯这么简单的东西都找不到。过了一会儿，他发现一个穿着商店制服的年轻人，便向他求助，但那个年轻人并没有帮上什么忙。

沮丧之下，约翰走向客服台，终于开口请求帮助：“请问有人能告诉我车头灯在哪里吗？”

柜台后的男人问约翰：“先生，您以前买过汽车配件吗？”

“没有，”约翰回答，“这是我第一次买。您怎么知道的？”

“先生，”那人回答道，“这是一家游泳池用品店。”

如果你想建立自己的人脉网络，那么你需要做两件非常重要的事情。首先，你必须以某种方式与他人互动。其次，你必须确保自己一开始就在正确的房间里。

### 如何感到舒适

那么，如何克服那些让你不适的感觉呢？

找出让你最不舒服的事情，然后别去做那些事。

听起来很简单，对吧？嗯，这要看情况。

首先，要确定什么让你最不舒服可能很困难。我认识一些人害怕与新人交谈，因为他们担心可能会给人留下坏印象。我还有一些人害怕新人，因为他们自己本身就不够自在。你需要*确切*找出你与新人见面时最大的担忧是什么。这样做将使你有机会克服这些挑战，而这恰好也是你面临的第二个困难。

大多数人都会觉得参加不认识任何人的活动很不舒服。如果这描述的也是你，那么努力找一个朋友陪你一起去参加活动。但要确保你不会整晚都只和你的朋友聊天，而一个新人都没认识。

**提示：** 准备一个谈话开场白总是有帮助的。如果实在想不出别的，你总可以从天气开始聊。

另一个对很多人有帮助的建议是早点到场。这样更容易与其他早到的人开始交谈。如果你姗姗来迟，可能会发现更难加入一群正在进行的谈话中。当你早到时，你不仅可以和少数其他早到的人交谈，还可能发现更容易从你当前所在的小组移动到另一个小组。

这就是为什么有些人更喜欢社交网络的原因之一。在线与人见面比面对面更舒适。然而，我要提醒你，在线时你并非匿名。在论坛上发表任何评论时都要注意言行。

### 如何接近他人

我绝非天生健谈的人。走到陌生人面前和他们交谈，这对我来说并不是一件很自在的事。不过，这些年来我学到了一些东西，帮助我理解了在社交或人脉活动中如何以及何时接近他人。

首先要注意的是人们如何相互互动。无论一个小组有多少人，该小组总是被视为开放或封闭的。

开放小组意味着人们的身体姿态表明他们邀请他人加入他们的谈话。当人们站着交谈时，他们会根据讨论的性质自然地以特定方式摆放身体。大多数讨论都适合形成开放小组。

当谈话的性质使得参与者彼此相对而立，且姿态不具邀请性时，就形成了封闭小组。对于这样的小组，你不想尝试插入他们的谈话，你应该扫视房间，寻找开放的小组。

**提示：** 我喜欢寻找那些正在寻找他人的人。希望这能说得通。换句话说，通常会有一些人，他们脸上带着邀请的表情，那表情仿佛在说“来和我聊聊吧”或者“我希望有人能和我说话”。——西尔维斯特·卡斯塔芬

一旦你弄清楚哪些小组是开放的，你就需要走过去自我介绍。这不需要很花哨或困难。一句简单的“你好，我叫汤姆，很高兴认识你。”就足够了。当然，你应该用你自己的名字。之后，你可以用一个基本的问题开始对话，比如“你是怎么知道这个活动的？”或“你是做什么工作的？”你甚至可以陈述一下，比如“我很紧张。”或“我不擅长认识新人。”这些都是可以接受的对话开场白。

在你待了一段时间后，你可能想转到一个新的小组。关于必须在任何一个小组待多久，没有硬性规定。但在离开你所在的小组去另一个小组之前，你需要有一个退出策略。当然，不要粗鲁到把一个人单独丢下，但简单地说声“不好意思”然后移动到房间的另一部分是完全没问题的。是的，真的就这么简单。

这类活动我最喜欢的部分之一，是你连接他人时所拥有的力量。你可能和一个小组待了五分钟，了解到关于某个人的几件事。然后，二十分钟后，你可能会发现一个新的人和你之前遇到的那个人有共同之处。于是，你花时间把这两个人介绍认识。

你为什么要这么做？记住，你们都在寻求建立互利的关系。如果你能把两个可能互相帮助的人联系起来，那么你就是在帮助他人建立他们的人脉网络。你可以肯定，那两个人会记得你是把他们介绍给彼此的人。很可能他们会投桃报李，把你介绍给与你志趣相投的人。

成为帮助连接他人的人，拥有真正的力量。

### 选择正确的房间

当你寻找建立人脉的场所时，有哪些好去处？嗯，我已经谈过专业协会的价值。如果你参加由你的专业协会主办的活动，你至少会知道参加活动的人是来自与你相同领域或行业的人。这可能会让对话更容易一些。一些常见的结识他人的场所包括会议、用户组、本地活动和培训研讨会。

如果你想与他人建立互利关系，那么你需要确保你们双方都有对彼此有价值和感兴趣的东西。在游泳池用品店购买车头灯没什么意义，那么为什么要在一个充满你专业或兴趣领域之外的人的房间里建立人脉呢？然而，我看到并从人们那里听说了正是这种情况，他们告诉我他们对社交活动有多么失望。

在抱怨你的结果有多糟糕之前，请确保你一开始就在正确的房间里。

### 让事情变得更简单



我常听人说建立人脉有多难。虽然我喜欢提醒大家，生活中有价值的事物往往也伴随着困难，但我本人也属于喜欢把事情简单化的人。让社交变得简单的一个方法是明确自己的目标。如果你想提升销售额，那么你应当寻找特定类型的社交场合。如果你只是想扩大职业联系人名单，那么你需要的场合又会有所不同。最不希望看到的情况莫过于投入了时间和精力去社交，最后却发现待错了地方。

### **身处其位**

成为一名 `DBA` 有时可能看起来令人生畏。一旦你发现自己身处你曾想象过的那个角色中，你可能会不时感到不堪重负。即使是经验丰富的数据库管理员，面对眼前堆积如山的任务，有时也会感到力不从心。你可能无法避免在某一天被工作淹没，但你可以采取一些措施来尽量减少这种日子。

首先，你需要打下坚实的基础。做到这一点的最佳方法之一就是尽可能多地进行文档记录。写下你能写的一切，无论当时看起来重要与否。文档记录将对你大有裨益，因为通过反复记忆来学习是记住信息的好方法。

其次，在你开始记录你所能找到的一切之后，你就可以与同事分享你的知识和经验。想办法增加你所在机构所有团队之间的沟通交流。有 `Sharepoint` 吗？给自己建一个门户并用它来发布有用的信息。你甚至可以用它来发布 `T-SQL` 的技巧和诀窍，或者讨论一些数据库设计的基础知识。随着时间的推移，你会发现哪种沟通形式对你的团队最有效。
