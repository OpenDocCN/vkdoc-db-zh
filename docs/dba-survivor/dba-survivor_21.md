# 第三章：基础知识

任何时候，只要系统出现哪怕一丝异常迹象，人们首先会做的就是责怪数据库服务器。尽管在你的屏幕和数据库服务器之间存在许多层，但第一个接到的电话总会是你。你会被期望立即调查问题，同时也会惊讶地看到这样的邮件：“我已联系了数据库管理员，他们将修复这个问题。”等等！我们从未说过数据库服务器有问题——你们为什么告诉别人我们要修复什么？问题可能是网络，或者是无法扩展的糟糕设计，又或是*任何原因*。然而，人们就是会死死盯住数据库服务器，因为（1）它是已知的，（2）大多数人不理解它的工作原理。在某些情况下，如果他们收到诸如“无法连接到数据库”之类的错误消息（或者错误消息中简单地出现了“数据库”这个词），人们就会自动假定问题*一定*出在数据库上。

我已经记不清有多少次被告知服务器有问题，结果发现问题是用户或账户没有登录权限。抱歉，但这并不是服务器的问题。同样，如果你试图将 100GB 的数据加载到一个只有 10GB 可用空间的磁盘上，这也不是服务器的问题。

33GB 的 tempdb 驱动器被填满也是同样道理；问题不在于服务器，而在于低效的代码。然而，你的服务器（最终是你自己）将被迫承担过错的负担。

没关系，因为你之所以是一名数据库管理员，原因之一就是你有能力承担这种足以压垮你大多数同行的负担。

## 责备与功劳

在上一节中，我们讨论了人们倾向于责怪他们不理解的事物。另一个你需要习惯的事实是，人们很少会特意为你所做的工作或建议应做的事情给予你功劳。我从未在上班时听到有人对我说：“昨晚干得漂亮，我们的批处理加载比预期快了两秒。”但如果它慢了两秒，那么你可能就会被要求开始进行一种星际迷航式的五级诊断，不管那是什么意思。

你很快会发现，唯一能让你引人注目的时候就是出问题的时候。一些数据库管理员开始认为，展示自己价值的唯一方式就是总在身边去修复出错的事情。就我个人而言，我宁愿从一开始就构建一个稳定运行、不出任何问题的环境，即使这意味着我永远不会被注意到。

如果你发现自己处于不被注意的境地，那么你需要做的就是开始建立一些指标。从简单的事情开始，比如你的数据库备份。追踪过去一周进行了多少次备份，以及有多少次备份失败。经过一段时间，也许一两个月后，你可以用这些数据向你的经理汇报。如果你随着时间的推移减少了备份失败的次数，那么这份报告将帮助你向公司展示你的价值，而不需要等到出了问题才被注意到。

**提示：** 指标和报告是你展示价值的方式；明智地使用它们。

指标是为你构建稳定环境、报告你所做的良好工作以及在灾难之外展示价值的绝佳方式。除了追踪备份和备份失败次数，以下是其他一些值得长期追踪的良好项目：

- 数据库数量（按生产环境、非生产环境等细分）
- 数据库文件在磁盘上使用的空间量
- 数据库备份使用的空间量
- 备份数据库所花费的时间量
- 数据库还原次数
- 新创建的数据库数量

所有这些信息在系统数据库中都很容易获得，并将极大地帮助你确保人们了解你在幕后所做的工作。

## 正确与讨喜的需求

如果你总是正确，没人会喜欢这一点。而每个人又都会希望你总是正确。祝你好运，调和这两件事。

人们来找你，是因为他们想知道把事情做对的正确方法。在这种时候，你希望尽力帮助他们找到正确的解决方案。虽然在这种讨论中未能始终正确并不会很快导致人们要求解雇你，但如果你有必须始终正确的需求，那么你会发现你与同事之间会产生问题。

如果出了问题（而且*总是*会出问题），而你的数据库服务器受到责备（70%的情况下会发生），你告诉人们问题不在数据库服务器（90%的情况下确实如此），结果也的确证明问题不在数据库服务器，那么通过提醒人们你始终是对的，这对你自己没有任何好处。

在这一点上请相信我。不要觉得有必要提醒人们他们一开始就该听你的。随着时间的推移，人们要么会因为你的建议和经验而被你吸引，要么会因为讨厌每次他们在办公室里跑来跑去大喊天要塌了，而你让他们坐下别喊时的那种感觉而远离你。

请记住，在薪资评估时，你的“支持率”很重要。你可能总是正确，但由于你的态度而被忽视，结果你不会获得任何绩效加薪。然而，你可能非常受欢迎，几乎总是错的，却发现自己在薪资等级上不断攀升。

你会遇到与你共事的这些人，他们要么（1）自己想正确，要么（2）希望你偶尔错一次。这会发生，就像太阳会从东方升起一样。当这种情况发生时不要惊慌；当你感觉到别人情绪激动时，保持冷静。你最不希望看到的就是问题升级。如有必要，提议与一个中立的第三方进行讨论，也许是你们都尊重的一位同事或经理。陈述你的理由，并明智地选择你的“战斗”。有时，即使对方是错的，做出让步也会带来明显的好处。

当你做出让步时，这将有助于在一定程度上表明你是可以讲道理的。这也可能会为你在同事那里赢得一些好感分。事实上，如果你让步足够多，你会发现你可能会在办公室里非常受欢迎。

虽然这可能会让你自我感觉良好，但你需要平衡你对受人喜爱的渴望与你执行标准和遵守准则的需求。这是一条难以把握的界限，但随着时间的推移，你将学会哪些“战斗”是值得去打的。
