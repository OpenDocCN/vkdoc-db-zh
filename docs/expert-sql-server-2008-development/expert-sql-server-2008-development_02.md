# 精通 SQL Server 2008 开发

以及创建一流数据库应用所需的技能，这些应用体现了数据库开发的最佳实践。

*《SQL Server 2008 空间数据开发入门》*

本书涵盖的主题代表了数据库开发中有趣、有时复杂且常被误解的方面。理解这些领域将使你成为一名与众不同的 SQL Server 专家开发者。其中一些主题在软件社区内争论激烈，对于任何给定的问题，并不总有一个单一的“最佳”解决方案。相反，我将向你展示多种方法，并提供信息和工具，以便你决定哪种方法最适合你的特定环境。

阅读本书后，你将深刻理解测试和异常处理等领域，以确保你的代码健壮、可扩展且易于维护。你将学习如何通过控制对敏感信息的访问来创建安全的数据库，并了解如何加密数据以保护其免受窥探。你还将学习如何使用动态 SQL 和 SQLCLR 创建灵活、高性能的应用程序，并会发现处理数据库并发用户的各种模型。最后，我将教你如何处理代表时间、空间和层级信息的复杂数据。我们将一起揭示在这些情况下可能出现的一些有趣问题。

我为这本书付出了巨大努力，力求使其对所有技能水平的读者都有用。无论你是初学者、专家还是介于两者之间，你都能在本书中找到有用的内容。我希望它能帮助你成为一名真正精通 SQL Server 的开发者。
