# 关于作者

**Alastair Aitchison** 是一位常驻英国诺里奇的独立技术顾问。他拥有广泛的软件和服务平台经验，并自 SQL Server 2008 最早的公开技术预览版起便开始使用它。他在金融服务领域实施过多种需要高度并发处理和大型数据仓库的 SQL Server 解决方案，并结合了基于 Microsoft 商业智能堆栈的报告和分析功能。Alastair 对空间数据分析有特别的兴趣，是《*SQL Server 2008 空间应用入门*》（Apress，2009）一书的作者。他在用户组和会议上发表演讲，并且是多个在线支持社区（包括 Microsoft SQL Server 开发者中心论坛）的非常活跃的贡献者。
