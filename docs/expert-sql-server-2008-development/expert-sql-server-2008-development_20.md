# 致谢

当我被邀请撰写本书时，我欣然接受了再次与 Apress 优秀团队合作的机会。我特别幸运能再次得到 Jonathan Gennick 和 Evan Terry 这两位才华横溢的人士的协助。作为我的编辑，Jonathan 在整个写作过程中鼓励、教导和指导我，即使在遇到困难时（就像任何出版时间表一样，有时确实会遇到！）他也从未动摇过他的支持。Evan 不仅提供了他丰富的技术知识，还有他的写作专业知识，有时他只是提供了一个理智的声音，所有这些都极大地帮助了本书的改进。我还要感谢 Mary Tobin，她负责跟踪所有的截止日期和项目管理问题；感谢 Damon Larson，他纠正了我对英语语言的随意使用；以及所有其他帮助将本书呈现在您手中的个人。谢谢大家。

我的家人再次忍受了我长时间在键盘上打字的时光，我感谢他们的宽容、耐心和支持。没有他们，我什么都做不了。

最后，感谢您——读者——购买本书。我希望您发现内容有趣、有用，并且最重要的是，阅读愉快。
