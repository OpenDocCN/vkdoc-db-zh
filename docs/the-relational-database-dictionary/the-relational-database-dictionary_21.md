#### 等价类

在某个给定集合 `s` 中，满足以下性质的一个子集 `ss`：(a) `ss` 中的元素在某个给定的等价定义下彼此等价；(b) 在相同的等价定义下，它们与 `s` 中的任何其他元素都不等价。（注意此概念与关系分组和汇总的相关性，q.v.；*另见* 映像关系）。关于更正式的定义，*参见* 等价关系。

*示例*：1. 令 `s` 为所有正整数的集合，并定义正整数 `x` 和 `y` 等价，当且仅当它们在常规十进制记法（无前导零）中具有相同的位数。那么，`s` 中包含所有一位整数的子集，在此等价定义下就是一个等价类；由所有两位整数、所有三位整数等组成的子集也同样如此。2. 考虑关系变量 `P` 当前表示的零件集合。定义两个零件等价，当且仅当它们颜色相同。那么，`P` 当前表示的所有红色零件的集合，在此等价定义下是一个等价类；所有蓝色零件的集合、所有黄色零件的集合等也同样如此。3. 考虑关系变量 `SP` 当前值中的元组集合。定义两个元组等价，当且仅当它们包含相同的 `S#` 值。那么，所有供应商编号为 `S1` 的此类元组的集合，在此等价定义下是一个等价类；所有供应商 `S2` 的此类元组集合、所有供应商 `S3` 的此类元组集合等也同样如此。

#### 等价关系

令 `r` 为一个二元关系。那么，`r` 是一个等价关系，当且仅当它是自反的、对称的和传递的（q.v.）。此外，令 `x` 为一个值，使得对于某个 `y`，元组 `<x, y>` 出现在 `r` 中。给定该值 `x`，那么所有此类值 `y` 的集合是关于 `r` 的一个等价类——即与给定值 `x` 对应的那个特定等价类（*参见* 等价类）。观察到，如果 `ry` 是出现在 `r` 中的所有值 `y` 的集合，那么 `ry` 中的每个值恰好出现在关于 `r` 的一个等价类中——换句话说，等价类两两不相交，它们共同划分了相关的值集合。
