# **关系数据库词典（扩展版）**

• 简洁、正确、明确的定义，附有适当示例
• C. J. Date 对术语使用的独特态度与见解 由于本书专为关系数据库专业人士编写，您无需费力查找那些具有特殊数据库含义、令人烦恼的常用术语。它们都收录于此，并且定义仅限于它们在关系数据库中的相关含义。

C. J. Date 是一位独立作家、讲师、研究员和顾问，专攻关系数据库技术（他是该领域的先驱之一）。他因其著作 *《数据库系统导论》*（第 8 版，2004 年）而闻名，该书已售出超过 75 万册，被全球数百所学院和大学采用。他还撰写了多本关于关系数据库管理的其他书籍，最近的一本是 *《逻辑与数据库：关系理论之根源》*（Trafford 出版社，2007 年）。他于 2004 年入选 computing industry hall of fame。
