# 连接词
一种只读的一元或二元逻辑运算符。在二值逻辑中，正好有 20 个连接词，其中 4 个一元，16 个二元（直接对应于 4 种可能的一元和 16 种可能的二元真值表）。实践中最常遇到的连接词是`NOT`（否定）、`OR`（析取）、`AND`（合取）、`IMPLIES`（蕴涵）和`EQUIV`（等价）；其他连接词包括`NAND`、`NOR`和`XOR`，*q.v.*
*注意*：还有各种其他符号和关键字也用于表示这些连接词，其中一些（但非全部）已包含在本词典中。

#### 后件
*参见* 蕴涵。

#### 一致性
完整性，*q.v.* *对比* 正确性。

#### 一致的
1.  （*逻辑*）一组谓词是一致的，当且仅当存在至少一组参数，可以替换这些谓词的参数，使得每个结果命题的计算结果都为`TRUE`。
2.  （*指数据库*）满足所有定义的完整性约束。
