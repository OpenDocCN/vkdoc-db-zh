## 12\. *在编写这个扩展版时，我略感惊讶地发现，数据库概念在多大程度上最终依赖于某些数学术语和构造。因此，我决定收录一些带有一定数学色彩的条目；这里的示例包括布尔代数、群、逆、非负、偏序，以及关系和元组的数学（相对于关系模型）定义。这些条目的相关性可能不会立即显现，但我认为将它们收集到一处是有用的，可以为任何希望更深入探究“关系代数”这样的术语（或“关系”这个术语本身）的精确含义和起源的人提供一个方便的参考。*
