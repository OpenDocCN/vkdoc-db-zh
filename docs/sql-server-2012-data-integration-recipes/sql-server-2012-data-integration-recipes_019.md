# 关于技术审阅者

**本·伊顿（Ben Eaton）** 是一位常驻英格兰中部郡县的独立顾问，专注于使用微软技术栈进行商业智能、软件架构和应用开发。本在 1990 年代末期以微软 `Access` 开始其专业开发生涯，直到他发现，要在企业级别提供同样的端到端数据管理功能，需要一套全新的工具集。`Access` 数据库搭配 `SQL Server` 后端很快被 `.NET SOA` 应用程序和对 `Reporting Services` 的早期采用所取代。除了偶尔涉足 `SharePoint`，他现在主要为广泛的私营和公共部门客户使用 `SQL Server` 技术栈（`SSIS`、`SSAS`、`SSRS`）以及大部分 `.NET framework`（`WCF`、`WPF`、`ASP.NET`）。

**贾森·布里姆霍尔（Jason Brimhall）** 首先是一位顾家的男人。他在 `IT` 领域拥有超过 15 年的经验，从 `SQL Server 6.5` 开始就与 `SQL Server` 打交道。他曾在不同行业的大小公司工作过。他在性能调优、高事务处理环境、大型环境以及 `VLDBs` 方面拥有丰富经验。他目前是一名 `DB 架构师`，并且是 `SQL 2008` 的 `MCITP`。贾森定期为 `PASS` 志愿服务，并且是拉斯维加斯用户组（`SSSOLV`）的 `VP`（副主席）。您可以在他的博客阅读更多来自贾森的文章：`http://jasonbrimhall.info`。
