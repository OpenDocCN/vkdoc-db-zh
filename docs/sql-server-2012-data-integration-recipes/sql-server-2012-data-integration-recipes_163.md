# 导入/导出向导中的默认数据映射

导入/导出向导的数据转换基于一组 XML 文件，你可以在以下目录中找到它们：`C:\Program Files (x86)\Microsoft SQL Server\110\DTS\MappingFiles` 和/或 `C:\Program Files\Microsoft SQL Server\110\DTS\MappingFiles`。

了解这些文件的存在有三个充分的理由：

*   它们提供了数据类型映射的基准参考，虽然可能不是最终的，但可以作为一个有价值的指南。
*   可以修改这些文件以适应你偏好使用的数据类型映射。
*   你可以编写自己的数据类型映射文件以供导入/导出向导使用——尽管我不会在此展示如何操作。

表 A-3 到 表 A-35 以表格形式提供了来自其中一些文件的映射数据，因此你可以看到导入/导出向建议的基本数据类型映射，并可能将其作为你自己转换过程的基础。

## 示例数据库与脚本

### 示例数据库与文件
正如我在引言中所说，我倾向于使用一组不同于经典的 `AdventureWorks` 示例数据库，因为后者可能过于复杂，无法简单清晰地演示数据集成过程。我提议的数据库架构极其简单且非常轻量级。实际上，它将导致你所遇到过的最微小的数据加载。然而，考虑到现实工作很可能对此有所弥补，我希望没有人会介意太多。

示例数据包含五个数据库：
| 数据库 | 描述 |
| --- | --- |
| `CarSales` | 主数据库。有时用作源数据。 |
| `CarSales_Staging` | 用于暂存过程和作为 SQL Server 数据源的数据库。 |
| `CarSales_DW` | 用于将数据加载到 SSAS 多维数据集中的维度数据库。 |
| `CarSales_Logging` | 用于记录和审核 ETL 过程的独立数据库。 |
| `CarSales_Cube` | 作为维度数据源的 SSAS 数据库。 |

这些数据库——连同本书中使用的所有其他示例文件——都包含在 `SQL2012DIRecipes.Zip` 文件中，可在 `www.apress.com/9781430247913` 获取。只需点击页面中部附近标有“Source Code/Downloads”的标签即可下载该文件。然后使用你首选的工具解压缩。请确保保留底层的目录结构。

### CarSales
使用的主数据库是 `CarSales` 数据库。它由六张表组成：
*   `Client` (客户)
*   `Colors` (颜色)
*   `Countries` (国家)
*   `Invoice` (发票)
*   `Invoice_Lines` (发票明细)
*   `Stock` (库存)

选择此设计是因为它几乎涵盖了所有 SQL Server 数据类型，并且只有最少的参照完整性约束来模拟真实世界环境。

`C:\SQLDIRecipes\Backups` 文件夹（假设你已从本书网站下载了示例）包含以下文件：
| 文件 | 描述 |
| --- | --- |
| `CarSales.Bak` | `CarSales` 数据库备份文件。 |
| `CarSales_Staging.Bak` | `CarSales_Staging` 数据库备份文件。 |
| `CarSales_DW.Bak` | `CarSales_DW` 数据库备份文件。 |
| `CarSales_Cube.abf` | `CarSales_Cube` SSAS 数据库备份文件。 |
| `CarSales_Logging.Bak` | `CarSales_Logging` 数据库备份文件。 |

### 创建 CarSales 数据库
可以使用以下脚本（`C:\SQLDIRecipes\Databases\CarSalesDatabaseCreation.Sql`）创建 `CarSales` 数据库：
```sql
USE master
GO
CREATE DATABASE CarSales
 CONTAINMENT = NONE
 ON PRIMARY
     (NAME = N'CarSales', FILENAME = N'C:\SQLDIRecipes\Databases\CarSales.mdf',
       SIZE = 33792KB , MAXSIZE = UNLIMITED, FILEGROWTH = 1024KB )
 LOG ON
     (NAME = N'CarSales_log', FILENAME = N'C:\SQLDIRecipes\Databases\CarSales_log.ldf',
       SIZE = 4224KB , MAXSIZE = 2048GB , FILEGROWTH = 10%)
GO

ALTER DATABASE CarSales SET COMPATIBILITY_LEVEL = 110
GO
ALTER DATABASE CarSales SET QUOTED_IDENTIFIER OFF
GO
ALTER DATABASE CarSales SET RECOVERY SIMPLE
GO
ALTER DATABASE CarSales SET  READ_WRITE
GO
```
如果数据库已存在，可以使用以下脚本删除它： (`C:\SQLDIRecipes\Databases\DropDatabase.Sql`)
```sql
IF db_id('CarSales') IS NOT NULL
DROP DATABASE CarSales;
GO
```

) ;
GO

If OBJECT_ID('dbo.Colours ') IS NOT NULL DROP TABLE dbo.Colours

GO

) ;
GO

If OBJECT_ID('dbo.Invoice_Lines ') IS NOT NULL DROP TABLE dbo.Invoice_Lines
