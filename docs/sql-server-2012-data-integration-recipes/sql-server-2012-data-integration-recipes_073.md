# SSIS 2005/2008 中的文件 DSN

`文件 DSN`也可用于配置`SSIS ODBC`连接管理器。但值得注意的是，文件`DSN`中的配置信息仅被用作来源，并不会与该文件保持链接，因为所有配置信息都存储在`SSIS`中。

1.  按照“`SSIS 2005/2008`中的`ODBC`”迷你指南中的步骤 1 至 5 操作。
2.  确保选中`使用连接字符串`单选按钮。点击`构建`。
3.  浏览到您创建——或被提供的文件`DSN`。
4.  选择文件`DSN`后，点击`确定`。
5.  提供任何所需的安全/登录信息。对话框应类似于图 6-31。
6.  测试连接。点击`确定`。
