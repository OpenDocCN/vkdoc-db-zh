# 关于作者

![image](img/9781430247913_unFM-01.jpg)

**亚当·阿斯平（Adam Aspin）** 是一位常驻英国的独立商业智能顾问。他使用 `SQL Server` 已有十七年。在此期间，他基于微软 `BI` 产品套件开发了数十个报告和分析系统。

亚当毕业于牛津大学，其职业生涯始于出版业，之后转入 `IT` 领域。数据库很快成为他的热情所在，他在这一领域的经验涵盖从 `dBase` 到 `Oracle`，以及从 `Access` 到 `MySQL`，其间也偶尔涉足 `DB2` 的世界。然而，在使用 `SQL Server Analysis Services`、`SQL Server Reporting Services`，尤其是 `SQL Server Integration Services` 时，他在微软的宇宙中最为得心应手。

近十年来，商业智能一直是他的主要专注领域。他曾为众多客户提供服务，应用其技能，客户包括摩根大通（J.P. Morgan）、经济合作与发展组织（`OECD`）、乐购（Tesco）、森特理克（Centrica）、哈罗德百货（Harrods）、沃达丰（Vodafone）、法国农业信贷银行（Crédit Agricole）、卡地亚（Cartier）以及 `EMC Conchango`。

亚当多年来一直是 `SQLServerCentral.com` 的积极投稿人。他曾为多家法国 `IT` 出版物撰写大量文章。亚当能说流利的法语，并在法国和瑞士工作多年。

可通过 `adam@calidra.co.uk` 联系他。
