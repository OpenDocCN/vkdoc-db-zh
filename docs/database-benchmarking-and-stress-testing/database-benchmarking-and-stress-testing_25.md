# 总结

在本章中，我向你介绍了一款免费的数据库压力测试工具 `DBBENCHMARK`，它易于使用，并且不像 `SLOB` 那样需要编译任何代码。此外，你可以轻松编辑该脚本，使其适用于几乎任何数据库。我提供了 Oracle 和 MySQL 两个版本，但要切换到 `PostgreSQL` 或其他任何关系型数据库，最多只需一小时的工作量。`DBBENCHMARK` 真正的妙处在于它可以将结果绘制成图表，这样你就不必翻阅大量的监控或日志输出来寻找规律。一图胜千言。我还展示了，通过使用 `DBBENCHMARK`，我在极短的时间内、花费很少的精力就发现，我的笔记本电脑仅配备一块 `SSD`，最多只能处理 180 个并发数据库会话。为了实现更高的扩展性，我需要添加更多磁盘来分散 IO。而找到这个神奇数字，花费了不到一个小时。具体结果因人而异，但你应该也能同样轻松地使用此工具找出任何数据库服务器的最大工作负载。
