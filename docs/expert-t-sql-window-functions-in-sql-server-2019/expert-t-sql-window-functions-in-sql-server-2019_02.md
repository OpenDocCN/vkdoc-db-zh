# 引言

几年前，我会为每个新版本的 `SQL Server` 制作用户组演示，介绍新的 `T-SQL` 功能。2012 年有太多内容可讲，我决定专门就那年引入的窗口函数做一个演示。最终，我积累了如此多的材料，以至于它变成了两次会议。多年来，我可能在美国和英国各地的活动中至少演示过 50 次这些信息。尽管如此，大多数人仍然没有使用窗口函数，因为他们没有听说过，或者没有意识到其好处。

## 本书内容

本书涵盖了每种类型的窗口函数，从 `SQL Server 2005` 引入的排名函数开始，一直到 2012 年引入的统计函数。每章都解释了如何使用这些函数及其选项，并提供了几个如何使用的简单示例。遗憾的是，微软上一次添加任何新的窗口函数是在 2012 年，不过最近有一些性能改进。其中一章专门讨论窗口函数的性能。

最后，本书的最后两章涵盖了一些实际案例。在 `第 9 章`，你将学习如何分析一个大型数据集——超过 100 年的棒球统计数据。`第 10 章` 和 `第 11 章` 展示了如何在数据仓库计算中使用窗口函数，而不是构建多维数据集。

## 目标读者

本书适合已经具备良好 `T-SQL` 技能的人员。他们知道如何连接表、使用子查询和 `CTEs`，以及编写聚合查询。尽管具备这些技能，他们偶尔会遇到一些不易用基于集合的方式解决的问题。如果没有窗口函数，其中一些问题只能使用游标或昂贵的三角连接来解决。通过运用本书中传授的概念，你的 `T-SQL` 技能将提升到一个新的水平。一旦你开始使用窗口函数，你会发现更多学习它们的理由。
